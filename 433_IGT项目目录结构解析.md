### 二、实践操作

#### 2.1 目录导航脚本

```bash
#!/bin/bash
# navigate_igt.sh - IGT 目录导航辅助

IGT_DIR="${1:-$HOME/igt-gpu-tools}"

if [ ! -d "$IGT_DIR" ]; then
    echo "错误: IGT 目录不存在: $IGT_DIR"
    exit 1
fi

echo "=== IGT 项目结构概览 ==="
echo "目录: $IGT_DIR"
echo ""

# 统计各目录文件数
echo "文件统计:"
for dir in lib tests tools runner docs scripts data; do
    if [ -d "$IGT_DIR/$dir" ]; then
        count=$(find "$IGT_DIR/$dir" -type f | wc -l)
        echo "  $dir/: $count 文件"
    fi
done
echo ""

# 列出核心库文件
echo "核心库文件 (lib/):"
ls -la "$IGT_DIR/lib/"*.c 2>/dev/null | awk '{print "  " $9}' | head -20
echo ""

# 列出测试分类
echo "测试分类 (tests/):"
ls -la "$IGT_DIR/tests/" 2>/dev/null | grep "^d" | awk '{print "  " $9}' | grep -v "^\.$\|^\.\.$"
echo ""

# 列出 AMDGPU 测试
echo "AMDGPU 专用测试 (tests/amdgpu/):"
if [ -d "$IGT_DIR/tests/amdgpu" ]; then
    ls -la "$IGT_DIR/tests/amdgpu/" 2>/dev/null | grep "\.c$" | awk '{print "  " $9}'
else
    echo "  目录不存在"
fi
echo ""

# 列出工具
echo "辅助工具 (tools/):"
ls -la "$IGT_DIR/tools/" 2>/dev/null | grep "\.c$" | awk '{print "  " $9}' | head -10
echo ""
```

#### 2.2 快速查找功能

```bash
#!/bin/bash
# find_igt_function.sh - 在 IGT 中查找函数定义

set -e

IGT_DIR="${1:-$HOME/igt-gpu-tools}"
FUNCTION="$2"

if [ -z "$FUNCTION" ]; then
    echo "用法: $0 <igt-dir> <function-name>"
    exit 1
fi

echo "=== 查找函数: $FUNCTION ==="
echo ""

# 在头文件中查找声明
echo "头文件声明:"
grep -rn "^.*$FUNCTION.*(" "$IGT_DIR/include/" 2>/dev/null | head -5 || true
grep -rn "$FUNCTION" "$IGT_DIR/lib/"*.h 2>/dev/null | head -5 || true
echo ""

# 在源码中查找定义
echo "源码定义:"
grep -rn "^.*$FUNCTION.*(" "$IGT_DIR/lib/" 2>/dev/null | head -5 || true
grep -rn "^.*$FUNCTION.*(" "$IGT_DIR/tests/" 2>/dev/null | head -5 || true
echo ""

# 查找使用示例
echo "使用示例:"
grep -rn "$FUNCTION(" "$IGT_DIR/tests/" 2>/dev/null | head -10 || true
echo ""
```

## 今日小结

- IGT 项目分为 lib/（核心库）、tests/（测试）、tools/（工具）、runner/（执行器）四大模块
- lib/ 目录包含测试框架核心、DRM 辅助、KMS 辅助、GEM 辅助等关键组件
- tests/amdgpu/ 目录包含 AMD GPU 专用测试
- 理解目录结构有助于快速定位源码和编写新测试
