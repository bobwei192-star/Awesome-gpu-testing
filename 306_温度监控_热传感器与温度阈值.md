## 知识详解

### 1. 概念原理

#### 1.1 GPU 温度监控的重要性

GPU 作为高功耗半导体器件，在运行过程中会产生大量热量。有效的温度监控对于以下方面至关重要：

1. **硬件保护**：防止 GPU 因过热而永久损坏（结温超过最大额定值）。
2. **性能稳定**：维持 GPU 在安全温度范围内稳定运行，避免因热节流导致的性能波动。
3. **能效优化**：温度直接影响半导体器件的漏电流，进而影响功耗与能效。
4. **散热系统设计**：为散热方案（风扇、散热片、液冷）提供设计依据。
5. **用户体验**：控制噪音（风扇转速与温度相关）和表面温度。

AMD GPU 的温度监控系统是一个分层架构：

#### 1.2 热传感器类型与工作原理

AMD GPU 内部集成多种类型的热传感器，用于监测不同部位的温度：

| 传感器类型 | 位置 | 工作原理 | 精度 | 采样率 | 典型用途 |
|-----------|------|----------|------|--------|----------|
| **结温传感器** | GPU 芯片核心内部（临近计算单元） | 基于半导体 PN 结的温度‑电压特性 | ±1‑2°C | 10‑100 Hz | 核心温度监控，热节流触发 |
| **热点传感器** | GPU 芯片表面最热区域 | 多个传感器中读数最高的一个 | ±2‑3°C | 10‑50 Hz | 散热设计验证，峰值温度监控 |
| **显存温度传感器** | GDDR/HBM 显存颗粒附近 | 集成在显存控制器或 PHY 中 | ±3‑5°C | 5‑20 Hz | 显存过热保护，内存频率调整 |
| **VRM 温度传感器** | 电压调节模块（VRM） | 热敏电阻或集成传感器 | ±3‑5°C | 1‑10 Hz | 供电系统保护 |
| **板载环境传感器** | GPU PCB 上（远离核心） | 热敏电阻或数字传感器 | ±2‑4°C | 1‑5 Hz | 环境温度参考，风扇控制 |

**传感器工作原理详解**：

**1. 结温传感器（Junction Temperature Sensor）**：
基于硅 PN 结的正向电压与温度的关系（约 -2 mV/°C）。GPU 内部集成带隙基准源和 ADC，将温度转换为数字读数。

```
┌─────────────────────────────────────────────────────────┐
│            On‑die Junction Temperature Sensor           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─────┐     ┌─────┐     ┌─────┐     ┌─────────────┐  │
│  │ PTAT│     │ Band│     │ 10‑ │     │  Digital    │  │
│  │Current├──►│ Gap ├──►│ bit ├──►│  Temperature  │  │
│  │Source│    │ Ref │    │ ADC │    │   Register   │  │
│  └─────┘     └─────┘     └─────┘     └──────┬──────┘  │
│                                              │         │
│  ┌─────┐     ┌─────┐     ┌─────┐            │         │
│  │Thermal│   │Voltage│   │  MUX │            │         │
│  │Diode ├──►│ to    ├──►│ (8:1)├────────────┘         │
│  │(PN结)│   │Current│   │      │                      │
│  └─────┘     └─────┘     └─────┘                      │
│                                                         │
└─────────────────────────────────────────────────────────┘

数据流：
1. PTAT（Proportional To Absolute Temperature）电流源产生与绝对温度成正比的电流
2. 该电流流过热敏二极管，产生正向电压 Vf
3. Vf 与带隙基准电压比较，差值经 ADC 转换为数字值
4. 数字温度值存储在寄存器中，供 SMU/驱动读取
```

**2. 热点传感器（Hot Spot Sensor）**：
在 GPU 芯片上分布多个温度传感器（如 RDNA 3 有 10+ 个），实时比较所有读数，报告最高值。

**3. 显存温度传感器**：
对于 HBM（High Bandwidth Memory）堆栈，温度传感器集成在内存堆栈内或基础芯片（base die）中。对于 GDDR6，传感器位于显存控制器或 PHY 附近。

**传感器校准与补偿**：
- **工厂校准**：生产过程中在已知温度下测量传感器输出，存储校准系数。
- **实时补偿**：根据电压、老化等因素进行动态补偿。
- **交叉验证**：多个传感器相互验证，检测异常读数。

#### 1.3 温度阈值（Thermal Threshold）

温度阈值是 GPU 热管理系统的关键参数，定义了不同温度区间对应的行为：

| 阈值类型 | 典型值 | 触发动作 | 可配置性 | 内核源码中的定义 |
|----------|--------|----------|----------|------------------|
| **低温阈值** | 0‑20°C | 启动时预热，防止冷启动损坏 | 部分可调 | `THM_THERMAL_LOW_TEMP` |
| **正常操作范围** | 20‑85°C | 无特殊动作，性能不受限 | 无 | - |
| **热节流阈值** | 85‑95°C | 降低频率（SCLK/MCLK）以减少发热 | 可配置 | `amdgpu_thermal_throttle_temp` |
| **临界温度阈值** | 95‑105°C | 强制降频，可能降低电压 | 部分可调 | `THM_THERMAL_CRITICAL_TEMP` |
| **紧急关机阈值** | 105‑115°C | 立即关闭 GPU 或系统关机 | 硬件固定 | 熔断机制，不可配置 |
| **显存温度阈值** | 90‑100°C | 降低显存频率，减少显存负载 | 可配置 | `amdgpu_mem_temp_throttle_temp` |

**阈值迟滞（Hysteresis）机制**：
为避免在阈值附近频繁切换状态，系统采用迟滞设计。例如：
- **升温触发**：温度达到 95°C 时触发节流。
- **降温恢复**：温度降至 90°C 时才解除节流。
- **迟滞宽度**：5°C 的差值防止状态振荡。

#### 1.4 温度监控与电源管理的交互

温度监控不是独立系统，它与 GPU 电源管理（DPM）紧密集成：

```
│                                                                          │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────┐  │
│  │  Thermal    │    │  Threshold  │    │  Decision   │    │ Action  │  │
│  │  Sensors    │───►│ Comparison  │───►│  Engine     │───►│Executor │  │
│  │             │    │             │    │ (SMU/Driver)│    │         │  │
│  └─────────────┘    └─────────────┘    └─────────────┘    └─────────┘  │
│         │                   │                   │               │       │
│  ┌──────▼──────┐    ┌──────▼──────┐    ┌──────▼──────┐    ┌────▼────┐  │
│  │ TJunction   │    │  Temp >=    │    │  Reduce     │    │  Lower  │  │
│  │   = 98°C    │    │  Throttle   │    │  SCLK by    │    │  SCLK   │  │

│  1. 降低频率 → 功耗下降 → 温度下降 → 频率逐步恢复                         │
```

### 2. 实践操作

#### 2.1 查看当前温度信息

**通过 rocm‑smi**：
```bash
# 显示概要信息，包含当前温度
rocm-smi

# 显示详细温度信息
rocm-smi --showtemp

# 显示所有传感器信息
rocm-smi --showallinfo | grep -i temp

# 以 JSON 格式输出温度信息
rocm-smi --showtemp --json

# 显示温度历史（部分版本支持）
rocm-smi --showtempgraph
```

示例输出（Navi 31）：
```

=================================  Card 0  =================================
GPU ID: 0
GPU Part Number: Navi 31 [Radeon RX 7900 XTX]
...

  Temperature (Sensor 2): 72.5 C (Hot Spot)
  Temperature (Sensor 3): 65.0 C (Memory Junction)
```

**通过 sysfs（hwmon）**：
```bash
# 查找 hwmon 设备路径
HWMON_PATH=$(find /sys/class/drm/card0/device/hwmon/ -name "hwmon*" | head -1)
echo "HWMON path: $HWMON_PATH"

# 查看所有可用的温度传感器
ls $HWMON_PATH/temp*_input 2>/dev/null

# 读取各个温度传感器的值（单位：毫摄氏度，1°C = 1000 m°C）
for temp_file in $HWMON_PATH/temp*_input; do
    sensor_name=$(basename $temp_file _input)
    label_file="$HWMON_PATH/${sensor_name}_label"
    if [ -f "$label_file" ]; then
        sensor_label=$(cat $label_file)
    else
        sensor_label="Unknown"
    fi
    temp_mc=$(cat $temp_file)
    temp_c=$((temp_mc / 1000))
    echo "$sensor_label: $temp_c °C ($temp_mc m°C)"
done

# 查看温度阈值（如果支持）
if [ -f "$HWMON_PATH/temp1_crit" ]; then
    crit_temp=$(cat $HWMON_PATH/temp1_crit)
    echo "Critical threshold: $((crit_temp / 1000)) °C"
fi

if [ -f "$HWMON_PATH/temp1_max" ]; then
    max_temp=$(cat $HWMON_PATH/temp1_max)
    echo "Maximum threshold: $((max_temp / 1000)) °C"
fi
```

### 3. 案例分析

#### 3.1 内核 Bug：温度传感器读数错误导致虚假热节流（commit a1b3c1d）

在 Linux 内核 v6.3 中，一个 Bug 导致在某些 RDNA 2 GPU 上，温度传感器偶尔返回错误的高温读数，触发不必要的热节流。

**问题表现**：
- GPU 在轻负载下突然降频，性能下降
- 温度监控显示瞬间温度 spike（如从 60°C 跳到 110°C 再返回）
- 内核日志中出现虚假的热节流消息
- 实际 GPU 温度正常，散热系统工作正常

**根本原因**：
`amdgpu_thermal.c` 中的温度读取函数 `amdgpu_thermal_get_temperature()` 在某些情况下未正确处理传感器读取失败，而是返回上次缓存的值。当 SMU 繁忙或传感器通信超时时，驱动可能读取到过时的或错误的数据。

**修复前的问题代码**：
```c
static int amdgpu_thermal_get_temperature(struct amdgpu_device *adev)
{
    int temp = 0;
    
    if (!adev->pm.dpm_enabled)
        return 0;
    
    // 尝试从 SMU 读取温度
    if (adev->powerplay.pp_funcs->read_sensor) {
        // 问题：如果读取失败，函数可能返回错误码，但调用者将其解释为温度值
        adev->powerplay.pp_funcs->read_sensor(adev,
            AMDGPU_PP_SENSOR_GPU_TEMP,
            (void *)&temp);
    }
    
    // 没有适当的错误检查
    return temp;  // 可能返回错误码（负数）作为温度！
}
```

**修复后的代码**：
```c
static int amdgpu_thermal_get_temperature(struct amdgpu_device *adev)
{
    int temp = 0;
    int ret;
    
    if (!adev->pm.dpm_enabled)
        return 0;
    
    if (adev->powerplay.pp_funcs->read_sensor) {
        ret = adev->powerplay.pp_funcs->read_sensor(adev,
            AMDGPU_PP_SENSOR_GPU_TEMP,
            (void *)&temp);
        
        // 添加错误检查
        if (ret) {
            // 读取失败，返回安全值或上次有效读数
            if (adev->pm.thermal.last_valid_temp > 0)
                return adev->pm.thermal.last_valid_temp;
            else
                return 0;  // 返回 0 而不是错误码
        }
        
        // 验证温度值在合理范围内
        if (temp < -20 || temp > 120) {
            // 明显不合理的温度值
            if (adev->pm.thermal.last_valid_temp > 0)
                return adev->pm.thermal.last_valid_temp;
            else
                return 0;
        }
        
        // 存储最后一次有效读数
        adev->pm.thermal.last_valid_temp = temp;
    }
    
    return temp;
}
```

#### 3.2 实际案例：数据中心 GPU 温度监控优化

在云计算环境中，AMD Instinct MI250X 加速卡的温度监控面临特殊挑战：

**挑战**：
1. **高密度部署**：多卡机箱内热量积聚，局部温度梯度大。
2. **持续高负载**：AI 训练任务连续运行数天，温度稳定性要求高。
3. **远程监控**：需要通过网络远程监控温度，不能依赖本地工具。
4. **预测性维护**：需要提前预测散热问题，避免硬件故障。

**解决方案**：
1. **多传感器融合**：同时监控核心温度、热点温度、显存温度、进风口温度、出风口温度。
2. **自适应阈值**：根据环境温度和负载动态调整温度阈值。
3. **温度预测模型**：基于历史数据预测未来温度趋势，提前调整风扇转速。
4. **分布式监控系统**：通过 IPMI 和 ROCm SMI 远程收集温度数据。

**实施代码示例**：
```python
# temperature_monitoring_dashboard.py
# 数据中心 GPU 温度监控面板

import json
import time
import requests
from datetime import datetime
import matplotlib.pyplot as plt

class GPUTemperatureMonitor:
    def __init__(self, gpu_nodes):
        self.gpu_nodes = gpu_nodes  # 节点列表 [{'ip': '10.0.0.1', 'gpu_count': 8}, ...]
        self.temperature_history = {}
        
    def fetch_temperature(self, node_ip):
        """通过 ROCm SMI REST API 获取 GPU 温度"""
        try:
            # 假设 ROCm SMI 提供 REST API
            response = requests.get(f"http://{node_ip}:5000/api/v1/gpu/temperature", timeout=5)
            if response.status_code == 200:
                return response.json()
        except Exception as e:
            print(f"Error fetching temperature from {node_ip}: {e}")
        return None
    
    def monitor_all_nodes(self):
        """监控所有节点"""
        all_temps = {}
        for node in self.gpu_nodes:
            temps = self.fetch_temperature(node['ip'])
            if temps:
                all_temps[node['ip']] = temps
                self._update_history(node['ip'], temps)
        return all_temps
    
    def _update_history(self, node_ip, temperatures):
        """更新温度历史记录"""
        timestamp = datetime.now().isoformat()
        if node_ip not in self.temperature_history:
            self.temperature_history[node_ip] = []
        
        # 保留最近 1000 个记录
        self.temperature_history[node_ip].append({
            'timestamp': timestamp,
            'temperatures': temperatures
        })
        if len(self.temperature_history[node_ip]) > 1000:
            self.temperature_history[node_ip].pop(0)
    
    def check_thresholds(self, temperatures, thresholds):
        """检查温度是否超过阈值"""
        alerts = []
        for gpu_id, temp_data in temperatures.items():
            for sensor_type, temp in temp_data.items():
                threshold = thresholds.get(sensor_type, 85)  # 默认阈值 85°C
                if temp > threshold:
                    alerts.append({
                        'gpu': gpu_id,
                        'sensor': sensor_type,
                        'temperature': temp,
                        'threshold': threshold,
                        'timestamp': datetime.now().isoformat()
                    })
        return alerts
    
    def predict_temperature_trend(self, node_ip, lookahead_minutes=30):
        """预测未来温度趋势（简化线性回归）"""
        if node_ip not in self.temperature_history or len(self.temperature_history[node_ip]) < 10:
            return None
        
        # 使用简单线性回归预测
        history = self.temperature_history[node_ip]
        # 提取最近的核心温度
        recent_temps = []
        for record in history[-20:]:  # 最近 20 个记录
            # 获取第一个 GPU 的核心温度
            if 'gpu0' in record['temperatures'] and 'junction' in record['temperatures']['gpu0']:
                recent_temps.append(record['temperatures']['gpu0']['junction'])
        
        if len(recent_temps) < 5:
            return None
        
        # 简单线性外推
        from scipy import stats
        x = list(range(len(recent_temps)))
        slope, intercept, r_value, p_value, std_err = stats.linregress(x, recent_temps)
        
        # 预测未来值
        future_x = len(recent_temps) + lookahead_minutes  # 假设每分钟一个样本
        predicted_temp = slope * future_x + intercept
        
        return {
            'current': recent_temps[-1],
            'predicted': predicted_temp,
            'trend': 'increasing' if slope > 0.1 else 'decreasing' if slope < -0.1 else 'stable',
            'confidence': r_value**2  # R² 值作为置信度
        }
    
    def generate_report(self):
        """生成温度报告"""
        report = {
            'timestamp': datetime.now().isoformat(),
            'nodes': [],
            'alerts': [],
            'recommendations': []
        }
        
        for node in self.gpu_nodes:
            temps = self.fetch_temperature(node['ip'])
            if not temps:
                continue
                
            node_report = {
                'ip': node['ip'],
                'gpu_count': node['gpu_count'],
                'temperatures': temps,
                'max_temp': 0,
                'avg_temp': 0
            }
            
            # 计算统计信息
            all_temps = []
            for gpu_data in temps.values():
                for temp in gpu_data.values():
                    all_temps.append(temp)
            
            if all_temps:
                node_report['max_temp'] = max(all_temps)
                node_report['avg_temp'] = sum(all_temps) / len(all_temps)
            
            report['nodes'].append(node_report)
            
            # 检查阈值
            thresholds = {'junction': 90, 'hotspot': 95, 'memory': 95}
            alerts = self.check_thresholds(temps, thresholds)
            report['alerts'].extend(alerts)
            
            # 温度趋势预测
            trend = self.predict_temperature_trend(node['ip'])
            if trend and trend['predicted'] > 90:
                report['recommendations'].append({
                    'node': node['ip'],
                    'action': 'increase_fan_speed',
                    'reason': f"Predicted temperature will reach {trend['predicted']:.1f}°C in 30 minutes",
                    'priority': 'high' if trend['predicted'] > 95 else 'medium'
                })
        
        return report

# 使用示例
if __name__ == "__main__":
    # 配置监控节点
    nodes = [
        {'ip': '10.0.0.1', 'gpu_count': 8},
        {'ip': '10.0.0.2', 'gpu_count': 8},
        {'ip': '10.0.0.3', 'gpu_count': 8}
    ]
    
    monitor = GPUTemperatureMonitor(nodes)
    
    # 持续监控循环
    while True:
        report = monitor.generate_report()
        
        print(f"\n=== Temperature Report {datetime.now().strftime('%Y-%m-%d %H:%M:%S')} ===")
        
        # 显示节点温度
        for node in report['nodes']:
            print(f"Node {node['ip']}: Max={node['max_temp']}°C, Avg={node['avg_temp']:.1f}°C")
        
        # 显示警报
        if report['alerts']:
            print(f"\n⚠️  ALERTS ({len(report['alerts'])}):")
            for alert in report['alerts']:
                print(f"  GPU {alert['gpu']} {alert['sensor']}: {alert['temperature']}°C > {alert['threshold']}°C")
        
        # 显示建议
        if report['recommendations']:
            print(f"\n💡 RECOMMENDATIONS:")
            for rec in report['recommendations']:
                print(f"  {rec['node']}: {rec['action']} ({rec['reason']})")
        
        # 保存报告
        with open(f"temperature_report_{datetime.now().strftime('%Y%m%d_%H%M%S')}.json", 'w') as f:
            json.dump(report, f, indent=2)
        
        time.sleep(60)  # 每分钟检查一次
```

### 5. 附录1：温度传感器校准方法

#### 5.1 软件校准流程

片上温度传感器在出厂时经过校准，但由于工艺偏差和老化，可能需要现场校准：

**三点校准法**：

```
T_corrected = a * T_raw + b

其中：
  T_raw: 原始传感器读数
  a: 增益校准系数
  b: 偏移校准系数
```

| 校准点 | 参考温度源 | 方法 |
|--------|-----------|------|
| 低温（0°C） | 冰水浴或恒温箱 | GPU 空闲状态下置于恒温环境 |
| 常温（25°C） | 精密温度计 | 测量散热片底部温度 |
| 高温（80°C） | 恒温箱或热风枪 | GPU 满载状态下控制环境温度 |

**校准 Python 工具**：

```python
#!/usr/bin/env python3
"""GPU 温度传感器校准工具"""
import subprocess
import time
import numpy as np
from scipy import stats  # 需要 scipy

class ThermalSensorCalibrator:
    def __init__(self, gpu_id=0):
        self.gpu_id = gpu_id
        self.hwmon_path = f"/sys/class/drm/card{gpu_id}/device/hwmon"
        self.calibration_points = []  # (ref_temp, raw_temp)
    
    def read_gpu_temp(self):
        """从 hwmon 读取当前 GPU 温度"""
        import os, glob
        hwmon_dirs = glob.glob(f"{self.hwmon_path}/hwmon*/temp1_input")
        if not hwmon_dirs:
            return None
        try:
            with open(hwmon_dirs[0], 'r') as f:
                return int(f.read().strip()) / 1000.0
        except (IOError, OSError, ValueError):
            return None
    
    def add_calibration_point(self, ref_temp_c, samples=10):
        """在稳定温度下添加校准点"""
        raw_temps = []
        for _ in range(samples):
            t = self.read_gpu_temp()
            if t is not None:
                raw_temps.append(t)
            time.sleep(0.5)
        
        if not raw_temps:
            print("无法读取 GPU 温度")
            return
        
        raw_avg = np.mean(raw_temps)
        self.calibration_points.append((ref_temp_c, raw_avg))
        print(f"校准点: 参考={ref_temp_c:.1f}°C, 原始平均={raw_avg:.1f}°C")
    
    def compute_calibration(self):
        """使用最小二乘法计算校准系数"""
        if len(self.calibration_points) < 2:
            print("需要至少 2 个校准点")
            return None
        
        ref_temps = np.array([p[0] for p in self.calibration_points])
        raw_temps = np.array([p[1] for p in self.calibration_points])
        
        # 线性回归
        slope, intercept, r_value, p_value, std_err = stats.linregress(raw_temps, ref_temps)
        
        print(f"\n校准结果:")
        print(f"  a (增益) = {slope:.6f}")
        print(f"  b (偏移) = {intercept:.4f}")
        print(f"  R² = {r_value**2:.6f}")
        print(f"  标准误差 = {std_err:.4f}°C")
        print(f"\n校准公式: T_corrected = {slope:.6f} * T_raw + {intercept:.4f}")
        
        return {'a': slope, 'b': intercept}

if __name__ == "__main__":
    calibrator = ThermalSensorCalibrator()
    
    print("GPU 温度传感器校准工具")
    print("请将 GPU 置于已知温度环境...")
    
    # 示例：在三个温度点校准
    calibrator.add_calibration_point(25.0)  # 常温
    calibrator.add_calibration_point(50.0)  # 中温
    calibrator.add_calibration_point(80.0)  # 高温
    
    result = calibrator.compute_calibration()
    if result:
        print(f"\n请在驱动中应用: a={result['a']:.6f}, b={result['b']:.4f}")
```

#### 5.2 传感器精度验证方法

**交叉验证方法**：

1. **多传感器一致性校验**：同一 GPU 上多个热二极管读数差异 < 3°C
2. **热平衡验证**：GPU 稳态运行时，结温与环境温度之差符合热阻模型
3. **时间序列平滑性**：温度变化率不超过物理极限（通常 < 20°C/s）
4. **外部参考测量**：使用热电偶或红外热像仪对比

**精度验证脚本**：

```python
#!/usr/bin/env python3
"""温度传感器精度验证"""
import subprocess
import time
import math

def verify_sensor_accuracy(gpu_id=0, duration_sec=300):
    """验证传感器精度的综合测试"""
    import os, glob
    hwmon_base = f"/sys/class/drm/card{gpu_id}/device/hwmon"
    hwmon_dir = glob.glob(f"{hwmon_base}/hwmon*")[0]
    
    temps = {sensor: [] for sensor in ['temp1_input', 'temp2_input']}
    timestamps = []
    
    # 收集数据
    start = time.time()
    while time.time() - start < duration_sec:
        for sensor in temps.keys():
            path = os.path.join(hwmon_dir, sensor)
            try:
                with open(path, 'r') as f:
                    temps[sensor].append(int(f.read().strip()) / 1000.0)
            except (IOError, OSError):
                pass
        timestamps.append(time.time() - start)
        time.sleep(1)
    
    # 统计验证
    print("温度传感器精度验证报告")
    print("=" * 60)
    
    for sensor, values in temps.items():
        if not values:
            continue
        mean = sum(values) / len(values)
        variance = sum((v - mean) ** 2 for v in values) / len(values)
        std = math.sqrt(variance)
        max_rate = max(abs(values[i+1] - values[i]) for i in range(len(values)-1))
        
        print(f"\n{sensor}:")
        print(f"  采样数: {len(values)}")
        print(f"  平均值: {mean:.2f}°C")
        print(f"  标准差: {std:.3f}°C")
        print(f"  最大变化率: {max_rate:.2f}°C/s")
        print(f"  精密度(3σ): {3*std:.3f}°C")
        print(f"  状态: {'PASS' if max_rate < 20 and 3*std < 5 else 'WARNING'}")
    
    # 传感器间一致性
    if len(temps) >= 2:
        keys = list(temps.keys())
        if temps[keys[0]] and temps[keys[1]]:
            diffs = [abs(temps[keys[0]][i] - temps[keys[1]][i]) for i in range(min(len(temps[keys[0]]), len(temps[keys[1]])))]
            max_diff = max(diffs)
            avg_diff = sum(diffs) / len(diffs)
            print(f"\n传感器间一致性:")
            print(f"  最大差异: {max_diff:.2f}°C")
            print(f"  平均差异: {avg_diff:.2f}°C")
            print(f"  状态: {'PASS' if max_diff < 5 else 'WARNING - 传感器不一致'}")

if __name__ == "__main__":
    verify_sensor_accuracy(duration_sec=60)
```

### 6. 附录2：GPU 温度仿真与建模

#### 6.1 热阻网络模型

GPU 热系统可以用 RC 热阻网络建模，类似于电路分析：

```
         P (功耗)
         |
    ┌────┴────┐
    │ Junction │  T_junction
    │    ▼     │
    └────┬────┘
         │ R_jc (结-壳热阻)
         │
    ┌────┴────┐
    │  Case   │  T_case
    │    ▼    │
    └────┬────┘
         │ R_cs (壳-散热器热阻)
         │
    ┌────┴────┐
    │ Heatsink│  T_heatsink
    │    ▼    │
    └────┬────┘
         │ R_sa (散热器-环境热阻)
         │
     T_ambient (环境温度)
```

**热平衡方程**：

```
T_junction = T_ambient + P * (R_jc + R_cs + R_sa)
```

其中 P 为 GPU 功耗（W），R 为热阻（°C/W）。

**瞬态热响应模型**：

```
T_junction(t) = T_initial + P * R_th * (1 - e^(-t / τ))

其中：
  τ = R_th * C_th （热时间常数）
  R_th: 总热阻
  C_th: 热容
```

#### 6.2 Python 热仿真工具

```python
#!/usr/bin/env python3
"""GPU 热行为仿真"""
import numpy as np

class GPUThermalSimulator:
    def __init__(self, r_jc=0.15, r_cs=0.05, r_sa=0.30, 
                 c_jc=100, c_cs=50, c_sa=200):
        """
        初始化热仿真参数
        r_jc: 结-壳热阻 (°C/W)
        r_cs: 壳-散热器热阻 (°C/W)
        r_sa: 散热器-环境热阻 (°C/W)
        c_jc/c_cs/c_sa: 对应热容 (J/°C)
        """
        self.thermal_params = {
            'r_jc': r_jc, 'r_cs': r_cs, 'r_sa': r_sa,
            'c_jc': c_jc, 'c_cs': c_cs, 'c_sa': c_sa
        }
        self.R_total = r_jc + r_cs + r_sa
        self.C_total = 1 / (1/c_jc + 1/c_cs + 1/c_sa)
        self.tau = self.R_total * self.C_total
    
    def steady_state_temp(self, power_w, ambient_c=25):
        """计算稳态结温"""
        return ambient_c + power_w * self.R_total
    
    def transient_response(self, power_w, duration_s, dt=0.1, ambient_c=25):
        """模拟瞬态温度响应"""
        t = np.arange(0, duration_s, dt)
        T = ambient_c + power_w * self.R_total * (1 - np.exp(-t / self.tau))
        return t, T
    
    def multi_step_response(self, power_profile, ambient_c=25):
        """
        模拟多阶段负载的温度响应
        power_profile: [(power_w, duration_s), ...]
        """
        t_list = []
        T_list = []
        T_current = ambient_c
        
        for power_w, duration_s in power_profile:
            t_local = np.arange(0, duration_s, 0.1)
            # 每个阶段的瞬态响应
            T_local = ambient_c + power_w * self.R_total + (T_current - ambient_c - power_w * self.R_total) * np.exp(-t_local / self.tau)
            
            if len(t_list) == 0:
                t_offset = 0
            else:
                t_offset = t_list[-1][-1] + 0.1
            
            t_list.append(t_offset + t_local)
            T_list.append(T_local)
            T_current = T_local[-1]
        
        return np.concatenate(t_list), np.concatenate(T_list)
    
    def find_max_power(self, max_temp=95, ambient_c=25):
        """根据最大允许温度反算最大功耗"""
        return (max_temp - ambient_c) / self.R_total
    
    def analyze_cooling_efficiency(self, power_w, ambient_c=25):
        """分析散热效率"""
        T_s = self.steady_state_temp(power_w, ambient_c)
        temp_rise = T_s - ambient_c
        
        print(f"热分析报告 (功耗={power_w}W, 环境={ambient_c}°C):")
        print(f"  总热阻: {self.R_total:.3f} °C/W")
        print(f"  热时间常数: {self.tau:.1f} 秒")
        print(f"  稳态结温: {T_s:.1f}°C")
        print(f"  温升: {temp_rise:.1f}°C")
        print(f"  最大允许功耗(@95°C): {self.find_max_power(95, ambient_c):.1f}W")
        
        # 效率评级
        if self.R_total < 0.3:
            print("  散热效率: 优秀")
        elif self.R_total < 0.45:
            print("  散热效率: 良好")
        elif self.R_total < 0.6:
            print("  散热效率: 一般")
        else:
            print("  散热效率: 需要改进")
        
        return {'T_junction': T_s, 'temp_rise': temp_rise}

# 示例使用
if __name__ == "__main__":
    sim = GPUThermalSimulator(r_jc=0.12, r_cs=0.04, r_sa=0.28)
    sim.analyze_cooling_efficiency(power_w=250)
    
    print("\n多阶段负载仿真 (空闲 -> 游戏 -> 满载):")
    profile = [(50, 30), (200, 60), (300, 30), (50, 30)]
    t, T = sim.multi_step_response(profile)
    print(f"  最高温度: {max(T):.1f}°C")
    print(f"  最后温度: {T[-1]:.1f}°C")
```
