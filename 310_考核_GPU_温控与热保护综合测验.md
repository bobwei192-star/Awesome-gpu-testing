### 第 1 题
以下关于 AMD GPU 结温传感器的描述，哪项是正确的？

A. 结温传感器通常位于 GPU PCB 边缘，用于测量环境温度
B. 结温传感器基于半导体 PN 结的温度-电压特性工作
C. 结温传感器的典型精度为 ±5°C
D. 结温传感器只能在 GPU 空闲时读取

**正确答案**：B
**解析**：
- A 错误：结温传感器位于 GPU 芯片核心内部，临近计算单元
- B 正确：基于硅 PN 结的正向电压与温度的关系（约 -2 mV/°C）
- C 错误：结温传感器的典型精度为 ±1-2°C
- D 错误：结温传感器可以在任何状态下实时读取

### 第 2 题
GPU 热节流（Thermal Throttle）的主要触发条件是什么？

A. GPU 利用率达到 100%
B. GPU 温度超过设定的热节流阈值
C. GPU 功耗达到最大限制
D. GPU 显存使用达到 100%

**正确答案**：B
**解析**：
- A 错误：高利用率是发热的原因，但不是节流的直接触发条件
- B 正确：当温度达到设定阈值时，驱动会降低频率以减少发热
- C 错误：功耗限制是另一个独立的控制机制（Power Limit）
- D 错误：显存满载与热节流无直接关系

**资料来源**：

温度阈值迟滞（Hysteresis）机制的主要作用是什么？

A. 提高温度读数的精度
B. 避免在阈值附近频繁切换状态
C. 降低传感器的功耗

B. 热点传感器报告所有分布传感器中的最低温度

温度与频率-电压（V/F）曲线的关系是？

A. 高温下，相同频率需要更高电压

### 题目 1：温度监控脚本开发（15 分）

**任务描述**：
编写一个 Bash 脚本，实时监控 GPU 温度变化，当温度超过阈值时触发警报并记录相关信息。

**要求**：
1. 支持通过 hwmon、thermal_zone、rocm-smi 三种方式读取温度
2. 实现可配置的温度阈值（警告阈值和紧急阈值）
3. 当温度超过警告阈值时，打印警告信息并记录日志
4. 当温度超过紧急阈值时，打印紧急信息并执行清理动作
5. 输出温度历史记录到 CSV 文件

**评分标准**：
- 温度读取功能（4 分）
- 阈值检测与警报逻辑（4 分）
- 日志记录功能（3 分）
- 错误处理与鲁棒性（2 分）
- 代码注释与可读性（2 分）

**参考解答**：

```bash
#!/bin/bash
# gpu_temperature_monitor.sh
# GPU 温度监控脚本 - 考核参考实现

# ============= 配置区 =============
# 监控间隔（秒）
INTERVAL=2

# 温度阈值（°C）
WARNING_THRESHOLD=85
CRITICAL_THRESHOLD=95

# 日志文件
LOG_FILE="gpu_temp_monitor.log"
CSV_FILE="gpu_temp_history.csv"

# 警报命令（可自定义）
ALERT_CMD="echo"
# 紧急动作（如发送邮件、记录系统日志）
EMERGENCY_CMD="logger -t GPU_TEMP_ALERT"

# ============= 初始化 =============

echo "========================================"
echo "  GPU Temperature Monitor Started"
echo "========================================"
echo "Warning threshold: ${WARNING_THRESHOLD}°C"
echo "Critical threshold: ${CRITICAL_THRESHOLD}°C"
echo "Log file: $LOG_FILE"
echo "CSV file: $CSV_FILE"
echo "========================================"
echo ""

# 查找 GPU hwmon 路径
find_hwmon_path() {
    local hwmon_path=$(find /sys/class/drm/card0/device/hwmon/ -name "hwmon*" | head -1)
    if [ -n "$hwmon_path" ] && [ -d "$hwmon_path" ]; then
        echo "$hwmon_path"
    else
        echo ""
    fi
}

HWMON_PATH=$(find_hwmon_path)

# 查找 GPU thermal zone
find_thermal_zone() {
    for zone in /sys/class/thermal/thermal_zone*; do
        if [ -d "$zone" ]; then
            local zone_type=$(cat "$zone/type" 2>/dev/null)
            if echo "$zone_type" | grep -iq "amdgpu\|gpu"; then
                echo "$zone"
                return
            fi
        fi
    done
    echo ""
}

THERMAL_ZONE=$(find_thermal_zone)

# 初始化 CSV
if [ ! -f "$CSV_FILE" ]; then
    echo "Timestamp,Source,Temperature(°C),Status" > "$CSV_FILE"
fi

# ============= 温度读取函数 =============

# 方法 1: 通过 hwmon 读取
read_temp_hwmon() {
    if [ -z "$HWMON_PATH" ] || [ ! -d "$HWMON_PATH" ]; then
        echo ""
        return
    fi
    
    local temp_file=$(find "$HWMON_PATH" -name "temp1_input" 2>/dev/null | head -1)
    if [ -f "$temp_file" ]; then
        local temp_mc=$(cat "$temp_file" 2>/dev/null)
        if [ -n "$temp_mc" ] && [ "$temp_mc" -gt 0 ]; then
            echo $((temp_mc / 1000))
        else
            echo ""
        fi
    else
        echo ""
    fi
}

# 方法 2: 通过 thermal_zone 读取
read_temp_thermal() {
    if [ -z "$THERMAL_ZONE" ] || [ ! -d "$THERMAL_ZONE" ]; then
        echo ""
        return
    fi
    
    local temp=$(cat "$THERMAL_ZONE/temp" 2>/dev/null)
    if [ -n "$temp" ] && [ "$temp" -gt 0 ]; then
        echo $((temp / 1000))
    else
        echo ""
    fi
}

# 方法 3: 通过 rocm-smi 读取
read_temp_rocmsmi() {
    if ! command -v rocm-smi &> /dev/null; then
        echo ""
        return
    fi
    
    local temp_output=$(rocm-smi --showtemp 2>/dev/null)
    local temp=$(echo "$temp_output" | grep -i "temperature" | grep -v "critical" | head -1 | grep -oE '[0-9]+\.?[0-9]*' | head -1)
    if [ -n "$temp" ]; then
        # 转换为整数
        echo "$temp" | awk '{printf "%d\n", $1}'
    else
        echo ""
    fi
}

# 获取所有可用温度读数
get_all_temps() {
    local temps=()
    
    local temp_hwmon=$(read_temp_hwmon)
    if [ -n "$temp_hwmon" ]; then
        temps+=("hwmon:$temp_hwmon")
    fi
    
    local temp_thermal=$(read_temp_thermal)
    if [ -n "$temp_thermal" ]; then
        temps+=("thermal:$temp_thermal")
    fi
    
    local temp_rocm=$(read_temp_rocmsmi)
    if [ -n "$temp_rocm" ]; then
        temps+=("rocm-smi:$temp_rocm")
    fi
    
    echo "${temps[@]}"
}

# ============= 主监控循环 =============

count=0
in_warning=0
in_critical=0

trap 'echo -e "\n\n========================================"; echo "  Monitor stopped by user"; echo "========================================"; exit 0' SIGINT SIGTERM

while true; do
    timestamp=$(date "+%Y-%m-%d %H:%M:%S")
    count=$((count + 1))
    
    echo "[$timestamp] === Monitoring cycle #$count ==="
    
    # 获取所有温度读数
    temp_array=($(get_all_temps))
    
    if [ ${#temp_array[@]} -eq 0 ]; then
        echo "  ERROR: Could not read any temperature sensor"
        echo "[$timestamp] ERROR: Could not read any temperature sensor" >> "$LOG_FILE"
        sleep $INTERVAL
        continue
    fi
    
    # 检查每个温度读数
    max_temp=0
    max_source=""
    
    for temp_info in "${temp_array[@]}"; do
        source=${temp_info%%:*}
        temp=${temp_info##*:}
        
        echo "  $source: ${temp}°C"
        
        # 记录到 CSV
        echo "$timestamp,$source,$temp,Normal" >> "$CSV_FILE"
        
        # 更新最高温度
        if [ "$temp" -gt "$max_temp" ]; then
            max_temp=$temp
            max_source=$source
        fi
    done
    
    # 温度状态判断
    echo ""
    echo "  Highest: ${max_temp}°C (from $max_source)"
    echo "  Status: \c"
    
    if [ "$max_temp" -ge "$CRITICAL_THRESHOLD" ]; then
        echo -e "\033[31mCRITICAL!\033[0m"
        
        # 首次进入紧急状态
        if [ $in_critical -eq 0 ]; then
            echo "  >>> CRITICAL ALERT: Temperature reached ${max_temp}°C!"
            echo "[$timestamp] CRITICAL ALERT: Temperature ${max_temp}°C from $max_source" >> "$LOG_FILE"
            $EMERGENCY_CMD "GPU temperature critical: ${max_temp}°C"
            in_critical=1
            in_warning=1
        fi
        
    elif [ "$max_temp" -ge "$WARNING_THRESHOLD" ]; then
        echo -e "\033[33mWARNING\033[0m"
        
        # 首次进入警告状态
        if [ $in_warning -eq 0 ]; then
            echo "  >>> WARNING: Temperature reached ${max_temp}°C!"
            echo "[$timestamp] WARNING: Temperature ${max_temp}°C from $max_source" >> "$LOG_FILE"
            $ALERT_CMD "GPU temperature warning: ${max_temp}°C"
            in_warning=1
        fi
        
    else
        echo -e "\033[32mNormal\033[0m"
        
        # 从警告/紧急状态恢复
        if [ $in_critical -eq 1 ]; then
            echo "  >>> Recovered from CRITICAL state"
            echo "[$timestamp] RECOVERED: Temperature dropped to ${max_temp}°C" >> "$LOG_FILE"
            in_critical=0
        fi
        if [ $in_warning -eq 1 ]; then
            echo "  >>> Recovered from WARNING state"
            echo "[$timestamp] RECOVERED: Temperature dropped to ${max_temp}°C" >> "$LOG_FILE"
            in_warning=0
        fi
    fi
    
    echo ""
    sleep $INTERVAL
done
```
