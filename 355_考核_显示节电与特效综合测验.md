### 第一部分：显示节电技术（30分）

#### 1.1 PSR 技术核心要点

**PSR1 工作原理：**

PSR1 的关键特征是在静态画面时将面板切换到 Self-Refresh 模式，完全关闭 Main Link：

```
 ┌───────────┐     Main Link On      ┌───────────┐
 │   GPU     │ ────────────────────► │   Panel   │
 │  (Source) │     (视频数据流)       │  (Sink)   │
 └───────────┘                        └───────────┘
       │                                    │
       │ 画面静止检测                        │
       ▼                                    │
 ┌───────────┐                              │
 │ 进入 PSR  │◄──── 使用面板缓冲 ────────────┘
 └───────────┘
       │
       ▼
 ┌───────────┐     Main Link Off     ┌───────────┐
 │   GPU     │ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─► │   Panel   │
 │  (Source) │     (链路关闭)         │  (Sink)   │
 └───────────┘                       └───────────┘
       │                                    │
       │ 画面变化触发                        │
       ▼                                    │
 ┌───────────┐                              │
 │  退出 PSR │──── 重新发送整个帧 ────────────►
 └───────────┘
```

**PSR2 选择性更新：**

PSR2 通过 Selective Update (SU) 只传输变化的区域，链路保持活动但降速：

```
PSR2 帧结构：
┌─────────────────────────────────────────────┐
│  PSR2 SU 帧头                                │
│  ├── SU 区域数量: N                          │
│  ├── SU 区域 1: {x, y, w, h}               │
│  ├── SU 区域 2: {x, y, w, h}               │
│  ├── ...                                     │
│  └── SU 区域 N: {x, y, w, h}               │
├─────────────────────────────────────────────┤
│  更新数据（每个 SU 区域对应的实际像素数据）     │
│  ├── SU 区域 1 像素数据                      │
│  ├── SU 区域 2 像素数据                      │
│  └── ...                                     │
└─────────────────────────────────────────────┘
```

**关键 DPCD 寄存器地址：**

| 寄存器 | 地址 | 位域 | 描述 |
|--------|------|------|------|
| DPCD 0x070 | PSR_SUPPORT | [0] | PSR 支持位 |
| DPCD 0x071 | PSR_CAPS | [0] | PSR1 支持 |
| DPCD 0x071 | PSR_CAPS | [1] | PSR2 支持 |
| DPCD 0x071 | PSR_CAPS | [2] | SU 区域支持 |
| DPCD 0x080 | PSR_ENABLE | [2:0] | 启用配置 |
| DPCD 0x081 | PSR_STATUS | [2:0] | 当前状态 |
| DPCD 0x2000-0x2FFF | SU 区域描述符 | - | PSR2 SU 区域 |

**Panel Replay 特性：**

- 基于 ALPM (Alternate Low Power Mode)
- 不需要面板缓冲（Panel Buffer）
- 通过 `PSR_ERROR_SEL` 寄存器检测错误
- 兼容 PSR1/PSR2 的进入/退出机制

#### 2.1 光标 (Cursor) Plane

**三种 Plane 类型对比：**

| 特性 | PRIMARY | CURSOR | OVERLAY |
|------|---------|--------|---------|
| 用途 | 主帧缓冲 | 鼠标指针 | 视频/UI 叠加 |

| 透明度 | 无 | ARGB 支持 | 支持 |

### 第三部分：协同工作机制（20分）

#### 3.1 ABM + PSR 协同

```
ABM + PSR 协同调度时序图：

时间 │ ABM 事件                │ PSR 事件                │ 功耗状态
─────┼─────────────────────────┼─────────────────────────┼──────────
 t0  │ 画面静止检测             │                         │ Active
 t1  │ ABM Level 2 生效        │                         │ 节能 ~15%
 t2  │                         │ 进入 PSR1, Main Link 关闭│ 节能 ~35%
 t3  │ ABM 重新分析 APL         │ PSR1 保持               │ 节能 ~35%
 t4  │                         │ 鼠标移动 → 退出 PSR1     │ Active
 t5  │ ABM 保持 Level 2        │                         │ 节能 ~15%
 t6  │ 场景切换 (视频→Word)    │                         │ 节能 ~15%
 t7  │ ABM 提升到 Level 3      │                         │ 节能 ~30%
 t8  │                         │ 画面静止 → 进入 PSR1     │ 节能 ~50%
```

**协同节能效果叠加计算：**

```
总节能 = 1 - (1 - PSR_节省) × (1 - ABM_节省)

示例：
  PSR 节省: 20% (链路功耗占比)
  ABM 节省: 25% (背光功耗占比)
  总节能 = 1 - (1 - 0.20) × (1 - 0.25)
         = 1 - 0.80 × 0.75
         = 1 - 0.60
         = 40%

注意事项：
  - ABM 修改背光不影响 PSR 进入条件
  - PSR 进入/退出不影响 ABM 级别
  - 两者完全正交，可独立配置
```

### 第四部分：关键数据结构与 API（15分）

#### 4.1 DRM 原子 API 核心结构

```c
/* 原子提交的核心数据结构链 */
struct drm_mode_atomic {
    __u32 flags;              /* 原子提交标志 */
    __u32 count_objs;         /* 对象数量 */
    __u64 objs_ptr;           /* 对象 ID 数组 */
    __u64 count_props_ptr;    /* 各对象的属性数量 */
    __u64 props_ptr;          /* 属性 ID 数组 */
    __u64 prop_values_ptr;    /* 属性值数组 */
    __u64 user_data;          /* 用户数据 */
};

/* 原子提交流程 */
int atomic_commit(int fd, struct amdgpu_display_config *cfg) {
    drmModeAtomicReqPtr req = drmModeAtomicAlloc();
    if (!req) return -ENOMEM;

    /* 设置 CRTC */
    drmModeAtomicAddProperty(req, cfg->crtc_id,
                              cfg->props.mode_id, cfg->mode_blob_id);
    drmModeAtomicAddProperty(req, cfg->crtc_id,
                              cfg->props.active, 1);

    /* 设置 PRIMARY plane */
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.fb_id, cfg->fb_id);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.crtc_id, cfg->crtc_id);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.src_x, 0);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.src_y, 0);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.src_w, cfg->width << 16);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.src_h, cfg->height << 16);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.crtc_x, 0);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.crtc_y, 0);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.crtc_w, cfg->width);
    drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                              cfg->props.crtc_h, cfg->height);

    /* 可选：设置旋转 */
    if (cfg->rotation != DRM_MODE_ROTATE_0) {
        drmModeAtomicAddProperty(req, cfg->primary_plane_id,
                                  cfg->props.rotation, cfg->rotation);
    }

    /* 可选：Overlay plane */
    if (cfg->overlay_plane_id) {
        drmModeAtomicAddProperty(req, cfg->overlay_plane_id,
                                  cfg->props.fb_id, cfg->overlay_fb_id);
        drmModeAtomicAddProperty(req, cfg->overlay_plane_id,
                                  cfg->props.crtc_id, cfg->crtc_id);
        /* ... 其他属性 ... */
    }

    int ret = drmModeAtomicCommit(fd, req,
                   DRM_MODE_ATOMIC_ALLOW_MODESET, NULL);
    drmModeAtomicFree(req);
    return ret;
}
```

#### 4.2 AMDGPU DC 电源管理接口

```c
/* DC 电源状态管理 */
enum dc_power_state {
    DC_POWER_STATE_ON,           /* 完全供电 */
    DC_POWER_STATE_LIGHT_SLEEP,  /* 轻度休眠 (PSR 级别) */
    DC_POWER_STATE_DEEP_SLEEP,   /* 深度休眠 */
    DC_POWER_STATE_OFF,          /* 关闭 */
};

/* DC PSR 控制接口 */
struct dc_psr_config {
    bool enable;                 /* 启用 PSR */
```

### 简答题参考

**问题 1：PSR 进入和退出条件**

PSR 进入条件（至少 3 个）：
1. 画面静止（没有帧更新请求）超过预设阈值（通常 50-100ms）
2. 没有光标移动或光标更新
3. 没有 VRR 帧率变化
4. Panel 支持 PSR 且已启用
5. 没有 pending 的原子提交

PSR 退出条件（至少 3 个）：
1. 新帧提交（画面变化）
2. 光标移动
3. VRR 帧率变化
4. 模式设置（modeset）
5. 热插拔事件
6. 电源状态变化

DPCD 0x081 状态值：
- 0x00: SINK_IN_SELF_REFRESH_ACTIVE (PSR 激活)
- 0x01: SINK_IN_SELF_REFRESH_INACTIVE (未激活)
- 0x02: SINK_IN_DEEP_SLEEP (深度休眠)
- 0x03: SINK_IN_SLEEP (轻度休眠)
- 0x04: SINK_IN_WAKEUP (唤醒中)

**问题 2：旋转 90° 的映射关系**

旋转 90° 时，Viewport 的宽高与 Buffer 的宽高交换：

```
Buffer: 1920x1080 (原始帧缓冲)
Viewport 旋转后: 1080x1920 (输出)

原子提交配置：
src_x = 0                      (源 X 不变)
src_y = 0                      (源 Y 不变)
src_w = 1920 << 16 = 125829120 (源宽度 = 原始宽度)
src_h = 1080 << 16 = 70778880  (源高度 = 原始高度)

crtc_x = 0                     (目标 X = 0)
crtc_y = 0                     (目标 Y = 0)
crtc_w = 1080                  (目标宽度 = 原始高度 = 旋转后宽度)
crtc_h = 1920                  (目标高度 = 原始宽度 = 旋转后高度)

注意：drm_atomic_helper_check_plane_state() 会检查：
  src_x + src_w <= fb_width    (0 + 1920 <= 1920 ✓)
  src_y + src_h <= fb_height   (0 + 1080 <= 1080 ✓)
旋转 90° 时宽高交换由硬件自动处理
```

```c
// psr_diag.c - PSR 状态监控与诊断工具
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <unistd.h>
#include <fcntl.h>
#include <getopt.h>
#include <time.h>
#include <json-c/json.h>

#define PSR_STATE_PATH    "/sys/kernel/debug/dri/0/amdgpu_dm_psr_state"
#define PSR_CAPS_PATH     "/sys/kernel/debug/dri/0/amdgpu_dm_psr_capability"

/* PSR 状态枚举 */
enum psr_state {
    PSR_STATE_INACTIVE  = 0,
    PSR_STATE_ACTIVE    = 1,  /* PSR1 Active */
    PSR_STATE_PSR2      = 2,  /* PSR2 Active */
    PSR_STATE_REPLAY    = 3,  /* Panel Replay */
    PSR_STATE_UNKNOWN   = -1,
};

/* PSR 能力位解析 */
struct psr_caps {
    int psr1_supported;
    int psr2_supported;
    int su_region_supported;
    int panel_replay_supported;
    int alpm_supported;
};

static const char *psr_state_str(enum psr_state state) {
    switch (state) {
        case PSR_STATE_INACTIVE: return "PSR OFF";
        case PSR_STATE_ACTIVE:   return "PSR1 ACTIVE";
        case PSR_STATE_PSR2:     return "PSR2 ACTIVE";
        case PSR_STATE_REPLAY:   return "PANEL REPLAY";
        default:                 return "UNKNOWN";
    }
}

static int read_sysfs_value(const char *path) {
    char buf[64];
    int fd = open(path, O_RDONLY);
    if (fd < 0) return -1;
    int n = read(fd, buf, sizeof(buf) - 1);
    close(fd);
    if (n <= 0) return -1;
    buf[n] = '\0';
    return atoi(buf);
}

static enum psr_state read_psr_state(void) {
    int val = read_sysfs_value(PSR_STATE_PATH);
    if (val < 0) return PSR_STATE_UNKNOWN;
    return (enum psr_state)val;
}

static struct psr_caps read_psr_caps(void) {
    struct psr_caps caps = {0};
    int val = read_sysfs_value(PSR_CAPS_PATH);
    if (val < 0) return caps;

    caps.psr1_supported     = (val >> 0) & 1;
    caps.psr2_supported     = (val >> 1) & 1;
    caps.su_region_supported = (val >> 2) & 1;
    caps.panel_replay_supported = (val >> 3) & 1;
    caps.alpm_supported     = (val >> 4) & 1;

    return caps;
}

static void print_human_readable(void) {
    enum psr_state state = read_psr_state();
    struct psr_caps caps = read_psr_caps();

    printf("=== PSR 状态诊断 ===\n");
    printf("当前状态: %s\n", psr_state_str(state));

    printf("\n--- PSR 能力 ---\n");
    printf("PSR1 支持:         %s\n", caps.psr1_supported ? "✅" : "❌");
    printf("PSR2 支持:         %s\n", caps.psr2_supported ? "✅" : "❌");
    printf("SU 区域支持:       %s\n", caps.su_region_supported ? "✅" : "❌");
    printf("Panel Replay 支持: %s\n", caps.panel_replay_supported ? "✅" : "❌");
    printf("ALPM 支持:         %s\n", caps.alpm_supported ? "✅" : "❌");
}

static void print_json(void) {
    enum psr_state state = read_psr_state();
    struct psr_caps caps = read_psr_caps();

    struct json_object *root = json_object_new_object();
    json_object_object_add(root, "state",
        json_object_new_int(state));
    json_object_object_add(root, "state_str",
        json_object_new_string(psr_state_str(state)));
    json_object_object_add(root, "psr1",
        json_object_new_boolean(caps.psr1_supported));
    json_object_object_add(root, "psr2",
        json_object_new_boolean(caps.psr2_supported));
    json_object_object_add(root, "su_region",
        json_object_new_boolean(caps.su_region_supported));
    json_object_object_add(root, "panel_replay",
        json_object_new_boolean(caps.panel_replay_supported));
    json_object_object_add(root, "alpm",
        json_object_new_boolean(caps.alpm_supported));

    printf("%s\n", json_object_to_json_string_ext(root,
        JSON_C_TO_STRING_PRETTY));
    json_object_put(root);
}

static void monitor_mode(int interval_ms) {
    enum psr_state prev = PSR_STATE_UNKNOWN;
    int changes = 0;

    printf("PSR 连续监控模式 (间隔 %dms)\n", interval_ms);
    printf("时间\t\t状态\t变化\n");
    printf("----\t\t----\t----\n");

    while (1) {
        enum psr_state cur = read_psr_state();
        time_t now = time(NULL);
        struct tm *tm = localtime(&now);

        printf("%02d:%02d:%02d\t%s\t%s\n",
               tm->tm_hour, tm->tm_min, tm->tm_sec,
               psr_state_str(cur),
               (cur != prev && prev != PSR_STATE_UNKNOWN) ?
               "✦ 变化" : "—");

        if (cur != prev && prev != PSR_STATE_UNKNOWN) {
            changes++;
        }
        prev = cur;
        usleep(interval_ms * 1000);
    }
}

int main(int argc, char *argv[]) {
    int opt;
    int monitor = 0;
    int json_output = 0;
    int interval = 1000;

    while ((opt = getopt(argc, argv, "mji:")) != -1) {
        switch (opt) {
            case 'm': monitor = 1; break;
            case 'j': json_output = 1; break;
            case 'i': interval = atoi(optarg); break;
            default:
                fprintf(stderr, "用法: %s [-m] [-j] [-i 间隔ms]\n", argv[0]);
                return 1;
        }
    }

    if (monitor) {
        monitor_mode(interval);
    } else if (json_output) {
        print_json();
    } else {
        print_human_readable();
    }

    return 0;
}
```

```bash
#!/bin/bash
# abm_optimize.sh - ABM 级别自动调优脚本

ABM_DEBUGFS="/sys/kernel/debug/dri/0/amdgpu_abm_level"
LOG_FILE="/var/log/abm_optimize.log"
DAEMONIZE=0
QUERY=0

# 日志函数
log() {
    local msg="[$(date '+%Y-%m-%d %H:%M:%S')] $1"
    echo "$msg" | tee -a "$LOG_FILE"
}

# 检测前台应用类型
detect_app_type() {
    local active_window=""
    
    # 尝试多种方式获取前台窗口
    if command -v xdotool &>/dev/null; then
        active_window=$(xdotool getactivewindow getwindowname 2>/dev/null)
    elif command -v xprop &>/dev/null; then
        local id=$(xprop -root _NET_ACTIVE_WINDOW 2>/dev/null | awk '{print $5}')
        [ -n "$id" ] && active_window=$(xprop -id "$id" _NET_WM_NAME 2>/dev/null | \
            cut -d'"' -f2)
    fi

    # 根据窗口标题判断应用类型
    local app_type="unknown"
    local lower=$(echo "$active_window" | tr '[:upper:]' '[:lower:]')

    if echo "$lower" | grep -qE 'chrome|firefox|edge|browser|chromium'; then
        app_type="browser"
    elif echo "$lower" | grep -qE 'word|libreoffice|doc|pdf|evince'; then
        app_type="document"
    elif echo "$lower" | grep -qE 'code|terminal|term|vim|bash|ssh|idea|eclipse'; then
        app_type="ide"
    elif echo "$lower" | grep -qE 'vlc|mpv|player|video|youtube|netflix'; then
        app_type="video"
    elif echo "$lower" | grep -qE 'steam|game|wine|proton'; then
        app_type="game"
    fi

    echo "$app_type"
}

# 根据应用类型返回 ABM 级别
app_to_abm_level() {
    case "$1" in
        browser|document) echo 3 ;;
        ide|terminal)     echo 2 ;;
        video)            echo 1 ;;
        game)             echo 0 ;;
        *)               echo 2 ;;
    esac
}

# 设置 ABM 级别
set_abm_level() {
    local level=$1
    if [ ! -w "$ABM_DEBUGFS" ]; then
        log "ERROR: 无法写入 $ABM_DEBUGFS（需要 root 权限）"
        return 1
    fi
    echo "$level" > "$ABM_DEBUGFS" 2>/dev/null
    log "ABM 级别已设置为: $level"
    return 0
}

# 查询当前状态
query_status() {
    echo "=== ABM 优化器状态 ==="
    echo "当前 ABM 级别: $(cat $ABM_DEBUGFS 2>/dev/null || echo '不可用')"
    echo "前台应用类型: $(detect_app_type)"
    echo "推荐 ABM 级别: $(app_to_abm_level $(detect_app_type))"
    echo "日志文件: $LOG_FILE"
    echo "运行模式: $([ $DAEMONIZE -eq 1 ] && echo '守护进程' || echo '单次')"
}

# 主循环
main_loop() {
    log "ABM 优化器已启动 (PID: $$)"
    local prev_level=-1

    while true; do
        local app_type=$(detect_app_type)
        local recommended=$(app_to_abm_level "$app_type")
        local current=$(cat "$ABM_DEBUGFS" 2>/dev/null || echo "-1")

        if [ "$recommended" -ne "$current" ] && [ "$recommended" -ge 0 ]; then
            set_abm_level "$recommended"
            log "切换: $(basename $ABM_DEBUGFS): $current → $recommended (应用: $app_type)"
        fi

        [ $DAEMONIZE -eq 0 ] && break
        sleep 5
    done
}

# 参数解析
while getopts "ds" opt; do
    case $opt in
        d) DAEMONIZE=1 ;;
        s) QUERY=1 ;;
        *) echo "用法: $0 [-d] (守护进程) [-s] (状态查询)"; exit 1 ;;
    esac
done

if [ $QUERY -eq 1 ]; then
    query_status
elif [ $DAEMONIZE -eq 1 ]; then
    main_loop &
    echo "ABM 优化器已在后台启动 (PID: $!)"
else
    main_loop
fi
```

```c
// rot_overlay_test.c - 旋转与 Overlay 组合测试
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#include <fcntl.h>
#include <unistd.h>
#include <errno.h>
#include <xf86drm.h>
#include <xf86drmMode.h>

#define TEST_WIDTH  1920
#define TEST_HEIGHT 1080

struct drm_test_ctx {
    int fd;
    drmModeRes *res;
    drmModePlaneRes *plane_res;
    uint32_t crtc_id;
    uint32_t primary_plane_id;
    uint32_t overlay_plane_id;

    /* 属性 ID */
    struct {
        uint32_t crtc_id;
        uint32_t fb_id;
        uint32_t src_x, src_y, src_w, src_h;
        uint32_t crtc_x, crtc_y, crtc_w, crtc_h;
        uint32_t rotation;
        uint32_t zorder;
        uint32_t mode_id;
        uint32_t active;
    } props;

    /* FB 对象 */
    uint32_t primary_fb_id;
    uint32_t overlay_fb_id;
    uint32_t mode_blob_id;
};

static int find_plane_by_type(drmModePlaneRes *plane_res, uint32_t type) {
    for (uint32_t i = 0; i < plane_res->count_planes; i++) {
        drmModePlane *plane = drmModeGetPlane(
            plane_res->planes[i].fd, plane_res->planes[i].plane_id);
        if (plane && (plane->possible_crtcs & (1 << 0))) {
            if (plane->plane_type == type) {
                int id = plane->plane_id;
                drmModeFreePlane(plane);
                return id;
            }
        }
        drmModeFreePlane(plane);
    }
    return -1;
}

static uint32_t get_prop_id(int fd, uint32_t obj_id, uint32_t obj_type,
                             const char *name) {
    drmModeObjectProperties *props = drmModeObjectGetProperties(fd, obj_id, obj_type);
    if (!props) return 0;

    for (uint32_t i = 0; i < props->count_props; i++) {
        drmModePropertyRes *prop = drmModeGetProperty(fd, props->props[i]);
        if (prop && strcmp(prop->name, name) == 0) {
            uint32_t id = prop->prop_id;
            drmModeFreeProperty(prop);
            drmModeFreeObjectProperties(props);
            return id;
        }
        drmModeFreeProperty(prop);
    }
    drmModeFreeObjectProperties(props);
    return 0;
}

static int setup_drm(struct drm_test_ctx *ctx) {
    ctx->fd = open("/dev/dri/card0", O_RDWR);
    if (ctx->fd < 0) {
        perror("打开 card0");
        return -1;
    }

    ctx->res = drmModeGetResources(ctx->fd);
    if (!ctx->res) {
        perror("获取资源");
        return -1;
    }

    ctx->plane_res = drmModeGetPlaneResources(ctx->fd);
    if (!ctx->plane_res) {
        perror("获取 Plane 资源");
        return -1;
    }

    /* 使用第一个 CRTC */
    ctx->crtc_id = ctx->res->crtcs[0];

    /* 查找 PRIMARY 和 OVERLAY plane */
    ctx->primary_plane_id = find_plane_by_type(ctx->plane_res, DRM_PLANE_TYPE_PRIMARY);
    ctx->overlay_plane_id = find_plane_by_type(ctx->plane_res, DRM_PLANE_TYPE_OVERLAY);

    if (ctx->primary_plane_id < 0) {
        fprintf(stderr, "未找到 PRIMARY plane\n");
        return -1;
    }

    /* 获取属性 ID */
    ctx->props.crtc_id   = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "CRTC_ID");
    ctx->props.fb_id     = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "FB_ID");
    ctx->props.src_x     = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "SRC_X");
    ctx->props.src_y     = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "SRC_Y");
    ctx->props.src_w     = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "SRC_W");
    ctx->props.src_h     = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "SRC_H");
    ctx->props.crtc_x    = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "CRTC_X");
    ctx->props.crtc_y    = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "CRTC_Y");
    ctx->props.crtc_w    = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "CRTC_W");
    ctx->props.crtc_h    = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "CRTC_H");
    ctx->props.rotation  = get_prop_id(ctx->fd, ctx->primary_plane_id,
                             DRM_MODE_OBJECT_PLANE, "rotation");

    return 0;
}

static int test_rotation_90(struct drm_test_ctx *ctx) {
    int pass = 1;
    drmModeAtomicReqPtr req = drmModeAtomicAlloc();
    if (!req) return 0;

    printf("测试: PRIMARY Plane 旋转 90° + OVERLAY Plane 叠加\n");

    /* 配置 PRIMARY plane: 旋转 90° */
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.crtc_id, ctx->crtc_id);
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.fb_id, ctx->primary_fb_id);
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.src_x, 0);
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.src_y, 0);
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.src_w, TEST_WIDTH << 16);
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.src_h, TEST_HEIGHT << 16);
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.crtc_x, 0);
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.crtc_y, 0);
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.crtc_w, TEST_HEIGHT);  /* 旋转后宽度=原始高度 */
    drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                              ctx->props.crtc_h, TEST_WIDTH);   /* 旋转后高度=原始宽度 */

    /* 设置旋转 90° */
    if (ctx->props.rotation) {
        drmModeAtomicAddProperty(req, ctx->primary_plane_id,
                                  ctx->props.rotation,
                                  DRM_MODE_ROTATE_90);
        printf("  [配置] rotation = DRM_MODE_ROTATE_90\n");
    }

    /* 配置 OVERLAY plane（如果存在） */
    if (ctx->overlay_plane_id > 0) {
        printf("  [配置] OVERLAY plane ID: %d\n", ctx->overlay_plane_id);

        /* 在旋转后的右上角显示 Overlay */
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.crtc_id, ctx->crtc_id);
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.fb_id, ctx->overlay_fb_id);
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.src_x, 0);
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.src_y, 0);
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.src_w, 640 << 16);
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.src_h, 360 << 16);
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.crtc_x,
                                  TEST_HEIGHT - 640);  /* 右上角 */
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.crtc_y, 0);
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.crtc_w, 640);
        drmModeAtomicAddProperty(req, ctx->overlay_plane_id,
                                  ctx->props.crtc_h, 360);
    }

    int ret = drmModeAtomicCommit(ctx->fd, req,
                   DRM_MODE_ATOMIC_ALLOW_MODESET, NULL);
    drmModeAtomicFree(req);

    if (ret < 0) {
        fprintf(stderr, "  [失败] 原子提交错误: %s\n", strerror(-ret));
        pass = 0;
    } else {
        printf("  [成功] 原子提交成功 (rotation=90°, overlay=%s)\n",
               ctx->overlay_plane_id > 0 ? "已叠加" : "不可用");

        /* 验证提交后的配置 */
        drmModePlane *plane = drmModeGetPlane(ctx->fd, ctx->primary_plane_id);
        if (plane) {
            printf("  [验证] PRIMARY crtc=%d, fb=%d\n",
                   plane->crtc_id, plane->fb_id);
            drmModeFreePlane(plane);
        }
    }

    return pass;
}

static int cleanup_drm(struct drm_test_ctx *ctx) {
    if (ctx->primary_fb_id)
        drmModeRmFB(ctx->fd, ctx->primary_fb_id);
    if (ctx->overlay_fb_id)
        drmModeRmFB(ctx->fd, ctx->overlay_fb_id);
    if (ctx->mode_blob_id)
        drmModeDestroyPropertyBlob(ctx->fd, ctx->mode_blob_id);
    if (ctx->plane_res)
        drmModeFreePlaneResources(ctx->plane_res);
    if (ctx->res)
        drmModeFreeResources(ctx->res);
    if (ctx->fd >= 0)
        close(ctx->fd);
    return 0;
}

int main(int argc, char *argv[]) {
    struct drm_test_ctx ctx = {0};
    int passed = 0;
    int total = 1;

    printf("旋转与 Overlay 组合测试工具\n");
    printf("============================\n\n");

    if (setup_drm(&ctx) < 0) {
        fprintf(stderr, "DRM 初始化失败\n");
        return 1;
    }

    /* 执行测试 */
    printf("--- 测试场景 ---\n");
    printf("CRTC: %d\n", ctx.crtc_id);
    printf("PRIMARY Plane: %d\n", ctx.primary_plane_id);
    printf("OVERLAY Plane: %d\n", ctx.overlay_plane_id);
    printf("旋转: 90° (竖屏)\n");
    printf("OVERLAY 位置: 右上角 640x360\n\n");

    passed += test_rotation_90(&ctx);

    /* 输出结果 */
    printf("\n--- 测试结果 ---\n");
    printf("通过: %d/%d\n", passed, total);
    printf("结论: %s\n", passed == total ? "全部通过 ✓" : "部分失败 ✗");

    cleanup_drm(&ctx);
    return (passed == total) ? 0 : 1;
}
```

```python
#!/usr/bin/env python3
# power_save_verify.py - ABM + PSR 综合节能效果验证

import os
import time
import json
import subprocess
import argparse
import sys
from collections import OrderedDict

class PowerSaverVerifier:
    def __init__(self, duration=30, rapl_path=None):
        self.duration = duration
        self.rapl_path = rapl_path or self._detect_rapl()
        self.abm_debugfs = "/sys/kernel/debug/dri/0/amdgpu_abm_level"
        self.psr_debugfs = "/sys/kernel/debug/dri/0/amdgpu_dm_psr_state"
        self.results = OrderedDict()

    def _detect_rapl(self):
        """检测 RAPL 接口路径"""
        base = "/sys/class/powercap"
        if not os.path.exists(base):
            return None

        for name in os.listdir(base):
            path = os.path.join(base, name)
            if os.path.isdir(path) and "intel-rapl" in name.lower():
                return os.path.join(path, "energy_uj")
            # 检查子目录 (AMD CPUs may use different naming)
            for sub in os.listdir(path):
                subpath = os.path.join(path, sub)
                if os.path.isdir(subpath) and "psys" in sub:
                    return os.path.join(subpath, "energy_uj")
        return None

    def _read_rapl(self):
        """读取 RAPL 能量计数器"""
        if not self.rapl_path or not os.path.exists(self.rapl_path):
            return None
        with open(self.rapl_path, 'r') as f:
            return int(f.read().strip())

    def _read_powertop(self):
        """尝试通过 powertop 获取功耗"""
        try:
            output = subprocess.check_output(
                ["powertop", "--csv=/dev/stdout", "-t", str(self.duration)],
                stderr=subprocess.DEVNULL, timeout=self.duration + 5
            ).decode('utf-8', errors='replace')
            return output
        except (subprocess.CalledProcessError, FileNotFoundError, TimeoutError):
            return None

    def _estimate_power(self, energy_uj, elapsed_ms):
        """从能量 (uJ) 和时间 (ms) 估算功率 (W)"""
        if elapsed_ms <= 0:
            return 0
        return (energy_uj / 1_000_000) / (elapsed_ms / 1000)

    def _set_abm_level(self, level):
        """设置 ABM 级别"""
        try:
            with open(self.abm_debugfs, 'w') as f:
                f.write(str(level))
            return True
        except (PermissionError, FileNotFoundError):
            return False

    def _set_psr(self, enable):
        """启用/禁用 PSR（通过 sysfs）"""
        psr_paths = [
            "/sys/kernel/debug/dri/0/amdgpu_dm_psr_debug",
            "/sys/module/amdgpu/parameters/psr",
        ]
        for path in psr_paths:
            if os.path.exists(path):
                try:
                    with open(path, 'w') as f:
                        f.write('1' if enable else '0')
                    return True
                except (PermissionError, FileNotFoundError):
                    continue
            return False

    def run_test_combination(self, abm_level, psr_enable, test_name):
        """运行单次测试组合"""
        print(f"  测试: ABM={abm_level}, PSR={'On' if psr_enable else 'Off'} ({test_name})")

        if not self._set_abm_level(abm_level):
            print(f"    [警告] 无法设置 ABM Level {abm_level}")
        if not self._set_psr(psr_enable):
            print(f"    [警告] 无法{'启用' if psr_enable else '禁用'} PSR")

        time.sleep(2)  # 等待设置生效

        if self.rapl_path:
            energy_before = self._read_rapl()
            if energy_before is None:
                print("    [错误] 无法读取 RAPL")
                return None

            time.sleep(self.duration)

            energy_after = self._read_rapl()
            if energy_after is None:
                print("    [错误] 无法读取 RAPL")
                return None

            energy_uj = energy_after - energy_before
            power_w = self._estimate_power(energy_uj, self.duration * 1000)
            result = {
                "abm_level": abm_level,
                "psr_enabled": psr_enable,
                "test_name": test_name,
                "energy_uj": energy_uj,
                "duration_s": self.duration,
                "power_w": round(power_w, 3),
            }
            print(f"    [结果] 平均功耗: {power_w:.3f} W")
            return result
        else:
            print("    [跳过] 无 RAPL 接口，使用 powertop 估算")
            return self._estimate_with_powertop(test_name)

    def _estimate_with_powertop(self, test_name):
        """使用 powertop 估算功耗"""
        return {
            "abm_level": -1,
            "psr_enabled": False,
            "test_name": test_name,
            "energy_uj": 0,
            "duration_s": self.duration,
            "power_w": 0.0,
            "note": "使用 powertop 需要 root 权限"
        }

    def run_all_tests(self):
        """运行所有 8 种组合"""
        print("=" * 60)
        print("ABM + PSR 综合节能效果验证")
        print("=" * 60)

        scenarios = [
            ("文档浏览", 3, True),
            ("文档浏览", 3, False),
            ("编码工作", 2, True),
            ("编码工作", 2, False),
            ("视频播放", 1, True),
            ("视频播放", 1, False),
            ("游戏场景", 0, True),
            ("游戏场景", 0, False),
        ]

        for i, (name, abm, psr) in enumerate(scenarios, 1):
            print(f"\n[{i}/8] ", end="")
            result = self.run_test_combination(abm, psr, name)
            if result:
                self.results[name + f"_ABM{abm}_PSR{'On' if psr else 'Off'}"] = result

        return self.results

    def print_results_table(self):
        """以表格形式打印结果"""
        print("\n" + "=" * 60)
        print("测试结果汇总")
        print("=" * 60)

        print(f"{'场景':<20} {'ABM':<5} {'PSR':<6} {'功耗(W)':<10}")
        print("-" * 45)

        sorted_results = sorted(self.results.values(),
                                 key=lambda r: r["power_w"])

        best = sorted_results[0] if sorted_results else None

        for r in sorted_results:
            marker = " ★" if r == best else ""
            print(f"{r['test_name']:<20} {r['abm_level']:<5} "
                  f"{'On' if r['psr_enabled'] else 'Off':<6} "
                  f"{r['power_w']:<10.3f}{marker}")

        print("-" * 45)
        if best:
            print(f"★ 最优组合: ABM Level {best['abm_level']} + "
                  f"{'PSR On' if best['psr_enabled'] else 'PSR Off'}")

    def print_bar_chart(self):
        """打印 ASCII 柱状图"""
        print("\n\n能耗对比柱状图:")
        print("=" * 60)

        if not self.results:
            return

        max_power = max(r["power_w"] for r in self.results.values())
        if max_power == 0:
            print("  (无功耗数据)")
            return

        bar_max = 40
        sorted_results = sorted(self.results.values(),
                                 key=lambda r: r["power_w"])

        for r in sorted_results:
            bar_len = int((r["power_w"] / max_power) * bar_max)
            label = f"{r['test_name'][:12]} ABM{r['abm_level']} "
            label += f"{'POn' if r['psr_enabled'] else 'POff'}"
            bar = "█" * bar_len
            pct = (r["power_w"] / max_power) * 100
            print(f"  {label:<22} |{bar:<{bar_max}} {r['power_w']:.2f}W ({pct:.0f}%)")

    def save_results_json(self, filepath="power_save_results.json"):
        """保存结果到 JSON 文件"""
        data = {
            "test_date": time.strftime("%Y-%m-%d %H:%M:%S"),
            "duration_s": self.duration,
            "results": list(self.results.values()),
        }
        with open(filepath, "w") as f:
            json.dump(data, f, indent=2)
        print(f"\n结果已保存到: {filepath}")


def main():
    parser = argparse.ArgumentParser(
        description="ABM + PSR 综合节能效果验证工具")
    parser.add_argument("-d", "--duration", type=int, default=30,
                        help="每种组合测试时长（秒，默认30）")
    parser.add_argument("-o", "--output", type=str,
                        default="power_save_results.json",
                        help="输出 JSON 文件路径")
    parser.add_argument("--rapl-path", type=str, default=None,
                        help="RAPL 能量计数器路径（自动检测）")
    args = parser.parse_args()
```

### 案例分析参考答案

#### 案例一：笔记本合盖后无法恢复显示

**1. PSR 状态机中 SLEEP 到 WAKEUP 的转换条件**

PSR 状态机转换：
```
SLEEP (0x03): 面板处于轻度休眠，Main Link 关闭
    │
    │ 触发条件：
    │ 1. 新帧提交（画面变化）
    │ 2. 光标移动
    │ 3. VRR 帧率变化
    │ 4. 显示器唤醒信号
    ▼
WAKEUP (0x04): 面板正在唤醒，需要重新 Training Link
    │
    │ 转换条件：
    │ 1. Link Training 成功
    │ 2. 面板准备就绪
    ▼
ACTIVE (0x00): PSR 正常激活
```

关键转换条件：
- **SLEEP → WAKEUP**: 需要 GPU 通过 AUX 通道发送 WAKEUP 命令，面板响应后进入
- **WAKEUP → ACTIVE**: 需要 Link Training 成功（包括 Main Link 和 AUX 通道）

**2. "link training failed" 的可能原因**

1. **物理连接问题**：合盖休眠时 eDP 连接器可能接触不良
2. **固件 Bug**：面板固件在休眠后无法正确响应 Link Training 请求
3. **时序问题**：GPU 发起 WAKEUP 到 Link Training 之间的延迟不足
4. **电源状态不一致**：面板已进入 Deep Sleep (0x02) 但 GPU 尝试以 SLEEP (0x03) 状态唤醒
5. **BIOS 冲突**：BIOS 在休眠/恢复过程中重置了 eDP 控制器状态

**3. 解决方案**

**方案 A：内核参数禁用 PSR**
```
在 /etc/default/grub 中设置：
GRUB_CMDLINE_LINUX_DEFAULT="amdgpu.dcpsr=0"
然后 update-grub 并重启

影响：禁用 PSR 会略微增加功耗，但确保兼容性
```

**方案 B：udev 规则在休眠前禁用 PSR**
```bash
# /usr/lib/systemd/system-sleep/psr-fix.sh
#!/bin/bash
case "$1" in
    pre)
        echo 0 > /sys/kernel/debug/dri/0/amdgpu_dm_psr_debug 2>/dev/null
        ;;
    post)
        sleep 2
        echo 1 > /sys/kernel/debug/dri/0/amdgpu_dm_psr_debug 2>/dev/null
        ;;
esac
```

**方案 C：更新面板固件或内核**
- 检查厂商是否发布面板固件更新
- 更新到包含 PSR 恢复修复的新内核版本

**4. 验证测试步骤**

1. 应用修复（禁用 PSR / 添加 udev 规则 / 更新固件）
2. 重启系统
3. 合盖 → 等待 10 秒 → 开盖
4. 检查显示是否正常恢复
5. 检查 dmesg 日志：
   ```bash
   dmesg | grep -E 'PSR|link training|display'
   ```
6. 确认无 "panel refused to exit sleep mode" 错误
7. 检查 PSR 状态：
   ```bash
   cat /sys/kernel/debug/dri/0/amdgpu_dm_psr_state
   ```
8. 如果使用方案 B/C，额外验证 PSR 在正常使用中仍可工作

#### 案例二：视频播放时 ABM 导致亮度波动

**1. ABM 增益变化与感知亮度的关系**

视频内容动态变化（场景切换）导致 APL 剧烈波动 → ABM 增益随之调整 → Slew Rate 虽然限速但仍有 10% 的增益变化 → 人眼可感知的亮度波动

感知亮度变化公式：
```
Δ感知亮度 ≈ Δ增益 × (1 - 背光起始水平)
当增益从 95% → 85% (下降 10%)时：
  背光从 95% → 85%，感知亮度下降约 10%
  在暗场景更明显（韦伯-费希纳定律）
```

**2. 视频播放场景下 ABM 不稳定的原因**

- **APL 波动大**：视频场景切换（暗→亮→暗）导致 APL 在 15%-65% 间跳变
- **Overlay 硬件合成**：硬件 Overlay 单独更新，ABM 统计引擎可能无法正确处理合成后的画面
- **缺乏场景感知**：ABM 无法区分"电影场景切换"是内容本身还是需要稳定背光的场景
- **Slew Rate 时间不足**：默认 Slew Rate 太快，无法消除电影场景切换的亮度突变
- **HDR 降级检测缺失**：虽然检测到 HDR inactive，但 SDR 视频也有高动态范围

**3. 视频播放的 ABM 优化配置**

```bash
#!/bin/bash
# optimize-abm-for-video.sh

# 为视频播放优化的 ABM 配置
# 1. 降低到 Level 1（轻度节能，变化更平缓）
echo 1 > /sys/kernel/debug/dri/0/amdgpu_abm_level

# 2. 增大 Slew Rate 时间（使过渡更平滑）
# 注：Slew Rate 可通过内核参数或 debugfs 调整
echo 10 > /sys/kernel/debug/dri/0/amdgpu_abm_slew_rate 2>/dev/null || true

# 3. 增大 APL 统计窗口（减少帧间波动）
echo 8 > /sys/kernel/debug/dri/0/amdgpu_abm_apl_window 2>/dev/null || true

echo "视频播放 ABM 优化已应用"
echo "ABM Level: 1 (轻度节能)"
echo "Slew Rate: 10 (平滑过渡)"
```

**4. udev 规则 + 检测脚本**

```bash
#!/bin/bash
# /usr/local/bin/video_abm_detect.sh

# 检测视频播放进程并调整 ABM
VIDEO_PLAYERS="vlc mpv mplayer ffplay kodi"

while true; do
    video_running=false
    for player in $VIDEO_PLAYERS; do
        if pgrep -x "$player" > /dev/null 2>&1; then
            video_running=true
            break
        fi
    done

    current_abm=$(cat /sys/kernel/debug/dri/0/amdgpu_abm_level 2>/dev/null || echo "0")

    if [ "$video_running" = true ] && [ "$current_abm" -gt 1 ]; then
        echo 1 > /sys/kernel/debug/dri/0/amdgpu_abm_level
        logger "ABM 降级到 Level 1 (视频播放检测)"
    elif [ "$video_running" = false ] && [ "$current_abm" -eq 1 ]; then
        # 视频结束后恢复到默认 Level 2
        echo 2 > /sys/kernel/debug/dri/0/amdgpu_abm_level
        logger "ABM 恢复到 Level 2 (视频结束)"
    fi

    sleep 10
done
```

## 今日小结

### 核心概念回顾

本次综合测验涵盖了显示节电技术（PSR/ABM）和显示特效技术（光标/Overlay/旋转/缩放）两大领域，以及它们的协同工作机制。

**显示节电技术要点：**
- PSR1 关闭 Main Link 实现节电，PSR2 通过 Selective Update 实现部分更新
- Panel Replay 基于 ALPM，不需要面板缓冲
- ABM 通过内容分析动态调整背光，Level 0-3 对应 0%-38% 的背光节省
- ABM + PSR 协同节能效果可叠加

**显示特效技术要点：**
- CURSOR plane 支持硬件光标，64x64/128x128 ARGB 格式
- OVERLAY plane 实现硬件合成，降低内存带宽
- Rotation 6 种组合（4 旋转 × 反射修饰）
- Scaling 4 种模式（FULL/ASPECT/CENTER/FULL_ASPECT）

### 常见错误总结

1. 混淆 PSR1 和 PSR2 的链路状态（关闭 vs 降速）
2. 误认为 ABM 节能效果与内容无关
3. 错误理解旋转 90° 的 Buffer 映射（宽高交换）
4. 误认为光标在竖屏模式下需要旋转图像
5. 忽略 ABM 在 HDR 模式下的自动降级行为
