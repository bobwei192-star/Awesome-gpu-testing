### 概念原理

#### 多 GPU 拓扑结构

多 GPU 系统的常见拓扑结构：

```
拓扑 1: PCIe Root Complex 直连
┌─────────────────────────────────────────────────────────┐
│                    CPU (单 socket)                       │
│                                                         │
│  PCIe Root                                PCIe Root     │
│  ┌──────────┐  ┌──────────┐    ┌──────────┐  ┌────────┐│
│  │   x16    │  │   x16    │    │   x16    │  │  x16   ││
│  └────┬─────┘  └────┬─────┘    └────┬─────┘  └───┬────┘│
│       │              │              │             │      │
│    ┌──┴──┐       ┌──┴──┐        ┌──┴──┐      ┌──┴──┐  │
│    │GPU 0│       │GPU 1│        │GPU 2│      │GPU 3│  │
│    └─────┘       └─────┘        └─────┘      └─────┘  │
│       │              │              │             │      │
│  ┌────┴──────────────┴──────────────┴─────────────┴──┐  │
│  │                DDR4/DDR5 Memory                    │  │
│  └────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────┘
特点: 延迟低，带宽受限 (所有 GPU 共享 CPU 内存通道)

拓扑 2: PCIe Switch 连接
┌─────────────────────────────────────────────────────────┐
│              CPU (多 socket / EPYC)                     │
│                         │                               │
│                    PCIe Switch (PLX/PEX)                │
│                     ┌──────┴──────┐                    │
│                     │              │                    │
│            ┌────────┴──────┐  ┌───┴────────┐          │
│            │  x16          │  │  x16         │          │
│         ┌──┴──┐  ┌──┴──┐  │  │  ┌──┴──┐  ┌──┴──┐    │
│         │GPU 0│  │GPU 1│  │  │  │GPU 2│  │GPU 3│    │
│         └─────┘  └─────┘  │  │  └─────┘  └─────┘    │
│                           │  │                        │
│                  更多 Switch                          │
└─────────────────────────────────────────────────────────┘
特点: 扩展性强，跨 Switch 延迟较高

拓扑 3: NVLink / XGMI 直接互联
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  ┌──────────┐           XGMI          ┌──────────┐     │
│  │  GPU 0   │◄═══════════════════════►│  GPU 1   │     │
│  │          │                         │          │     │
│  │  HBM2e   │                         │  HBM2e   │     │
│  │  64GB    │                         │  64GB    │     │
│  └────┬─────┘                         └────┬─────┘     │
│       │                                    │           │
│  ┌────┴─────┐                         ┌────┴─────┐     │
│  │  GPU 2   │◄═══════════════════════►│  GPU 3   │     │
│  └──────────┘                         └──────────┘     │
│                                                         │
│  XGMI 带宽: ~200 GB/s (单向) per link                  │
│  PCIe 带宽: ~32 GB/s (x16 Gen4)                       │
└─────────────────────────────────────────────────────────┘
```

#### PCIe 带宽与多 GPU 通信

```c
// PCIe 带宽计算
// Gen4 x16: 16 GT/s per lane, 128b/130b 编码
// 每通道: 16 * 128/130 = 15.754 GB/s
// x16: 15.754 * 16 = 252 GB/s (双向)
// 单向: ~32 GB/s (实际可用)

struct pcie_bandwidth {
    int    generation;    // 3=Gen3, 4=Gen4, 5=Gen5
    int    lanes;         // 通道数 (x1, x4, x8, x16)
    double encoding;      // 编码效率
    double bw_gbps;       // 单向带宽 (GB/s)
    
    // 实际有效带宽 (考虑协议开销)
    double effective;
};

// 常见配置的带宽对比
// Gen3 x16:  ~16 GB/s  (每通道 1 GB/s)
// Gen4 x16:  ~32 GB/s  (每通道 2 GB/s)
// Gen5 x16:  ~64 GB/s  (每通道 4 GB/s)
// XGMI 1.0:  ~84 GB/s  (每链路)
// XGMI 2.0:  ~168 GB/s (每链路)
// NVLink 4:  ~900 GB/s (18 条链路)
```

#### 工作负载分配模型

```
多 GPU 工作分配模型：

1. 数据并行 (Data Parallelism)
┌─────────────────────────────────────────────────────────┐
│                    训练数据                              │
│                        │                                │
│               ┌────────┴────────┐                       │
│               │                 │                       │
│          ┌────┴────┐      ┌────┴────┐                  │
│          │ 分片 0   │      │ 分片 1   │    ...          │
│          └────┬────┘      └────┬────┘                  │
│               │                 │                       │
│          ┌────┴────┐      ┌────┴────┐                  │
│          │ GPU 0   │      │ GPU 1   │    ...           │
│          │ 梯度计算 │      │ 梯度计算 │                  │
│          └────┬────┘      └────┬────┘                  │
│               │______梯度 All-Reduce______│              │
│                              │                          │
│                         ┌────┴────┐                     │
│                         │ 参数更新 │                     │
│                         └─────────┘                     │
└─────────────────────────────────────────────────────────┘

2. 模型并行 (Model Parallelism)
┌─────────────────────────────────────────────────────────┐
│  输入                                                      │
│    │                                                     │
│  ┌─┴──────────┐                                          │
│  │  Layer 1-3  │ ← GPU 0                                 │
│  └──────┬──────┘                                          │
│         │ P2P 通信                                        │
│  ┌──────┴──────┐                                          │
│  │  Layer 4-6  │ ← GPU 1                                 │
│  └──────┬──────┘                                          │
│         │ P2P 通信                                        │
│  ┌──────┴──────┐                                          │
│  │  Layer 7-10 │ ← GPU 2                                 │
│  └──────┬──────┘                                          │
│         │                                                 │
│     输出                                                  │
└─────────────────────────────────────────────────────────┘

3. 流水线并行 (Pipeline Parallelism)
┌─────────────────────────────────────────────────────────┐
│  Microbatch 0 → Microbatch 1 → Microbatch 2 → ...      │
│                                                        │
│  GPU 0: ┌─────┐  ┌─────┐  ┌─────┐                     │
│         │ Fwd │  │ Bwd │  │ Fwd │    ...              │
│         └──┬──┘  └──┬──┘  └──┬──┘                     │
│            │        │        │                         │
│  GPU 1: ┌──┴──┐  ┌──┴──┐  ┌──┴──┐                    │
│         │ Fwd │  │ Bwd │  │ Fwd │    ...              │
│         └──┬──┘  └──┬──┘  └──┬──┘                     │
│            │        │        │                         │
│  GPU 2: ┌──┴──┐  ┌──┴──┐  ┌──┴──┐                    │
│         │ Fwd │  │ Bwd │  │ Fwd │    ...              │
│         └─────┘  └─────┘  └─────┘                     │
└─────────────────────────────────────────────────────────┘

4. 混合并行 (Hybrid Parallelism)
┌─────────────────────────────────────────────────────────┐
│  数据并行组间 + 模型并行组内                              │
│                                                        │
│  Group 0: GPU 0-3 (模型并行)                            │
│  Group 1: GPU 4-7 (模型并行)                            │
│  Group 0 → Group 1 (数据并行 all-reduce)               │
└─────────────────────────────────────────────────────────┘
```

#### AMDGPU 多 GPU 设备管理

```c
// AMDGPU 多 GPU 设备管理 (drivers/gpu/drm/amd/amdgpu/amdgpu_device.c)

// 多 GPU 设备探测
struct amdgpu_device_list {
    struct list_head        list;           // 设备链表
    struct amdgpu_device   *adev;           // 设备指针
    int                     device_index;   // 设备索引
    bool                    is_primary;     // 是否主设备
};

// 多 GPU XGMI 拓扑
struct amdgpu_xgmi_topology {
    uint64_t                node_id;        // XGMI 节点 ID
    uint8_t                 num_nodes;      // 节点数量
    struct amdgpu_xgmi_link *links;         // XGMI 链路
    int                     num_links;
    struct list_head        top_list;       // 拓扑链表
};

// XGMI 链路信息
struct amdgpu_xgmi_link {
    uint64_t                peer_node_id;   // 对端节点
    enum amdgpu_xgmi_link_type link_type;   // 链路类型
    uint32_t                bandwidth;      // 带宽 (MB/s)
    bool                    enabled;        // 是否启用
};

// P2P (Peer-to-Peer) 访问支持
// 同一 XGMI 域内的 GPU 可以直接访问对方显存
// 不同 XGMI 域或非 XGMI 需要通过 PCIe DMA
bool amdgpu_device_peer_enable(struct amdgpu_device *adev,
                                struct amdgpu_device *peer_adev)
{
    // 1. 检查 XGMI 连接
    if (amdgpu_xgmi_same_hive(adev, peer_adev)) {
        // XGMI P2P: 直接映射显存
        if (amdgpu_xgmi_add_device(adev, peer_adev) == 0)
            return true;
    }
    
    // 2. PCIe P2P: 通过 BAR 映射
    if (pci_p2pdma_distance(adev->pdev, peer_adev->pdev, false) >= 0) {
        // 配置 PCIe P2P DMA
        if (pci_enable_p2p_dma(adev->pdev, peer_adev->pdev) == 0)
            return true;
    }
    
    // 3. 回退: 通过 CPU 中转
    return false;  // 需要 CPU 做中间缓冲
}
```

### 实践操作

#### 多 GPU 拓扑检测

```bash
#!/bin/bash
# 文件名: multi_gpu_topology.sh
# 用途: 检测多 GPU 系统拓扑

echo "=============================================="
echo "  多 GPU 系统拓扑检测工具"
echo "=============================================="
echo ""

detect_gpu_devices() {
    echo "[1/5] GPU 设备列表"
    echo "------------------"
    
    lspci -D | grep -i "VGA\|Display\|3D" | while read line; do
        BDF=$(echo ${line} | awk '{print $1}')
        NAME=$(echo ${line} | cut -d' ' -f2-)
        DRIVER=$(readlink /sys/bus/pci/devices/${BDF}/driver 2>/dev/null | xargs basename 2>/dev/null || echo "无")
        echo "  ${BDF}: ${NAME}"
        echo "        驱动: ${DRIVER}"
    done
}

detect_pcie_topology() {
    echo ""
    echo "[2/5] PCIe 拓扑"
    echo "----------------"
    
    # 使用 lspci -t 显示树形拓扑
    echo "  PCIe 树形拓扑:"
    lspci -t -vv 2>/dev/null | grep -i "AMD\|ATI" | head -10 || \
    lspci -t 2>/dev/null | while read line; do
        if echo "${line}" | grep -q "\[03\]\|[VGA]"; then
            echo "    ${line}"
        fi
    done
    
    echo ""
    echo "  GPU PCIe 连接详情:"
    for gpu_bdf in $(lspci -D | grep -i "VGA\|Display\|3D" | grep "AMD\|ATI" | awk '{print $1}'); do
        echo "  GPU ${gpu_bdf}:"
        
        # PCIe 带宽
        lspci -vvvs ${gpu_bdf} | grep -i "Speed\|Width\|Lnk" | while read line; do
            echo "    ${line}"
        done
        
        # NUMA 节点
        NUMA_NODE=$(cat /sys/bus/pci/devices/${gpu_bdf}/numa_node 2>/dev/null)
        echo "    NUMA Node: ${NUMA_NODE:-未知}"
        
        # IOMMU 分组
        for group in /sys/kernel/iommu_groups/*; do
            if [ -e "${group}/devices/${gpu_bdf}" ]; then
                echo "    IOMMU Group: $(basename ${group})"
                break
            fi
        done
        
        echo ""
    done
}

detect_xgmi_topology() {
    echo "[3/5] XGMI 拓扑"
    echo "---------------"
    
    # 方法 1: 通过 amdgpu debugfs
    if [ -d /sys/kernel/debug/dri ]; then
        for gpu_dir in /sys/kernel/debug/dri/*; do
            if [ -f "${gpu_dir}/amdgpu_xgmi_map" ]; then
                echo "  XGMI 映射:"
                cat "${gpu_dir}/amdgpu_xgmi_map" | while read line; do
                    echo "    ${line}"
                done
            fi
        done
    fi
    
    # 方法 2: 通过 ROCm
    if command -v rocminfo &> /dev/null; then
        echo ""
        echo "  ROCm 拓扑:"
        rocminfo | grep -A5 "XGMI\|Link\|Connection" | while read line; do
            echo "    ${line}"
        done
    fi
    
    # 方法 3: 检查 XGMI 设备
    if [ -f /sys/kernel/debug/dri/0/amdgpu_xgmi_info ]; then
        echo ""
        echo "  XGMI 详细信息:"
        cat /sys/kernel/debug/dri/0/amdgpu_xgmi_info
    fi
}

detect_p2p_connectivity() {
    echo ""
    echo "[4/5] P2P 连接性"
    echo "-----------------"
    
    local gpu_list=$(lspci -D | grep -i "VGA\|Display\|3D" | grep "AMD\|ATI" | awk '{print $1}')
    local gpu_array=(${gpu_list})
    local num_gpus=${#gpu_array[@]}
    
    if [ ${num_gpus} -lt 2 ]; then
        echo "  仅检测到 1 块 GPU，跳过 P2P 测试"
        return
    fi
    
    echo "  P2P 连接矩阵 (GPU → GPU 直接访问):"
    echo ""
    echo -n "        "
    for ((i=0; i<${num_gpus}; i++)); do
        printf "GPU%-8d" ${i}
    done
    echo ""
    
    for ((i=0; i<${num_gpus}; i++)); do
        printf "GPU%-4d" ${i}
        for ((j=0; j<${num_gpus}; j++)); do
            if [ ${i} -eq ${j} ]; then
                echo -n "  self    "
            else
                # 检查 XGMI 连接
                if [ -f "/sys/kernel/debug/dri/${i}/amdgpu_xgmi_map" ]; then
                    if grep -q "peer ${j}" "/sys/kernel/debug/dri/${i}/amdgpu_xgmi_map" 2>/dev/null; then
                        echo -n "  XGMI    "
                        continue
                    fi
                fi
                
                # 检查 PCIe P2P
                # 模拟检查 (实际需要 P2P API)
                echo -n "  PCIe   "
            fi
        done
        echo ""
    done
}

detect_memory_topology() {
    echo ""
    echo "[5/5] 显存拓扑"
    echo "--------------"
    
    for gpu_dir in /sys/class/drm/card?; do
        if [ -f "${gpu_dir}/device/mem_info_vram_total" ]; then
            CARD=$(basename ${gpu_dir})
            VRAM_TOTAL=$(cat ${gpu_dir}/device/mem_info_vram_total 2>/dev/null)
            VRAM_USED=$(cat ${gpu_dir}/device/mem_info_vram_used 2>/dev/null)
            VRAM_FREE=$((VRAM_TOTAL - VRAM_USED))
            
            # 转换为 GB
            VRAM_GB=$(echo "scale=1; ${VRAM_TOTAL} / 1073741824" | bc)
            USED_GB=$(echo "scale=1; ${VRAM_USED} / 1073741824" | bc)
            FREE_GB=$(echo "scale=1; ${VRAM_FREE} / 1073741824" | bc)
            
            echo "  ${CARD}: ${VRAM_GB}GB (已用 ${USED_GB}GB, 空闲 ${FREE_GB}GB)"
            
            # 显存类型
            if [ -f "${gpu_dir}/device/mem_info_vram_type" ]; then
                echo "         类型: $(cat ${gpu_dir}/device/mem_info_vram_type)"
            fi
        fi
    done
}

# 主流程
detect_gpu_devices
detect_pcie_topology
detect_xgmi_topology
detect_p2p_connectivity
detect_memory_topology

echo ""
echo "=============================================="
echo "  检测完成"
echo "=============================================="
```

#### 多 GPU 性能测试

```python
#!/usr/bin/env python3
# 多 GPU 性能测试框架

import subprocess
import json
import time
import threading
import statistics
from typing import Dict, List, Tuple

class MultiGPUBenchmark:
    """多 GPU 性能测试框架"""
    
    def __init__(self, num_gpus: int = None):
        self.num_gpus = self._detect_gpus() if num_gpus is None else num_gpus
        self.results = {}
    
    def _detect_gpus(self) -> int:
        """检测系统 GPU 数量"""
        try:
            result = subprocess.run(
                ["lspci", "-D"],
                capture_output=True, text=True
            )
            gpu_count = sum(
                1 for line in result.stdout.split('\n')
                if 'VGA' in line or 'Display' in line or '3D' in line
            )
            return max(gpu_count, 1)
        except:
            return 1
    
    def measure_p2p_bandwidth(self, src_gpu: int, dst_gpu: int,
                              size_mb: int = 256) -> float:
        """测量 GPU P2P 带宽 (MB/s)"""
        # 实际实现需要使用 ROCm 或 CUDA API
        # 这里使用 rocm-bandwidth-test 工具
        try:
            result = subprocess.run([
                "rocm-bandwidth-test",
                "-p", str(src_gpu),
                "-q", str(dst_gpu),
                "-s", str(size_mb),
                "-t", "p2p"
            ], capture_output=True, text=True, timeout=30)
            
            # 解析输出中的带宽数据
            for line in result.stdout.split('\n'):
                if 'Bandwidth' in line or 'bandwidth' in line:
                    # 提取带宽数值 (MB/s)
                    import re
                    match = re.search(r'(\d+\.?\d*)\s*GB/s', line)
                    if match:
                        return float(match.group(1)) * 1024
                    match = re.search(r'(\d+\.?\d*)\s*MB/s', line)
                    if match:
                        return float(match.group(1))
            
            # 使用 rocprof 测量
            result = subprocess.run([
                "rocprof", "--stats",
                "--papi", f"GPU:{src_gpu},P2P_BANDWIDTH:{dst_gpu}",
                "--", "sleep", "1"
            ], capture_output=True, text=True, timeout=30)
            
            return None
            
        except (subprocess.TimeoutExpired, FileNotFoundError):
            return None
    
    def measure_all_p2p(self) -> Dict[Tuple[int, int], float]:
        """测量所有 GPU 对的 P2P 带宽"""
        p2p_results = {}
        
        print("测量 P2P 带宽 (GB/s):")
        print(f"{'':8}", end="")
        for j in range(self.num_gpus):
            print(f"GPU{j:<8}", end="")
        print()
        
        for i in range(self.num_gpus):
            print(f"GPU{i:<4}", end="")
            for j in range(self.num_gpus):
                if i == j:
                    print(f"{'self':8}", end="")
                else:
                    bw = self.measure_p2p_bandwidth(i, j)
                    if bw is not None:
                        bw_gbs = bw / 1024
                        p2p_results[(i, j)] = bw
                        print(f"{bw_gbs:<8.1f}", end="")
                    else:
                        print(f"{'N/A':8}", end="")
            print()
        
        return p2p_results
    
    def run_all_reduce_benchmark(self, size_mb: int = 128) -> Dict:
        """执行 All-Reduce 带宽测试"""
        results = {}
        
        print(f"\nAll-Reduce 测试 (数据大小: {size_mb}MB):")
        
        for n in [2, 4, 8]:
            if n > self.num_gpus:
                break
            
            try:
                result = subprocess.run([
                    "rocm-bandwidth-test",
                    "-s", str(size_mb),
                    "-n", str(n),
                    "-t", "all-reduce"
                ], capture_output=True, text=True, timeout=60)
                
                # 解析结果
                bw_value = None
                for line in result.stdout.split('\n'):
                    if 'Bus BW' in line or 'Algo BW' in line:
                        import re
                        match = re.search(r'(\d+\.?\d*)', line)
                        if match:
                            bw_value = float(match.group(1))
                
                results[n] = bw_value
                print(f"  {n} GPUs: {bw_value} GB/s" if bw_value else f"  {n} GPUs: N/A")
                
            except (subprocess.TimeoutExpired, FileNotFoundError):
                results[n] = None
                print(f"  {n} GPUs: 失败")
        
        return results
    
    def measure_gpu_compute_performance(self, gpu_id: int,
                                         workload: str = "gemm") -> float:
        """测量单个 GPU 的计算性能"""
        # 使用 rocblas 或 miopen 进行基准测试
        try:
            if workload == "gemm":
                result = subprocess.run([
                    "rocblas-bench",
                    "--gpu", str(gpu_id),
                    "--size", "8192",
                    "--transA", "N",
                    "--transB", "N"
                ], capture_output=True, text=True, timeout=60)
                
                for line in result.stdout.split('\n'):
                    if 'TFLOPS' in line or 'tflops' in line:
                        import re
                        match = re.search(r'(\d+\.?\d*)', line)
                        if match:
                            return float(match.group(1))
            
            elif workload == "conv":
                result = subprocess.run([
                    "miopen-bench",
                    "--gpu", str(gpu_id),
                    "--input", "256,3,224,224",
                    "--filter", "64,3,3,3"
                ], capture_output=True, text=True, timeout=60)
                
                for line in result.stdout.split('\n'):
                    if 'throughput' in line or 'Throughput' in line:
                        import re
                        match = re.search(r'(\d+\.?\d*)', line)
                        if match:
                            return float(match.group(1))
            
            return None
            
        except (subprocess.TimeoutExpired, FileNotFoundError):
            return None
    
    def compute_effective_bandwidth(self, num_gpus: int,
                                     iteration: int = 10) -> Dict:
        """计算多 GPU 的通信有效带宽"""
        results = {
            "num_gpus": num_gpus,
            "iterations": iteration,
            "bus_bw_gbs": [],
            "algo_bw_gbs": []
        }
        
        for i in range(iteration):
            try:
                result = subprocess.run([
                    "rocm-bandwidth-test",
                    "-n", str(num_gpus),
                    "-t", "all-reduce",
                    "-s", "256"
                ], capture_output=True, text=True, timeout=30)
                
                for line in result.stdout.split('\n'):
                    if 'Bus BW' in line:
                        match = __import__('re').search(r'(\d+\.?\d*)', line)
                        if match:
                            results["bus_bw_gbs"].append(float(match.group(1)))
                    elif 'Algo BW' in line:
                        match = __import__('re').search(r'(\d+\.?\d*)', line)
                        if match:
                            results["algo_bw_gbs"].append(float(match.group(1)))
                            
            except:
                pass
        
        if results["bus_bw_gbs"]:
            results["bus_bw_avg"] = statistics.mean(results["bus_bw_gbs"])
            results["bus_bw_p99"] = sorted(results["bus_bw_gbs"])[
                int(len(results["bus_bw_gbs"]) * 0.99)
            ]
        if results["algo_bw_gbs"]:
            results["algo_bw_avg"] = statistics.mean(results["algo_bw_gbs"])
        
        return results
    
    def generate_report(self) -> Dict:
        """生成完整的性能报告"""
        report = {
            "num_gpus": self.num_gpus,
            "timestamp": time.strftime("%Y-%m-%d %H:%M:%S"),
            "p2p_bandwidth": {},
            "all_reduce": {},
            "compute": {}
        }
        
        print(f"\n多 GPU 性能测试报告")
        print("=" * 60)
        print(f"GPU 数量: {self.num_gpus}")
        print(f"测试时间: {report['timestamp']}")
        print()
        
        # P2P 测试
        print("[1/3] P2P 带宽测试:")
        p2p = self.measure_all_p2p()
        for (i, j), bw in p2p.items():
            report["p2p_bandwidth"][f"gpu{i}_gpu{j}"] = bw
        
        # All-Reduce 测试
        print("\n[2/3] All-Reduce 带宽测试:")
        ar = self.run_all_reduce_benchmark()
        for n, bw in ar.items():
            report["all_reduce"][f"{n}_gpus"] = bw
        
        # 计算性能测试
        print("\n[3/3] 单 GPU 计算性能:")
        for gpu_id in range(self.num_gpus):
            tflops = self.measure_compute_performance(gpu_id, "gemm")
            report["compute"][f"gpu{gpu_id}"] = tflops
            print(f"  GPU {gpu_id}: {tflops} TFLOPS" if tflops else f"  GPU {gpu_id}: N/A")
        
        return report
    
    def save_report(self, report: Dict, filename: str = "multi_gpu_report.json"):
        """保存报告"""
        with open(filename, "w") as f:
            json.dump(report, f, indent=2)
        print(f"\n报告已保存: {filename}")


class MultiGPUStressTester:
    """多 GPU 压力测试"""
    
    def __init__(self, num_gpus: int):
        self.num_gpus = num_gpus
        self.running = False
        self.stats = {i: {"errors": 0, "throughput": []} for i in range(num_gpus)}
    
    def load_thread(self, gpu_id: int, workload_type: str = "compute"):
        """GPU 负载线程"""
        while self.running:
            try:
                # 执行 GPU 计算任务
                if workload_type == "compute":
                    subprocess.run([
                        "rocblas-bench",
                        "--gpu", str(gpu_id),
                        "--size", "4096",
                        "--ntrans", "N",
                        "--n", "100"
                    ], capture_output=True, timeout=10)
                elif workload_type == "memory":
                    subprocess.run([
                        "rocm-bandwidth-test",
                        "-p", str(gpu_id),
                        "-s", "512",
                        "-t", "p2p"
                    ], capture_output=True, timeout=10)
                
                self.stats[gpu_id]["throughput"].append(time.time())
                
            except subprocess.TimeoutExpired:
                self.stats[gpu_id]["errors"] += 1
            except Exception as e:
                self.stats[gpu_id]["errors"] += 1
    
    def run_stress_test(self, duration: int = 60,
                        workload: str = "compute"):
        """运行压力测试"""
        print(f"\n多 GPU 压力测试 (持续时间: {duration}s, 负载: {workload})")
        
        self.running = True
        threads = []
        
        for gpu_id in range(self.num_gpus):
            t = threading.Thread(
                target=self.load_thread,
                args=(gpu_id, workload)
            )
            threads.append(t)
            t.start()
        
        time.sleep(duration)
        self.running = False
        
        for t in threads:
            t.join()
        
        # 统计
        print("\n测试结果:")
        print(f"{'GPU':<8} {'操作数':<10} {'错误数':<10} {'错误率':<10}")
        print("-" * 38)
        
        for gpu_id in range(self.num_gpus):
            ops = len(self.stats[gpu_id]["throughput"])
            errors = self.stats[gpu_id]["errors"]
            err_rate = f"{errors / max(ops + errors, 1) * 100:.1f}%"
            print(f"GPU{gpu_id:<5} {ops:<10} {errors:<10} {err_rate:<10}")
        
        return self.stats
```

### 案例分析

#### 案例一：多 GPU 训练时 All-Reduce 通信瓶颈

**问题现象：**
```
多 GPU 训练吞吐量随 GPU 数量增加不线性扩展：
1 GPU:  100 images/sec
2 GPUs:  140 images/sec (预期: 200)
4 GPUs:  180 images/sec (预期: 400)
8 GPUs:  210 images/sec (预期: 800)
```

**诊断过程：**
```bash
# 1. 检查 P2P 带宽
rocm-bandwidth-test -t p2p -p 0 -q 1
# 结果: P2P BW = 12 GB/s (理论应为 32 GB/s)

# 2. 检查 PCIe 连接速度
lspci -vvvs 03:00.0 | grep -i "Speed\|Width"
# LnkSta: Speed 8GT/s (Gen3), Width x8 (应为 x16)

# 3. 检查 PCIe Switch 配置
lspci -vvvs 02:00.0 | grep -i "Speed\|Width"
# PCIe Switch 的上行端口也是 x8
```

**根本原因：** GPU 通过 PCIe Switch 连接到 CPU，但 Switch 的上行带宽只有 x8，导致所有 GPU 共享 16 GB/s 的上行带宽。

```bash
# 拓扑图示:
#     CPU
#      | x8 ← 瓶颈 (所有 4 个 GPU 共享)
#   PCIe Switch
#   /    |    \
# x16  x16  x16
# GPU0 GPU1 GPU2 GPU3

# 4 个 GPU 间的 All-Reduce 通信都需要经过 x8 上行链路
# 导致有效通信带宽 = 16 GB/s / 4 = 4 GB/s per GPU
```

**解决方案：**

```bash
# 方案 1: 使用更高效的通信拓扑
# 将 GPU 分组，减少跨 Switch 通信
# 每 2 个 GPU 使用本地 All-Reduce，减少上行带宽需求

# 方案 2: 调整 All-Reduce 算法
export RCCL_ALGO=Ring  # 使用 Ring All-Reduce (更适合小规模)
export RCCL_PROTO=Simple  # 简化协议

# 方案 3: 硬件升级
# - 使用更多 CPU PCIe lanes (如 EPYC 128 lanes)
# - 使用 x16 上行端口的 PCIe Switch
# - 使用 XGMI 互联 (如果支持)

# 方案 4: 软件优化 - 通信重叠
# 将计算和通信重叠，隐藏通信延迟
# NCCL/RCCL 的 compute/communication overlap 模式
export RCCL_COMM_OVERLAP=1
```

#### 案例二：异构 GPU 混合系统中的驱动兼容性问题

**问题现象：**
```bash
# 系统有两张不同代的 AMD GPU
lspci | grep AMD
# 03:00.0 VGA: Advanced Micro Devices [AMD/ATI] Vega 20 [Radeon Pro W5700]
# 04:00.0 VGA: Advanced Micro Devices [AMD/ATI] Navi 21 [Radeon RX 6900 XT]

# 加载 amdgpu 后
dmesg | grep amdgpu
# [    5.123] amdgpu: Vega20 (Vega 20) detected
# [    5.456] amdgpu: amdgpu_device_ip_init for Vega20 failed: -22
# [    5.789] amdgpu: skipping Nike (Navi 21) due to firmware missing
```

**诊断过程：**
```bash
# 1. 检查固件文件
ls /lib/firmware/amdgpu/ | grep -i "vega20\|navi21"
# 发现 Vega20 有固件，Navi 21 缺少固件

# 2. 检查内核版本
uname -r
# 5.15.0-generic (可能不支持 Navi 21)

# 3. 检查驱动参数
cat /sys/module/amdgpu/parameters/supported_asic_table 2>/dev/null || echo "不可用"
```

**解决方案：**
```bash
# 方案 1: 更新内核和固件
apt-get install --reinstall linux-firmware
# 或从 kernel.org 下载最新固件
wget https://git.kernel.org/pub/scm/linux/kernel/git/firmware/linux-firmware.git/amdgpu/navi21_*.bin

# 方案 2: 使用 DKMS 安装最新驱动
# 从 AMD 官网下载最新驱动
# ./amdgpu-install --usecase=rocm

# 方案 3: 为每块 GPU 绑定不同的驱动
# 在 modprobe.d 中配置:
# options amdgpu cik_support=1
# options amdgpu si_support=1
# softdep amdgpu pre: vfio-pci  # 如果一块 GPU 用于直通

# 方案 4: 使用 VFIO 直通不支持的 GPU
# 让内核 NATIVE 驱动支持一块，VFIO 直通另一块给 VM
echo "options vfio-pci ids=1002:73bf" > /etc/modprobe.d/vfio-pci.conf
```
