### 第三部分：简答题

**16. IGT 测试结构：**

- IGT_TEST_DESCRIPTION：测试功能描述
- igt_fixture：测试设置和清理代码
- igt_subtest：子测试定义
- igt_main：测试入口（宏展开为 main）

**17. igt_assert vs igt_require：**

- igt_assert：验证测试条件，失败标记 FAIL，继续后续测试
- igt_require：检查前置条件，失败标记 SKIP，跳过当前子测试

**18. drm_open_driver：**

- 功能：打开指定类型的 DRM 设备
- 参数：DRIVER_AMDGPU、DRIVER_INTEL、DRIVER_ANY 等
- 返回：文件描述符或 -1

**19. 五大设计原则：**

- 单一职责：每个子测试验证一个功能点
- 独立性：子测试之间无依赖
- 可重复性：相同输入产生相同结果
- 完备性：覆盖正常和错误路径
- 可维护性：代码清晰易读

**20. AMDGPU BO 测试：**

```c
igt_fixture {
    drm_fd = drm_open_driver(DRIVER_AMDGPU);
    igt_require(drm_fd >= 0);
}

igt_subtest("bo-create") {
    struct drm_mode_create_dumb create = {
        .width = 1024, .height = 768, .bpp = 32
    };
    do_ioctl(drm_fd, DRM_IOCTL_MODE_CREATE_DUMB, &create);
    igt_assert(create.handle != 0);
    
    struct drm_mode_destroy_dumb destroy = {.handle = create.handle};
    do_ioctl(drm_fd, DRM_IOCTL_MODE_DESTROY_DUMB, &destroy);
}
```

### 第四部分：综合实践题

**综合实践题一参考答案：**

```c
#include "igt.h"
#include <amdgpu.h>

IGT_TEST_DESCRIPTION("AMDGPU BO test");

static int drm_fd;
static amdgpu_device_handle device;

igt_fixture {
    drm_fd = drm_open_driver(DRIVER_AMDGPU);
    igt_require(drm_fd >= 0);
    amdgpu_device_initialize(drm_fd, NULL, NULL, &device);
}

igt_subtest("create") {
    amdgpu_bo_handle bo;
    amdgpu_bo_alloc_req req = {
        .alloc_size = 4096,
        .preferred_heap = AMDGPU_GEM_DOMAIN_GTT
    };
    igt_assert_eq(amdgpu_bo_alloc(device, &req, &bo), 0);
    amdgpu_bo_free(bo);
}

igt_subtest("mmap") {
    amdgpu_bo_handle bo;
    void *ptr;
    amdgpu_bo_alloc_req req = {
        .alloc_size = 4096,
        .preferred_heap = AMDGPU_GEM_DOMAIN_GTT
    };
    amdgpu_bo_alloc(device, &req, &bo);
    amdgpu_bo_cpu_map(bo, &ptr);
    memset(ptr, 0xAB, 4096);
    amdgpu_bo_cpu_unmap(bo);
    amdgpu_bo_free(bo);
}

igt_fixture {
    amdgpu_device_deinitialize(device);
    close(drm_fd);
}
```
