### 1. 概念原理

#### 1.1 GPU 风扇控制的重要性

GPU 风扇是散热系统的核心组件，其控制策略直接影响：

1. **温度管理**：有效控制 GPU 温度，防止过热导致的性能下降或硬件损坏。
2. **噪音控制**：风扇转速是系统噪音的主要来源，合理的控制策略可以平衡散热与静音。
3. **能耗优化**：风扇本身消耗电力，优化控制可减少整体功耗。
4. **寿命延长**：避免风扇长时间高速运转，延长风扇和轴承寿命。
5. **用户体验**：安静、稳定的运行环境提升用户体验。

#### 1.2 风扇控制硬件架构

**PWM（脉宽调制）控制原理**：
PWM 通过调节脉冲宽度来控制风扇转速，是现代 GPU 风扇控制的标准方法。

风扇内置霍尔传感器或光学传感器，每转产生 2 个脉冲（2-pole fan）或 4 个脉冲（4-pole fan）。

#### 1.3 风扇控制策略类型

AMD GPU 支持多种风扇控制策略，每种策略适用于不同场景：

| 控制策略 | 描述 | 优点 | 缺点 | 适用场景 |
|----------|------|------|------|----------|
| **自动控制** | SMU/驱动根据温度自动调整风扇转速 | 无需用户干预，平衡性能与噪音 | 可能不够激进或过于保守 | 默认设置，日常使用 |
| **手动固定转速** | 用户设置固定风扇转速百分比 | 完全控制，适合特定负载 | 可能过热（转速过低）或噪音大（转速过高） | 测试、调试、特殊环境 |
| **自定义曲线** | 用户定义温度-转速关系曲线 | 高度定制化，优化散热/噪音平衡 | 需要调校，可能不稳定 | 发烧友、特定工作负载 |
| **零转速模式** | 低温时风扇完全停转 | 完全静音，延长风扇寿命 | 高温时延迟响应 | 低负载、静音环境 |
| **激进模式** | 更高温度阈值，更快响应 | 更好散热，性能更稳定 | 更高噪音，更高功耗 | 超频、高强度计算 |
| **静音模式** | 降低最大转速，优先静音 | 低噪音 | 可能过热，性能受限 | 办公室、家庭影院 |

**自动控制算法原理**：
自动控制通常采用 PI（比例-积分）或 PID（比例-积分-微分）控制器：

```
自动风扇控制算法流程：
┌─────────────────────────────────────────────────────────┐
│            Automatic Fan Control Algorithm              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│ 输入：当前温度 (T_current)                               │
│ 目标：维持温度在目标范围 (T_target ± T_hysteresis)       │
│                                                         │
│ 算法步骤：                                               │
│ 1. 计算温度误差： error = T_current - T_target          │
│ 2. 比例项： P = Kp × error                             │
│ 3. 积分项： I = Ki × ∫error dt (累计误差)               │
│ 4. 微分项： D = Kd × d(error)/dt (误差变化率)           │
│ 5. 输出： PWM_output = P + I + D                       │
│                                                         │
│ 参数示例 (RDNA 3 默认)：                                 │
│  - Kp (比例增益)： 2.0                                  │
│  - Ki (积分增益)： 0.5                                  │
│  - Kd (微分增益)： 1.0                                  │
│  - T_target： 70°C                                      │
│  - T_hysteresis： 5°C                                   │
│                                                         │
│ 迟滞机制防止振荡：                                        │
│ 升温触发：温度 ≥ 75°C 时增加风扇转速                     │
│ 降温恢复：温度 ≤ 65°C 时才降低风扇转速                   │
│                                                         │
│ 响应曲线示例：                                           │
│ 温度 (°C)   误差     P项    I项    D项    PWM输出        │
│ 60          -10     -20    -5     0      25% (min)      │
│ 70          0       0      0      0      40%            │
│ 80          10      20     5      5      70%            │
│ 90          20      40     10     10     100% (max)     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

```
曲线特征：
- 线性曲线：温度与转速线性相关，平衡散热与噪音
- 激进曲线：低温时保持低转速，高温时快速提升，优化散热
- 静音曲线：全温度范围低转速，优先静音，适合低负载
- 零转速模式：低于阈值时风扇停转，完全静音
```

**自定义曲线设计原则**：
1. **起始温度**：风扇开始旋转的最低温度（通常 40-50°C）。
2. **最大温度**：风扇达到 100% 转速的温度（通常 80-90°C）。
3. **曲线斜率**：温度上升时转速增加的速度。
4. **迟滞宽度**：防止在阈值附近频繁启停。
5. **最小转速**：防止风扇停转导致轴承问题（通常 20-30%）。
6. **最大转速**：限制最大转速以控制噪音（通常 80-100%）。

**风扇曲线数学模型**：
```python
# 风扇曲线计算函数
def fan_curve(temperature, curve_type="linear", params=None):
    """
    计算给定温度下的风扇转速百分比
    
    参数：
    temperature: 当前温度 (°C)
    curve_type: 曲线类型 ("linear", "aggressive", "silent", "zero_rpm")
    params: 曲线参数字典
    
    返回：
    风扇转速百分比 (0-100)
    """
    if params is None:
        params = {}
    
    # 默认参数
    defaults = {
        "min_temp": 40,      # 风扇开始旋转的温度
        "max_temp": 80,      # 风扇达到100%转速的温度
        "min_speed": 20,     # 最小转速百分比
        "max_speed": 100,    # 最大转速百分比
        "zero_rpm_temp": 50, # 零转速模式的停转温度阈值
    }
    
    # 合并参数
    config = {**defaults, **params}
    
    # 温度边界检查
    if temperature < config["min_temp"]:
        if curve_type == "zero_rpm" and temperature < config["zero_rpm_temp"]:
            return 0  # 零转速模式
        else:
            return config["min_speed"]
    
    if temperature > config["max_temp"]:
        return config["max_speed"]
    
    # 计算温度在[min_temp, max_temp]区间的比例
    temp_range = config["max_temp"] - config["min_temp"]
    temp_ratio = (temperature - config["min_temp"]) / temp_range
    
    # 根据曲线类型计算转速
    if curve_type == "linear":
        # 线性曲线
        speed = config["min_speed"] + temp_ratio * (config["max_speed"] - config["min_speed"])
    
    elif curve_type == "aggressive":
        # 激进曲线：低温时增加缓慢，高温时快速上升
        # 使用二次函数：speed = min_speed + (temp_ratio^2) * (max_speed - min_speed)
        speed = config["min_speed"] + (temp_ratio ** 2) * (config["max_speed"] - config["min_speed"])
    
    elif curve_type == "silent":
        # 静音曲线：全范围较低转速
        # 使用平方根函数：speed = min_speed + sqrt(temp_ratio) * (max_speed*0.8 - min_speed)
        speed = config["min_speed"] + (temp_ratio ** 0.5) * (config["max_speed"] * 0.8 - config["min_speed"])
    
    elif curve_type == "zero_rpm":
        # 零转速模式：低于阈值停转，高于阈值使用激进曲线
        if temperature < config["zero_rpm_temp"]:
            return 0
        else:
            # 计算从zero_rpm_temp到max_temp的比例
            effective_ratio = (temperature - config["zero_rpm_temp"]) / (config["max_temp"] - config["zero_rpm_temp"])
            effective_ratio = max(0, min(1, effective_ratio))
            speed = config["min_speed"] + (effective_ratio ** 2) * (config["max_speed"] - config["min_speed"])
    
    else:
        raise ValueError(f"未知曲线类型: {curve_type}")
    
    return max(config["min_speed"], min(config["max_speed"], round(speed, 1)))
```

**风扇故障检测与保护**：
GPU 驱动持续监控风扇状态，检测异常情况：

1. **风扇停转检测**：Tachometer 信号长时间为 0。
2. **转速异常检测**：实际转速与 PWM 设置值偏差过大。
3. **轴承磨损检测**：转速波动或不稳定。
4. **电流异常检测**：风扇电机电流超出正常范围。

检测到故障时的保护措施：
- **日志警告**：记录到内核日志和系统日志。
- **性能限制**：降低 GPU 频率和功耗以减少发热。
- **紧急关机**：在严重故障且温度过高时关闭系统。
- **用户通知**：通过桌面通知或 LED 指示灯警告。

#### 2.3 自定义风扇曲线设置

**方法一：通过 sysfs 手动设置温度-PWM 映射点**
部分 GPU 支持通过 hwmon 接口设置自定义曲线：

```bash
#!/bin/bash
# set_custom_fan_curve.sh
# 设置自定义风扇曲线

HWMON_PATH=$(find /sys/class/drm/card0/device/hwmon/ -name "hwmon*" | head -1)
if [ -z "$HWMON_PATH" ]; then
    echo "Error: Could not find hwmon device"
    exit 1
fi

echo "Setting custom fan curve..."

# 首先设置为手动模式，以便设置曲线点
echo 1 > $HWMON_PATH/pwm1_enable

# 定义温度-PWM 映射点（温度 °C : PWM 值 0-255）
# 格式：温度 PWM值 PWM百分比%
declare -A curve_points=(
    [40]=51    # 20% at 40°C
    [50]=102   # 40% at 50°C
    [60]=153   # 60% at 60°C
    [70]=204   # 80% at 70°C
    [80]=255   # 100% at 80°C
)

# 检查是否支持自动控制点
if [ -f "$HWMON_PATH/pwm1_auto_point1_temp" ]; then
    echo "Setting auto control points..."
    
    # 设置温度点
    point_num=1
    for temp in $(echo "${!curve_points[@]}" | tr ' ' '\n' | sort -n); do
        pwm_value=${curve_points[$temp]}
        temp_file="$HWMON_PATH/pwm1_auto_point${point_num}_temp"
        pwm_file="$HWMON_PATH/pwm1_auto_point${point_num}_pwm"
        
        if [ -f "$temp_file" ] && [ -f "$pwm_file" ]; then
            echo $temp > $temp_file
            echo $pwm_value > $pwm_file
            pwm_percent=$(( (pwm_value * 100) / 255 ))
            echo "  Point $point_num: $temp°C -> $pwm_value ($pwm_percent%)"
            point_num=$((point_num + 1))
        fi
    done
    
    # 启用自动模式
    echo 2 > $HWMON_PATH/pwm1_enable
    echo "Custom fan curve enabled (auto mode)"
    
else
    echo "Auto control points not supported. Using manual control with monitoring script."
    
    # 创建监控脚本
    cat > fan_curve_monitor.sh << 'EOF'
#!/bin/bash
# fan_curve_monitor.sh
# 基于温度手动调整风扇的监控脚本

HWMON_PATH=$(find /sys/class/drm/card0/device/hwmon/ -name "hwmon*" | head -1)
INTERVAL=5  # 检查间隔（秒）

# 风扇曲线定义（温度 °C : PWM 百分比）
declare -A FAN_CURVE=(
    [40]=20
    [50]=40
    [60]=60
    [70]=80
    [80]=100
)

# 获取温度对应的 PWM 百分比
get_fan_speed() {
    local temp=$1
    local prev_temp=0
    local prev_speed=0
    
    # 按温度排序
    for t in $(echo "${!FAN_CURVE[@]}" | tr ' ' '\n' | sort -n); do
        if [ $temp -le $t ]; then
            if [ $prev_temp -eq 0 ]; then
                # 低于第一个点，使用第一个点的速度
                echo ${FAN_CURVE[$t]}
                return
            else
                # 线性插值
                local speed_range=$((${FAN_CURVE[$t]} - prev_speed))
                local temp_range=$(($t - prev_temp))
                local temp_ratio=$(( (temp - prev_temp) * 100 / temp_range ))
                local speed=$(( prev_speed + (speed_range * temp_ratio) / 100 ))
                echo $speed
                return
            fi
        fi
        prev_temp=$t
        prev_speed=${FAN_CURVE[$t]}
    done
    
    # 高于最后一个点，使用最后一个点的速度
    echo ${FAN_CURVE[$prev_temp]}
}

echo "Starting fan curve monitor..."
echo "Press Ctrl+C to stop."

# 设置手动模式
echo 1 > $HWMON_PATH/pwm1_enable

while true; do
    # 获取当前温度
    temp_file=$(find $HWMON_PATH -name "temp1_input" | head -1)
    if [ -f "$temp_file" ]; then
        temp_mc=$(cat $temp_file)
        temp_c=$((temp_mc / 1000))
    else
        temp_c=50  # 默认值
    fi
    
    # 计算目标风扇速度
    target_percent=$(get_fan_speed $temp_c)
    pwm_value=$(( (255 * target_percent) / 100 ))
    
    # 设置 PWM 值
    echo $pwm_value > $HWMON_PATH/pwm1
    
    # 获取实际转速
    fan_rpm=$(cat $HWMON_PATH/fan1_input 2>/dev/null || echo "N/A")
    
    # 显示状态
    echo "$(date '+%H:%M:%S') Temp: ${temp_c}°C -> Fan: ${target_percent}% (PWM: ${pwm_value}, RPM: ${fan_rpm})"
    
    sleep $INTERVAL
done
EOF
    
    chmod +x fan_curve_monitor.sh
    echo "Created fan_curve_monitor.sh. Run it to apply custom curve."
fi
```

#### 2.4 风扇性能测试与优化

**风扇响应测试脚本**：
```bash
#!/bin/bash
# fan_response_test.sh
# 测试风扇对温度变化的响应能力

HWMON_PATH=$(find /sys/class/drm/card0/device/hwmon/ -name "hwmon*" | head -1)
TEST_DURATION=300  # 测试时长（秒）
LOG_FILE="fan_response_test_$(date +%Y%m%d_%H%M%S).csv"

# 确保自动模式
echo 2 > $HWMON_PATH/pwm1_enable

echo "=== Fan Response Test ==="
echo "Duration: ${TEST_DURATION}s"
echo "Log file: $LOG_FILE"
echo ""

# 创建日志文件头
echo "Timestamp,Temp(°C),FanRPM,PWM(%),PWM(raw),Mode" > $LOG_FILE

# 启动温度负载（如果温度较低）
start_load_if_needed() {
    temp_file=$(find $HWMON_PATH -name "temp1_input" | head -1)
    if [ -f "$temp_file" ]; then
        temp_mc=$(cat $temp_file)
        temp_c=$((temp_mc / 1000))
        
        if [ $temp_c -lt 60 ]; then
            echo "Starting workload to increase temperature..."
            glmark2 --run-forever > /dev/null 2>&1 &
            WORKLOAD_PID=$!
            echo "Workload started (PID: $WORKLOAD_PID)"
        fi
    fi
}

# 监控循环
start_load_if_needed

for i in $(seq 1 $TEST_DURATION); do
    timestamp=$(date +%Y-%m-%d_%H:%M:%S)
    
    # 读取温度
    temp_file=$(find $HWMON_PATH -name "temp1_input" | head -1)
    temp_c=0
    if [ -f "$temp_file" ]; then
        temp_mc=$(cat $temp_file)
        temp_c=$((temp_mc / 1000))
    fi
    
    # 读取风扇转速
    fan_rpm=$(cat $HWMON_PATH/fan1_input 2>/dev/null || echo "0")
    
    # 读取 PWM 值
    pwm_raw=0
    pwm_percent=0
    if [ -f "$HWMON_PATH/pwm1" ]; then
        pwm_raw=$(cat $HWMON_PATH/pwm1)
        pwm_percent=$(( (pwm_raw * 100) / 255 ))
    fi
    
    # 读取模式
    mode=$(cat $HWMON_PATH/pwm1_enable 2>/dev/null || echo "N/A")
    
    # 记录数据
    echo "$timestamp,$temp_c,$fan_rpm,$pwm_percent,$pwm_raw,$mode" >> $LOG_FILE
    
    # 显示进度
    if [ $((i % 10)) -eq 0 ]; then
        echo "[$i/$TEST_DURATION] Temp: ${temp_c}°C, Fan: ${fan_rpm} RPM (${pwm_percent}%)"
    fi
    
    # 动态调整负载以创建温度变化
    if [ $i -eq 100 ] && [ -z "$WORKLOAD_PID" ]; then
        start_load_if_needed
    fi
    
    if [ $i -eq 200 ] && [ -n "$WORKLOAD_PID" ]; then
        echo "Stopping workload..."
        kill $WORKLOAD_PID 2>/dev/null
        WORKLOAD_PID=""
    fi
    
    sleep 1
done

# 清理
if [ -n "$WORKLOAD_PID" ]; then
    kill $WORKLOAD_PID 2>/dev/null
fi

echo ""
echo "Test completed. Analyzing results..."

# 分析响应特性
analyze_fan_response() {
    local log_file=$1
    
    echo "=== Fan Response Analysis ==="
    
    # 计算平均响应时间
    # 响应时间定义：温度变化后风扇达到目标转速的90%所需时间
    echo "Response time analysis (simplified):"
    
    # 找出温度显著上升的时间段
    awk -F',' 'NR>1 {
        if (prev_temp > 0) {
            temp_diff = $2 - prev_temp
            rpm_diff = $3 - prev_rpm
            
            if (temp_diff > 5 && $2 > 60) {
                printf "  Temp spike: %s (%.1f°C → %.1f°C, +%.1f°C)\n", 
                    $1, prev_temp, $2, temp_diff
                printf "    Fan response: %.0f RPM → %.0f RPM (%.0f%% → %.0f%%)\n",
                    prev_rpm, $3, prev_pwm_percent, $5
            }
        }
        
        prev_temp = $2
        prev_rpm = $3
        prev_pwm_percent = $5
    }' "$log_file"
    
    # 计算风扇转速范围
    min_rpm=$(tail -n +2 "$log_file" | cut -d',' -f3 | sort -n | head -1)
    max_rpm=$(tail -n +2 "$log_file" | cut -d',' -f3 | sort -n | tail -1)
    avg_rpm=$(tail -n +2 "$log_file" | cut -d',' -f3 | awk '{sum+=$1} END {if(NR>0) printf "%.0f", sum/NR}')
    
    echo ""
    echo "Fan RPM statistics:"
    echo "  Min: $min_rpm RPM"
    echo "  Max: $max_rpm RPM"
    echo "  Avg: $avg_rpm RPM"
    echo "  Range: $((max_rpm - min_rpm)) RPM"
    
    # 温度与风扇转速相关性
    temp_rpm_corr=$(tail -n +2 "$log_file" | awk -F',' '
    {
        temp[NR] = $2;
        rpm[NR] = $3;
        sum_temp += $2;
        sum_rpm += $3;
    }
    END {
        n = NR;
        if (n < 2) {
            print "N/A";
            exit;
        }
        
        mean_temp = sum_temp / n;
        mean_rpm = sum_rpm / n;
        
        num = 0;
        denom_temp = 0;
        denom_rpm = 0;
        
        for (i = 1; i <= n; i++) {
            num += (temp[i] - mean_temp) * (rpm[i] - mean_rpm);
            denom_temp += (temp[i] - mean_temp) ^ 2;
            denom_rpm += (rpm[i] - mean_rpm) ^ 2;
        }
        
        if (denom_temp > 0 && denom_rpm > 0) {
            corr = num / sqrt(denom_temp * denom_rpm);
            printf "%.3f", corr;
        } else {
            print "N/A";
        }
    }')
    
    echo "  Temperature-RPM correlation: $temp_rpm_corr"
    if [ "$temp_rpm_corr" != "N/A" ]; then
        if (( $(echo "$temp_rpm_corr > 0.7" | bc -l) )); then
            echo "    Interpretation: Strong positive correlation (fan responds well to temp)"
        elif (( $(echo "$temp_rpm_corr > 0.3" | bc -l) )); then
            echo "    Interpretation: Moderate correlation"
        else
            echo "    Interpretation: Weak correlation (fan may not be responding properly)"
        fi
    fi
    
    # 检查风扇稳定性（转速波动）
    echo ""
    echo "Fan stability analysis:"
    
    # 计算转速变化率
    tail -n +2 "$log_file" | awk -F',' '
    {
        if (prev_rpm > 0) {
            rpm_change = $3 - prev_rpm;
            rpm_change_abs = rpm_change < 0 ? -rpm_change : rpm_change;
            rpm_change_percent = (rpm_change_abs / prev_rpm) * 100;
            
            if (rpm_change_percent > 10) {
                large_changes++;
                printf "  Large change at %s: %.0f RPM → %.0f RPM (%.1f%% change)\n",
                    $1, prev_rpm, $3, rpm_change_percent;
            }
            
            total_changes += rpm_change_abs;
            change_count++;
        }
        prev_rpm = $3;
    }
    END {
        if (change_count > 0) {
            avg_change = total_changes / change_count;
            printf "  Average RPM change per second: %.1f RPM\n", avg_change;
            printf "  Large changes (>10%%): %d/%d (%.1f%%)\n", 
                large_changes, change_count, (large_changes/change_count)*100;
        }
    }'
}

analyze_fan_response "$LOG_FILE"

echo ""
echo "Detailed data saved to: $LOG_FILE"
echo "Use the following command to plot the data:"
echo "  python3 -c \"import pandas as pd; import matplotlib.pyplot as plt;"
echo "  df = pd.read_csv('$LOG_FILE');"
echo "  fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 8));"
echo "  ax1.plot(df['Temp(°C)'], label='Temperature');"
echo "  ax1.set_ylabel('Temperature (°C)'); ax1.legend();"
echo "  ax2.plot(df['FanRPM'], label='Fan RPM', color='red');"
echo "  ax2.set_ylabel('Fan RPM'); ax2.set_xlabel('Time (samples)');"
echo "  ax2.legend(); plt.tight_layout(); plt.show()\""
```

**风扇曲线优化建议**：
基于测试结果，优化风扇曲线：

1. **响应速度优化**：
   - 如果响应太慢：减小温度迟滞，增加比例增益（Kp）。
   - 如果振荡：增加迟滞，减小比例增益。

2. **噪音优化**：
   - 识别噪音敏感区间（如 1500-2000 RPM）。
   - 调整曲线避开该区间，或快速通过。

3. **散热优化**：
   - 在关键温度点（如 70°C、80°C）确保足够的风扇转速。
   - 考虑显存和 VRM 温度，可能需要更激进的曲线。

4. **能效优化**：
   - 在安全范围内尽可能降低风扇转速。
   - 考虑负载类型：计算密集型需要更好散热，内存密集型可能对温度不那么敏感。

#### 3.1 内核补丁：修复风扇控制模式切换导致的系统冻结（commit b2c3f4e）

在 Linux 内核 v6.4 中，一个 Bug 导致在特定 AMD GPU 上切换风扇控制模式时系统可能冻结。

**问题表现**：
- 从自动模式切换到手动模式时系统完全冻结。
- 需要硬重启才能恢复。
- 主要影响 RDNA 2 架构的某些型号。
- 在重负载下更容易触发。

**根本原因**：
`amdgpu_fan.c` 中的 `amdgpu_fan_set_mode()` 函数在切换模式时未正确同步 PWM 控制器状态。当从自动模式切换到手动模式时，SMU 固件仍在控制 PWM，而驱动尝试直接写入 PWM 寄存器，导致硬件冲突。

**修复前的有问题的代码**：
```c
static int amdgpu_fan_set_mode(struct amdgpu_device *adev, u32 mode)
{
    struct amdgpu_fan *fan = &adev->pm.fan;
    int ret;
    
    if (mode == fan->mode)
        return 0;
    
    // 问题：直接切换模式，没有同步 PWM 状态
    switch (mode) {
    case AMDGPU_FAN_CTRL_AUTO:
        ret = amdgpu_fan_set_auto_mode(adev);
        break;
    case AMDGPU_FAN_CTRL_MANUAL:
        ret = amdgpu_fan_set_manual_mode(adev);
        break;
    default:
        return -EINVAL;
    }
    
    if (!ret)
        fan->mode = mode;
    
    return ret;
}

static int amdgpu_fan_set_manual_mode(struct amdgpu_device *adev)
{
    // 直接设置手动模式，可能与其他控制源冲突
    return amdgpu_smu_set_fan_control_mode(adev, 
        AMD_FAN_CTRL_MANUAL);
}
```

**修复后的代码**：
```c
static int amdgpu_fan_set_mode(struct amdgpu_device *adev, u32 mode)
{
    struct amdgpu_fan *fan = &adev->pm.fan;
    int ret;
    
    if (mode == fan->mode)
        return 0;
    
    // 添加同步机制：先停止当前控制，再切换模式
    mutex_lock(&fan->mutex);
    
    // 保存当前 PWM 值
    u32 current_pwm = fan->current_pwm;
    
    // 停止当前控制源
    ret = amdgpu_fan_stop_control(adev);
    if (ret) {
        mutex_unlock(&fan->mutex);
        return ret;
    }
    
    // 切换模式
    switch (mode) {
    case AMDGPU_FAN_CTRL_AUTO:
        ret = amdgpu_fan_set_auto_mode(adev);
        break;
    case AMDGPU_FAN_CTRL_MANUAL:
        // 设置手动模式前，先应用当前 PWM 值
        ret = amdgpu_smu_set_fan_control_mode(adev, 
            AMD_FAN_CTRL_MANUAL);
        if (!ret) {
            // 应用保存的 PWM 值，避免突然变化
            ret = amdgpu_fan_set_pwm(adev, current_pwm);
        }
        break;
    default:
        ret = -EINVAL;
        break;
    }
    
    if (!ret) {
        fan->mode = mode;
        // 添加调试日志
        dev_dbg(adev->dev, "Fan control mode changed to %s\n",
            mode == AMDGPU_FAN_CTRL_AUTO ? "auto" : "manual");
    }
    
    mutex_unlock(&fan->mutex);
    return ret;
}

static int amdgpu_fan_stop_control(struct amdgpu_device *adev)
{
    // 停止所有 PWM 控制源
    int ret = 0;
    
    // 1. 通知 SMU 停止控制
    ret = amdgpu_smu_stop_fan_control(adev);
    if (ret) {
        dev_warn(adev->dev, 
            "Failed to stop SMU fan control: %d\n", ret);
    }
    
    // 2. 等待硬件稳定
    udelay(100);
    
    // 3. 验证 PWM 控制器状态
    ret = amdgpu_fan_verify_pwm_state(adev);
    
    return ret;
}
```

#### 3.2 实际案例：数据中心 GPU 风扇控制优化

在超算中心，AMD Instinct MI250X 加速卡的风扇控制面临特殊挑战：

**挑战**：
1. **高密度部署**：8 GPU 刀片服务器，散热空间有限。
2. **持续高负载**：7×24 小时 AI 训练，温度稳定性要求高。
3. **噪音限制**：数据中心噪音控制要求。
4. **能效目标**：PUE（Power Usage Effectiveness）优化。

**解决方案**：
1. **协同风扇控制**：所有 GPU 风扇同步调整，避免局部热点。
2. **预测性控制**：基于工作负载预测温度变化，提前调整风扇。
3. **自适应曲线**：根据环境温度、湿度、气压动态调整风扇曲线。
4. **故障容错**：单个风扇故障时，其他风扇自动补偿。

**实施代码示例**：
```python
# datacenter_fan_controller.py
# 数据中心 GPU 风扇协同控制器

import json
import time
import numpy as np
from dataclasses import dataclass
from typing import List, Dict, Tuple
from datetime import datetime
import statistics

@dataclass
class GPUFan:
    """单个 GPU 风扇状态"""
    gpu_id: int
    current_rpm: int
    target_rpm: int
    current_temp: float
    pwm_value: int
    mode: str  # 'auto', 'manual', 'sync'
    health_status: str  # 'healthy', 'warning', 'faulty'

@dataclass
class FanControlPolicy:
    """风扇控制策略"""
    name: str
    min_temp: float  # 开始加速的温度
    max_temp: float  # 100% 风扇转速的温度
    min_rpm: int     # 最小转速（防止轴承问题）
    max_rpm: int     # 最大转速（噪音限制）
    response_speed: float  # 响应速度系数 (0.0-1.0)
    noise_weight: float    # 噪音权重 (0.0-1.0)
    power_weight: float   # 功耗权重 (0.0-1.0)

class DatacenterFanController:
    """数据中心风扇控制器"""
    
    def __init__(self, gpu_count: int, policy: FanControlPolicy):
        self.gpu_count = gpu_count
        self.policy = policy
        self.fans = [GPUFan(i, 0, 0, 40.0, 0, 'auto', 'healthy') 
                    for i in range(gpu_count)]
        self.history = []
        self.sync_mode = True  # 是否同步所有风扇
        
    def update_temperatures(self, temps: List[float]):
        """更新 GPU 温度"""
        for i, temp in enumerate(temps):
            if i < len(self.fans):
                self.fans[i].current_temp = temp
    
    def calculate_target_rpms(self) -> List[int]:
        """计算目标转速"""
        target_rpms = []
        
        if self.sync_mode:
            # 同步模式：基于最高温度计算所有风扇转速
            max_temp = max(fan.current_temp for fan in self.fans)
            base_rpm = self._temp_to_rpm(max_temp)
            
            for fan in self.fans:
                # 考虑单个 GPU 的温度偏差
                temp_diff = fan.current_temp - max_temp
                if temp_diff < -5:  # 明显低于最高温度
                    # 适当降低该风扇转速
                    adjustment = int(base_rpm * 0.8)
                else:
                    adjustment = base_rpm
                
                target_rpms.append(adjustment)
        else:
            # 独立模式：每个风扇基于自身温度
            for fan in self.fans:
                target_rpms.append(self._temp_to_rpm(fan.current_temp))
        
        return target_rpms
    
    def _temp_to_rpm(self, temperature: float) -> int:
        """将温度转换为目标转速"""
        # 使用改进的曲线函数
        if temperature < self.policy.min_temp:
            return self.policy.min_rpm
        
        if temperature > self.policy.max_temp:
            return self.policy.max_rpm
        
        # 计算温度比例
        temp_range = self.policy.max_temp - self.policy.min_temp
        temp_ratio = (temperature - self.policy.min_temp) / temp_range
        
        # 考虑响应速度和噪音权重
        # 响应速度快 -> 更激进的曲线
        # 噪音权重高 -> 更平缓的曲线
        curve_exponent = 2.0 - self.policy.response_speed + self.policy.noise_weight
        
        # 计算转速比例
        rpm_ratio = temp_ratio ** curve_exponent
        
        # 计算目标转速
        rpm_range = self.policy.max_rpm - self.policy.min_rpm
        target_rpm = self.policy.min_rpm + int(rpm_ratio * rpm_range)
        
        # 应用功耗权重
        if self.policy.power_weight > 0.5:
            # 高功耗权重时，适当降低转速
            reduction = int(target_rpm * 0.1 * (self.policy.power_weight - 0.5) * 2)
            target_rpm -= reduction
        
        return max(self.policy.min_rpm, min(self.policy.max_rpm, target_rpm))
    
    def detect_fan_faults(self) -> List[Tuple[int, str]]:
        """检测风扇故障"""
        faults = []
        
        for fan in self.fans:
            # 检查转速异常
            if fan.current_rpm == 0 and fan.pwm_value > 50:
                faults.append((fan.gpu_id, "Fan stopped with PWM > 50%"))
                fan.health_status = 'faulty'
            
            # 检查响应延迟
            rpm_diff = abs(fan.current_rpm - fan.target_rpm)
            if rpm_diff > 500 and fan.pwm_value > 0:
                faults.append((fan.gpu_id, f"Poor fan response (diff: {rpm_diff} RPM)"))
                fan.health_status = 'warning'
            
            # 检查转速波动
            if self._check_rpm_fluctuation(fan.gpu_id):
                faults.append((fan.gpu_id, "Excessive RPM fluctuation"))
                fan.health_status = 'warning'
        
        return faults
    
    def _check_rpm_fluctuation(self, gpu_id: int, window_size: int = 10) -> bool:
        """检查转速波动"""
        # 从历史记录中获取最近转速
        recent_rpms = []
        for record in self.history[-window_size:]:
            if gpu_id < len(record['rpms']):
                recent_rpms.append(record['rpms'][gpu_id])
        
        if len(recent_rpms) < 5:
            return False
        
        # 计算标准差
        if len(recent_rpms) >= 2:
            stdev = statistics.stdev(recent_rpms)
            mean = statistics.mean(recent_rpms)
            if mean > 0:
                cv = stdev / mean  # 变异系数
                return cv > 0.15  # 波动超过15%认为异常
        
        return False
    
    def handle_fan_fault(self, gpu_id: int, fault_type: str):
        """处理风扇故障"""
        fan = self.fans[gpu_id]
        
        if "stopped" in fault_type.lower():
            # 风扇停转，提高其他风扇转速补偿
            compensation_factor = 1.2  # 其他风扇提高20%
            
            for other_fan in self.fans:
                if other_fan.gpu_id != gpu_id and other_fan.health_status == 'healthy':
                    new_target = int(other_fan.target_rpm * compensation_factor)
                    other_fan.target_rpm = min(new_target, self.policy.max_rpm)
            
            # 标记故障 GPU 需要降频
            self._trigger_throttling(gpu_id)
            
        elif "poor response" in fault_type.lower():
            # 响应不良，切换到更激进的曲线
            old_response = self.policy.response_speed
            self.policy.response_speed = min(1.0, old_response + 0.2)
            
        elif "fluctuation" in fault_type.lower():
            # 转速波动，增加滤波
            self._apply_rpm_filter(gpu_id)
    
    def _trigger_throttling(self, gpu_id: int):
        """触发 GPU 降频"""
        # 在实际实现中，这里会通过 sysfs 或 ROCm SMI 设置降频
        print(f"[WARNING] GPU {gpu_id} fan faulty, triggering performance throttling")
        # 示例：降低最大频率
        # subprocess.run(["rocm-smi", "--setsclk", "3", "-d", str(gpu_id)])
    
    def _apply_rpm_filter(self, gpu_id: int):
        """应用 RPM 滤波"""
        # 在实际实现中，这里会调整 PWM 滤波参数
        print(f"[INFO] Applying RPM filter to GPU {gpu_id}")
    
    def optimize_for_power(self, power_budget: float, current_power: float):
        """基于功耗预算优化风扇控制"""
        power_ratio = current_power / power_budget
        
        if power_ratio > 1.0:
            # 超过功耗预算，降低风扇转速
            reduction = 0.1 * (power_ratio - 1.0)
            new_max_rpm = int(self.policy.max_rpm * (1.0 - reduction))
            self.policy.max_rpm = max(self.policy.min_rpm + 500, new_max_rpm)
            print(f"[OPTIMIZE] Power exceeded, reducing max RPM to {self.policy.max_rpm}")
        
        elif power_ratio < 0.8:
            # 有余量，可以适当提高转速以改善散热
            increase = 0.05 * (0.8 - power_ratio)
            new_response = min(1.0, self.policy.response_speed + increase)
            self.policy.response_speed = new_response
    
    def run_control_loop(self, interval: float = 5.0):
        """运行控制循环"""
        print("Starting datacenter fan control loop...")
        
        try:
            iteration = 0
            while True:
                iteration += 1
                
                # 1. 模拟获取温度（实际中从传感器读取）
                # 这里使用模拟温度数据
                simulated_temps = []
                for fan in self.fans:
                    # 模拟温度：基值 + 随机波动 + 负载影响
                    base_temp = 50.0 if fan.health_status == 'healthy' else 60.0
                    load_effect = 10.0 if iteration % 20 < 10 else 5.0  # 模拟负载变化
                    random_variation = np.random.uniform(-2.0, 2.0)
                    temp = base_temp + load_effect + random_variation
                    simulated_temps.append(temp)
                
                # 2. 更新温度
                self.update_temperatures(simulated_temps)
                
                # 3. 计算目标转速
                target_rpms = self.calculate_target_rpms()
                for i, fan in enumerate(self.fans):
                    fan.target_rpm = target_rpms[i]
                
                # 4. 检测故障
                faults = self.detect_fan_faults()
                for gpu_id, fault_desc in faults:
                    print(f"[FAULT] GPU {gpu_id}: {fault_desc}")
                    self.handle_fan_fault(gpu_id, fault_desc)
                
                # 5. 记录状态
                record = {
                    'timestamp': datetime.now().isoformat(),
                    'iteration': iteration,
                    'temperatures': [fan.current_temp for fan in self.fans],
                    'target_rpms': target_rpms,
                    'current_rpms': [fan.current_rpm for fan in self.fans],
                    'faults': len(faults),
                    'policy': {
                        'max_rpm': self.policy.max_rpm,
                        'response_speed': self.policy.response_speed
                    }
                }
                self.history.append(record)
                
                # 6. 定期优化
                if iteration % 10 == 0:
                    # 模拟功耗优化
                    self.optimize_for_power(3000.0, 2800.0)  # 3kW 预算，当前 2.8kW
                
                # 7. 显示状态
                if iteration % 5 == 0:
                    self._print_status(iteration)
                
                # 8. 保存历史记录（每隔一段时间）
                if iteration % 50 == 0:
                    self._save_history()
                
                time.sleep(interval)
                
        except KeyboardInterrupt:
            print("\nControl loop stopped by user")
            self._save_history()
    
    def _print_status(self, iteration: int):
        """打印状态"""
        print(f"\n=== Status (Iteration {iteration}) ===")
        
        # 温度统计
        temps = [fan.current_temp for fan in self.fans]
        avg_temp = statistics.mean(temps) if temps else 0
        max_temp = max(temps) if temps else 0
        
        print(f"Temperatures: Avg={avg_temp:.1f}°C, Max={max_temp:.1f}°C")
        
        # 风扇统计
        healthy_fans = sum(1 for fan in self.fans if fan.health_status == 'healthy')
        print(f"Fans: {healthy_fans}/{self.gpu_count} healthy")
        
        # 显示每个 GPU 状态
        for fan in self.fans:
            status_char = '✓' if fan.health_status == 'healthy' else '⚠' if fan.health_status == 'warning' else '✗'
            print(f"  GPU{fan.gpu_id}: {status_char} {fan.current_temp:.1f}°C → {fan.target_rpm} RPM")
    
    def _save_history(self):
        """保存历史记录到文件"""
        filename = f"fan_control_history_{datetime.now().strftime('%Y%m%d_%H%M%S')}.json"
        with open(filename, 'w') as f:
            # 只保存最近1000条记录
            recent_history = self.history[-1000:] if len(self.history) > 1000 else self.history
            json.dump(recent_history, f, indent=2, default=str)
        print(f"History saved to {filename}")
```
