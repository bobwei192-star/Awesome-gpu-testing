### 1. 回归测试基础

#### 1.1 回归测试的定义与目标

回归测试（Regression Testing）是指在软件修改后重新运行已有的测试用例，以确认修改没有引入新的缺陷或导致原有功能失效的测试活动。

```
回归测试的核心目标:
  ┌──────────────────────────────────────────────────────────┐
  │                                                          │
  │  1. 验证修改正确性 — 确认代码变更实现了预期的功能         │
  │  2. 检测回归缺陷   — 发现因修改而破坏的原有功能           │
  │  3. 保障系统稳定性 — 确保修改不引入副作用                 │
  │  4. 积累测试资产   — 逐步建立可复用的测试用例库           │
  │                                                          │
  └──────────────────────────────────────────────────────────┘
```

在GPU驱动开发中，回归测试的重要性尤为突出：

```
GPU驱动回归测试的特殊意义:

  ┌──────────────────────────────────────────────────────────────┐
  │  GPU驱动是操作系统中最复杂的软件之一，每个版本涉及:          │
  │                                                              │
  │  驱动版本     ─── 每月1-2个正式版本更新                       │

#### 1.3 回归测试的关键挑战

```
挑战维度                  GPU驱动特定表现
────────────────────────────────────────────────────────────────────
测试套件规模爆炸    每次提交需要验证几十种GPU型号 × 几十个测试模块
测试执行时间长      全量回归测试周期可达24小时以上
非确定性失败        硬件时序依赖导致偶发失败，难以判断是真回归还是假失败
环境依赖度高        需要特定的GPU硬件和驱动配置，CI环境成本高昂

维护成本大          每代新GPU引入新功能，测试用例数量持续增长
```

### 2. GPU驱动回归测试策略分类

#### 2.1 按选择策略分类

```
回归测试策略分类体系:

  完整回归测试 (Complete Retest)
  ┌──────────────────────────────────────────────┐
  │ 特点: 重新运行所有测试用例                   │
  │ 优点: 最安全，覆盖全面                       │
  │ 缺点: 时间成本极高                           │
  │ 适用: 发布候选版本、重大重构后               │
  └──────────────────────────────────────────────┘

  选择性回归测试 (Selective Retest)
  ├── 基于影响分析的选择
  │   ├── 代码依赖分析: 识别受影响的模块和测试
  │   ├── 数据流分析: 追踪被修改数据的流向
  │   └── 变更影响范围: 按函数/文件/模块层级
  │
  ├── 基于风险的选择
  │   ├── 历史缺陷密度: 高缺陷率模块优先回归
  │   ├── 修改频率: 高频修改模块优先回归
  │   └── 功能关键性: 核心功能必须回归测试
  │
  └── 基于覆盖的选择
      ├── 测试用例与代码的映射关系
      ├── 增量覆盖分析
      └── 测试用例优先级排序

  风险驱动回归测试 (Risk-driven Regression)
  ┌──────────────────────────────────────────────┐
  │ 特点: 根据风险级别分配回归测试资源          │
  │ 方法: 风险 = 故障概率 × 故障影响            │
  │ 输出: 分级测试计划 (P0/P1/P2/P3)            │
  └──────────────────────────────────────────────┘

  变更驱动回归测试 (Change-driven Regression)
  ┌──────────────────────────────────────────────┐
  │ 特点: 只运行与本次变更相关的测试            │
  │ 关键: 精确的变更影响分析                     │
  │ 工具: 静态代码分析 + 动态追踪               │
  └──────────────────────────────────────────────┘
```

### 3. 选择性回归测试技术 (RTS)

#### 3.1 基于代码影响分析的RTS

代码影响分析（Change Impact Analysis, CIA）是选择性回归测试的核心技术。

```
代码影响分析流程:

  输入: 代码变更 (diff/patch)
    │
    ▼
  步骤1: 静态分析 → 构建依赖图
    │  ├─ 函数调用关系图 (Call Graph)
    │  ├─ 数据依赖关系 (Data Dependency)
    │  └─ 模块依赖关系 (Module Dependency)
    │
    ▼
  步骤2: 标记变更节点
    │  ├─ 修改的函数/变量
    │  ├─ 增加/删除的代码行
    │  └─ 变更的头文件
    │
    ▼
  步骤3: 向前遍历 → 识别受影响的范围
    │  ├─ 直接调用者
    │  ├─ 间接调用者 (通过函数指针)
    │  └─ 数据消费者
    │
    ▼
  步骤4: 映射到测试用例
    │  ├─ 测试用例—代码映射矩阵
    │  ├─ 覆盖率数据关联
    │  └─ 历史缺陷关联
    │
    ▼
  输出: 需要回归的测试用例集合

AMDGPU驱动影响分析的模块依赖关系示例:

  amdgpu_cs_ioctl()          ← 用户空间IOCTL入口
    ├── amdgpu_ctx_mgr()     ← 上下文管理
    ├── amdgpu_bo_list()     ← BO列表管理
    ├── amdgpu_sched_run_job() ← 调度器执行
    │     ├── amdgpu_ring_emit()    ← Ring提交
    │     ├── amdgpu_fence_emit()   ← Fence管理
    │     └── amdgpu_vm_flush()     ← TLB刷新
    └── amdgpu_job_timedout()  ← 超时处理
          └── amdgpu_gpu_recover() ← GPU恢复
                ├── amdgpu_reset_cancel()  ← 取消操作
                ├── amdgpu_ring_backup()   ← Ring备份
                └── amdgpu_ring_restore()  ← Ring恢复

  如果修改 amdgpu_vm_flush(), 影响范围:
    直接: amdgpu_cs_ioctl → amdgpu_sched_run_job → amdgpu_vm_flush
    间接: 所有使用 amdgpu_vm_flush 的函数
    测试映射: gem_exec_basic, gem_exec_fence, gem_busy
```

映射矩阵的维护方法:

  1) 静态插桩分析法:
     gcc -finstrument-functions 编译测试代码
     运行测试 → 记录函数调用序列 → 建立映射关系

  2) 覆盖率数据推导法:
     gcov 收集每个测试的覆盖率
     分析测试覆盖的文件/函数 → 建立映射
     lcov --extract 按测试拆分覆盖率数据

  3) 符号解析法:
     nm 分析测试二进制中的符号引用
     grep 测试代码中的API调用
     cscope/cflow 分析测试与源代码的调用链

  4) 注释标注法:
     // TEST_COVERS: amdgpu_cs.c, amdgpu_bo.c
     // TEST_COVERS_FUNC: amdgpu_cs_ioctl
     在测试用例的注释中标注覆盖的代码范围
```

#### 3.3 回归测试选择算法

```
RTS算法对比:

  ┌──────────────────┬────────────────────────────────────────────────────┐
  │ 算法             │ 说明                                                │
  ├──────────────────┼────────────────────────────────────────────────────┤
  │ Retest-All       │ 重新运行所有测试用例                                │
  │                  │ 召回率=100%, 精确率=low, 效率=low                 │
  ├──────────────────┼────────────────────────────────────────────────────┤
  │ Random Selection │ 随机选择部分测试用例                                │
  │                  │ 召回率=low, 精确率=low, 效率=high                 │
  ├──────────────────┼────────────────────────────────────────────────────┤
  │ Safe Selection   │ 选择所有可能受影响的测试用例                        │
  │                  │ 召回率=100%, 精确率=medium, 效率=medium           │
  │                  │ 基于静态分析，不遗漏任何潜在影响                     │
  ├──────────────────┼────────────────────────────────────────────────────┤
  │ Dataflow-based   │ 基于数据依赖关系选择测试用例                        │
  │                  │ 召回率=high, 精确率=high, 效率=medium             │
  │                  │ 分析变量的定义-使用链                               │
  ├──────────────────┼────────────────────────────────────────────────────┤
  │ Firewall-based   │ 基于模块间接口的防火墙选择                          │
  │                  │ 召回率=high, 精确率=high, 效率=high               │
  │                  │ 只测试跨越模块边界的交互                             │
  └──────────────────┴────────────────────────────────────────────────────┘

安全RTS算法实现 (Python):

```python
#!/usr/bin/env python3
"""驱动程序安全回归测试选择算法"""

import os
import re
import json
import subprocess
from pathlib import Path
from collections import defaultdict


class SafeRTS:
    def __init__(self, repo_root, test_map_file):
        self.repo_root = Path(repo_root)
        self.test_map = self._load_test_map(test_map_file)
        self.dep_graph = {}
        self.changed_files = set()
        self.affected_files = set()
        self.selected_tests = set()

    def _load_test_map(self, path):
        with open(path) as f:
            data = json.load(f)
        result = {}
        for entry in data:
            test_name = entry["test_name"]
            for source_file in entry.get("covered_files", []):
                if source_file not in result:
                    result[source_file] = []
                result[source_file].append(test_name)
        return result

    def detect_changes(self, base_commit, head_commit):
        """检测两个版本间的变更文件"""
        result = subprocess.run(
            ["git", "diff", "--name-only", base_commit, head_commit],
            capture_output=True, text=True, cwd=self.repo_root
        )
        self.changed_files = set(
            f for f in result.stdout.splitlines()
            if f.startswith("drivers/gpu/drm/amd/")
        )
        return self.changed_files

    def build_dep_graph(self, module_dir="drivers/gpu/drm/amd/amdgpu"):
        """构建模块内部的文件依赖图"""
        cwd = self.repo_root / module_dir
        files = list(cwd.glob("*.c"))
        self.dep_graph = defaultdict(set)
        include_pattern = re.compile(r'#include\s+"([^"]+\.h)"')
        for f in files:
            content = f.read_text()
            for match in include_pattern.finditer(content):
                header = match.group(1)
                header_path = cwd / header
                c_file = header_path.with_suffix(".c")
                if c_file.exists():
                    self.dep_graph[str(f.relative_to(self.repo_root))].add(
                        str(c_file.relative_to(self.repo_root))
                    )
        return self.dep_graph

    def transitive_closure(self):
        """计算变更文件的传递闭包 (受影响的所有文件)"""
        affected = set(self.changed_files)
        worklist = list(self.changed_files)
        while worklist:
            current = worklist.pop()
            for dependent, dependencies in self.dep_graph.items():
                if current in dependencies and dependent not in affected:
                    affected.add(dependent)
                    worklist.append(dependent)
        self.affected_files = affected
        return affected

    def select_tests(self):
        """根据受影响文件选择测试用例"""
        for affected_file in self.affected_files:
            relative_path = str(Path(affected_file))
            if relative_path in self.test_map:
                self.selected_tests.update(self.test_map[relative_path])
        return self.selected_tests

    def prioritize_by_risk(self, history_data=None):
        """按风险对测试用例排序"""
        risk_scores = defaultdict(int)
        for test in self.selected_tests:
            if history_data and test in history_data:
                risk_scores[test] = history_data[test].get("failure_rate", 0) * 10
                risk_scores[test] += history_data[test].get("recent_failures", 0) * 5
            risk_scores[test] += 1
        sorted_tests = sorted(
            self.selected_tests,
            key=lambda t: risk_scores.get(t, 0),
            reverse=True
        )
        return sorted_tests

    def generate_test_plan(self, base_commit, head_commit, history_path=None):
        """生成完整的回归测试计划"""
        changed = self.detect_changes(base_commit, head_commit)
        if not changed:
            return {"test_all": False, "selected_tests": [], "message": "无变更"}

        _ = self.transitive_closure()
        selected = self.select_tests()

        plan = {
            "test_all": False,
            "base_commit": base_commit,
            "head_commit": head_commit,
            "changed_files": len(changed),
            "affected_files": len(self.affected_files),
            "selected_tests": list(selected),
            "test_count": len(selected),
        }

        if len(selected) > 500:
            plan["test_all"] = True
            plan["message"] = "受影响范围过大，执行完整回归"
        elif len(selected) == 0:
            plan["message"] = "未匹配到相关测试"
        else:
            history_data = None
            if history_path:
                try:
                    with open(history_path) as f:
                        history_data = json.load(f)
                except (FileNotFoundError, json.JSONDecodeError):
                    pass
            sorted_tests = self.prioritize_by_risk(history_data)
            plan["prioritized_tests"] = sorted_tests
            plan["message"] = f"选择 {len(selected)} 个测试用例"
        return plan


if __name__ == "__main__":
    rts = SafeRTS("/home/user/linux", "test_map.json")
    plan = rts.generate_test_plan("v6.5-rc1", "v6.5-rc2")
    print(f"测试计划: {json.dumps(plan, indent=2, ensure_ascii=False)}")
```

                UNIQUE(commit_hash, test_name, gpu_model)
            );
            CREATE TABLE IF NOT EXISTS baseline (
                test_name TEXT NOT NULL,
                gpu_model TEXT NOT NULL,
                expected_status TEXT NOT NULL DEFAULT 'pass',
                median_duration_ms INTEGER,
                stability_score REAL DEFAULT 1.0,
                last_updated TEXT,
                run_count INTEGER DEFAULT 0,
                pass_count INTEGER DEFAULT 0,
                PRIMARY KEY(test_name, gpu_model)
            );
            CREATE TABLE IF NOT EXISTS regressions (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                commit_hash TEXT NOT NULL,
                test_name TEXT NOT NULL,
                gpu_model TEXT NOT NULL,
                old_status TEXT,
                new_status TEXT,
                regression_type TEXT,
                detected_at TEXT NOT NULL
            );
        """)
        self.conn.commit()

    def record_result(self, commit_hash, test_name, gpu_model, status, duration_ms):
        now = datetime.now().isoformat()
        cur = self.conn.cursor()
        cur.execute(
            """INSERT OR REPLACE INTO test_runs
               (commit_hash, test_name, gpu_model, status, duration_ms, timestamp)
               VALUES (?, ?, ?, ?, ?, ?)""",
            (commit_hash, test_name, gpu_model, status, duration_ms, now)
        )
        cur.execute(
            """INSERT OR REPLACE INTO baseline (test_name, gpu_model, last_updated)
               VALUES (?, ?, ?)""",
            (test_name, gpu_model, now)
        )
        self._update_baseline(test_name, gpu_model, status, duration_ms)
        self.conn.commit()

    def _update_baseline(self, test_name, gpu_model, status, duration_ms):
        cur = self.conn.cursor()
        row = cur.execute(
            "SELECT run_count, pass_count FROM baseline WHERE test_name=? AND gpu_model=?",
            (test_name, gpu_model)
        ).fetchone()
        if row:
            run_count, pass_count = row
            new_run = run_count + 1
            new_pass = pass_count + (1 if status == "pass" else 0)
            stability = new_pass / new_run if new_run > 0 else 0
            cur.execute(
                """UPDATE baseline SET run_count=?, pass_count=?,
                   stability_score=?, expected_status=?
                   WHERE test_name=? AND gpu_model=?""",
                (new_run, new_pass, stability,
                 "pass" if stability >= 0.8 else "flaky",
                 test_name, gpu_model)
            )
        else:
            cur.execute(
                """INSERT INTO baseline (test_name, gpu_model, run_count, pass_count,
                   stability_score, expected_status)
                   VALUES (?, ?, 1, ?, ?, ?)""",
                (test_name, gpu_model,
                 1 if status == "pass" else 0,
                 1.0 if status == "pass" else 0.0,
                 "pass" if status == "pass" else "flaky")
            )

    def detect_regression(self, commit_hash, test_name, gpu_model, status, duration_ms):
        """检测是否为回归"""
        cur = self.conn.cursor()
        row = cur.execute(
            """SELECT expected_status, stability_score, median_duration_ms
               FROM baseline WHERE test_name=? AND gpu_model=?""",
            (test_name, gpu_model)
        ).fetchone()

        if not row:
            return {"is_regression": False, "reason": "NO_BASELINE"}

        expected_status, stability, median_ms = row
        regressions = []

        new_status = "FAIL" if status != "pass" else "PASS"
        old_status = expected_status.upper()

        if status != "pass" and expected_status == "pass":
            regressions.append({
                "type": "PASS_TO_FAIL",
                "severity": "HIGH",
                "detail": f"从PASS变为FAIL (稳定性: {stability:.2f})"
            })

        if status == "pass" and expected_status == "pass" and median_ms:
            ratio = duration_ms / median_ms
            if ratio > 2.0:
                regressions.append({
                    "type": "PERFORMANCE_REGRESSION",
                    "severity": "MEDIUM",
                    "detail": f"执行时间从{median_ms}ms变为{duration_ms}ms (增长{ratio:.1f}倍)"
                })

        if regressions:
            self._record_regression(
                commit_hash, test_name, gpu_model,
                old_status, new_status,
                regressions[0]["type"]
            )

        return {
            "is_regression": len(regressions) > 0,
            "regressions": regressions,
            "baseline_status": expected_status,
            "current_status": status,
            "stability_score": stability
        }

    def _record_regression(self, commit_hash, test_name, gpu_model,
                           old_status, new_status, reg_type):
        cur = self.conn.cursor()
        cur.execute(
            """INSERT INTO regressions
               (commit_hash, test_name, gpu_model, old_status, new_status,
                regression_type, detected_at)
               VALUES (?, ?, ?, ?, ?, ?, ?)""",
            (commit_hash, test_name, gpu_model, old_status, new_status,
             reg_type, datetime.now().isoformat())
        )
        self.conn.commit()

    def flaky_filter(self, results, max_failure_rate=0.2):
        """过滤掉已知的Flaky测试"""
        filtered = []
        for result in results:
            row = self.conn.execute(
                "SELECT stability_score FROM baseline WHERE test_name=? AND gpu_model=?",
                (result["test_name"], result["gpu_model"])
            ).fetchone()
            if row and row[0] < (1 - max_failure_rate):
                result["flaky"] = True
                result["stability"] = row[0]
            filtered.append(result)
        return filtered

    def regression_summary(self, commit_hash):
        """生成回归摘要报告"""
        cur = self.conn.cursor()
        rows = cur.execute(
            """SELECT test_name, gpu_model, old_status, new_status, regression_type, detected_at
               FROM regressions WHERE commit_hash=?
               ORDER BY detected_at DESC""",
            (commit_hash,)
        ).fetchall()

        summary = {
            "commit": commit_hash,
            "total_regressions": len(rows),
            "by_type": {},
            "by_gpu": {},
            "details": []
        }

        for test_name, gpu_model, old_st, new_st, reg_type, detected in rows:
            summary["by_type"][reg_type] = summary["by_type"].get(reg_type, 0) + 1
            summary["by_gpu"][gpu_model] = summary["by_gpu"].get(gpu_model, 0) + 1
            summary["details"].append({
                "test": test_name,
                "gpu": gpu_model,
                "from": old_st,
                "to": new_st,
                "type": reg_type,
                "time": detected
            })
        return summary

    def weekly_trend(self, days=7):
        """生成周回归趋势"""
        since = (datetime.now() - timedelta(days=days)).isoformat()
        cur = self.conn.cursor()
        rows = cur.execute(
            """SELECT DATE(detected_at) as day, regression_type, COUNT(*) as cnt
               FROM regressions WHERE detected_at > ?
               GROUP BY day, regression_type ORDER BY day""",
            (since,)
        ).fetchall()
        trend = {}
        for day, rtype, cnt in rows:
            if day not in trend:
                trend[day] = {}
            trend[day][rtype] = cnt
        return trend

#!/usr/bin/env python3
"""Flaky测试检测与隔离工具"""

import json
import subprocess
from pathlib import Path
from collections import defaultdict


class FlakyDetector:
    def __init__(self, history_db="regression.db"):
        self.history_db = Path(history_db)
        self.flaky_tests = {}

    def detect(self, min_runs=10, max_pass_rate=0.8):
        """识别Flaky测试"""
        import sqlite3
        conn = sqlite3.connect(self.history_db)
        cur = conn.cursor()
        rows = cur.execute(
            """SELECT test_name, gpu_model, run_count, pass_count
               FROM baseline WHERE run_count >= ? AND
               CAST(pass_count AS REAL) / run_count > 0.05 AND
               CAST(pass_count AS REAL) / run_count < ?""",
            (min_runs, max_pass_rate)
        ).fetchall()

        flaky = defaultdict(list)
        for test_name, gpu_model, runs, passes in rows:
            rate = passes / runs
            key = f"{test_name}@{gpu_model}"
            flaky[key] = {
                "test_name": test_name,
                "gpu_model": gpu_model,
                "run_count": runs,
                "pass_count": passes,
                "pass_rate": round(rate, 3),
                "status": "flaky"
            }
        conn.close()
        self.flaky_tests = dict(flaky)
        return self.flaky_tests

    def isolate_flaky(self, flaky_list):
        """将Flaky测试隔离到单独的套件"""
        isolated = {"smoke": [], "regression": [], "flaky": []}
        for test_key, info in flaky_list.items():
            if info["pass_rate"] < 0.3:
                isolated["flaky"].append(test_key)
            elif info["pass_rate"] < 0.6:
                isolated["regression"].append(test_key)
            else:
                isolated["smoke"].append(test_key)
        return isolated

    def rerun_flaky(self, test_list, max_retries=5):
        """重复运行Flaky测试以确认稳定性"""
        results = {}
        for test in test_list:
            passes = 0
            for i in range(max_retries):
                result = subprocess.run(
                    ["./run_single_test.sh", test],
                    capture_output=True, text=True, timeout=60
                )
                if result.returncode == 0:
                    passes += 1
            results[test] = {
                "passes": passes,
                "total": max_retries,
                "pass_rate": passes / max_retries
            }
        return results

    def mark_fixed(self, test_key):
        """标记Flaky测试已修复"""
        import sqlite3
        conn = sqlite3.connect(self.history_db)
        cur = conn.cursor()
        test_name, gpu_model = test_key.split("@")
        cur.execute(
            "UPDATE baseline SET stability_score=1.0 WHERE test_name=? AND gpu_model=?",
            (test_name, gpu_model)
        )
        conn.commit()
        conn.close()


if __name__ == "__main__":
    detector = FlakyDetector()
    flaky = detector.detect(min_runs=10, max_pass_rate=0.85)
    print(f"检测到 {len(flaky)} 个Flaky测试")
    for name, info in list(flaky.items())[:5]:
        print(f"  {name}: 通过率={info['pass_rate']:.1%}")

  │                                                                     │

  │  审核人: ________________________________                           │
  └─────────────────────────────────────────────────────────────────────┘

  步骤6: 回归测试资产更新

#### 7.4 回归测试自动化脚本

以下是一个集成化的回归测试自动化脚本，包含变更分析、测试执行和报告生成的全流程。

```python
#!/usr/bin/env python3
"""
AMDGPU Regression Test Automation Suite
自动完成: 代码变更分析 → RTS选择 → 分层执行 → 结果比对 → 报告生成
"""

import os
import sys
import json
import time
import hashlib
import logging
import subprocess
import argparse
from pathlib import Path
from datetime import datetime
from typing import List, Dict, Optional, Tuple


class AMDGPURegressionRunner:
    """AMDGPU回归测试全自动运行器"""

    def __init__(self, repo_root: str, gpu_model: str, base_commit: str):
        self.repo_root = Path(repo_root)
        self.gpu_model = gpu_model
        self.base_commit = base_commit
        self.results_dir = self.repo_root / "regression_results"
        self.results_dir.mkdir(exist_ok=True)
        self.session_id = datetime.now().strftime("%Y%m%d_%H%M%S")
        self.logger = self._setup_logger()

    def _setup_logger(self) -> logging.Logger:
        logger = logging.getLogger(f"regression_{self.session_id}")
        logger.setLevel(logging.DEBUG)
        fh = logging.FileHandler(
            self.results_dir / f"session_{self.session_id}.log"
        )
        fh.setFormatter(logging.Formatter(
            "%(asctime)s [%(levelname)s] %(message)s"
        ))
        logger.addHandler(fh)
        return logger

    def analyze_changes(self, head_commit: str) -> Dict:
        """分析两个commit之间的变更"""
        result = subprocess.run(
            ["git", "diff", "--stat", f"{self.base_commit}..{head_commit}",
             "--", "drivers/gpu/drm/amd/"],
            capture_output=True, text=True, cwd=self.repo_root
        )
        changed_files = []
        for line in result.stdout.strip().split("\n"):
            if line and "changed" not in line:
                parts = line.split("|")
                if len(parts) >= 1:
                    changed_files.append(parts[0].strip())

        detail = subprocess.run(
            ["git", "diff", "--name-only",
             f"{self.base_commit}..{head_commit}",
             "--", "drivers/gpu/drm/amd/"],
            capture_output=True, text=True, cwd=self.repo_root
        )
        all_files = detail.stdout.strip().split("\n")

        modules_affected = set()
        for f in all_files:
            if f.startswith("drivers/gpu/drm/amd/amdgpu/"):
                modules_affected.add("amdgpu")
            elif f.startswith("drivers/gpu/drm/amd/display/"):
                modules_affected.add("display")
            elif f.startswith("drivers/gpu/drm/amd/pm/"):
                modules_affected.add("power")
            elif f.startswith("drivers/gpu/drm/amd/amdkfd/"):
                modules_affected.add("kfd")

        self.logger.info(
            f"变更分析: {len(all_files)} 个文件, "
            f"影响模块: {modules_affected}"
        )
        return {
            "commit": head_commit,
            "base": self.base_commit,
            "files_changed": len(all_files),
            "changed_files": all_files,
            "modules_affected": list(modules_affected)
        }

    def select_regression_tests(self, analysis: Dict) -> Dict[str, List[str]]:
        """根据变更分析选择回归测试用例"""
        test_map = {
            "amdgpu": {
                "smoke": [
                    "gem_basic@create", "gem_basic@close",
                    "gem_ring@exec"
                ],
                "full": [
                    "gem_basic@*", "gem_exec@*", "gem_ring@*",
                    "gem_mmap@*", "gem_sync@*", "gem_sched@*",
                    "amdgpu_*@*"
                ]
            },
            "display": {
                "smoke": [
                    "kms_flip@basic", "kms_atomic@basic"
                ],
                "full": [
                    "kms_*@*", "amdgpu_dm@*"
                ]
            },
            "power": {
                "smoke": [
                    "pm_backlight@basic", "pm_rpm@basic"
                ],
                "full": [
                    "pm_*@*"
                ]
            },
            "kfd": {
                "smoke": [
                    "kfd_test@basic"
                ],
                "full": [
                    "kfd_*@*"
                ]
            }
        }

        suite = {"smoke": [], "targeted": [], "full": [], "extended": []}
        affected = analysis["modules_affected"]

        for module in affected:
            if module in test_map:
                suite["smoke"].extend(test_map[module]["smoke"])
                suite["targeted"].extend(test_map[module]["full"])

        if len(affected) > 1:
            suite["full"].append("kms_*@*")
            suite["full"].append("gem_*@*")

        suite["extended"] = ["igt_runner@all"]

        self.logger.info(
            f"RTS选择: smoke={len(suite['smoke'])} tests, "
            f"targeted={len(suite['targeted'])} tests"
        )
        return suite

    def run_test_suite(self, suite: Dict[str, List[str]],
                       igt_binary: str = "igt_runner") -> Dict:
        """执行回归测试套件"""
        results = {}
        for level, tests in suite.items():
            self.logger.info(f"开始执行 {level} 级别回归测试")
            level_results = []
            for test in tests:
                if "*" in test:
                    pattern = test.replace("*", ".*")
                    matched = self._discover_tests(pattern, igt_binary)
                    for mt in matched:
                        r = self._run_single_test(mt, igt_binary, level)
                        level_results.append(r)
                else:
                    r = self._run_single_test(test, igt_binary, level)
                    level_results.append(r)
            results[level] = level_results
            self.logger.info(
                f"{level} 完成: "
                f"{sum(1 for r in level_results if r['status']=='PASS')}/"
                f"{len(level_results)} PASS"
            )
        return results

    def _discover_tests(self, pattern: str,
                        igt_binary: str) -> List[str]:
        """发现匹配模式的测试用例"""
        result = subprocess.run(
            [igt_binary, "--list", "--pattern", pattern],
            capture_output=True, text=True
        )
        return [t.strip() for t in result.stdout.split("\n")
                if t.strip()]

    def _run_single_test(self, test: str, igt_binary: str,
                         level: str) -> Dict:
        """运行单个测试"""
        start = time.time()
        try:
            result = subprocess.run(
                [igt_binary, "--run", test, "--device", self.gpu_model],
                capture_output=True, text=True, timeout=300
            )
            elapsed = time.time() - start
            status = "PASS" if result.returncode == 0 else "FAIL"
            return {
                "test": test, "status": status,
                "duration": round(elapsed, 2),
                "level": level,
                "output": result.stdout[-500:] if result.stdout else "",
                "error": result.stderr[-500:] if result.stderr else ""
            }
        except subprocess.TimeoutExpired:
            return {
                "test": test, "status": "TIMEOUT",
                "duration": 300.0, "level": level,
                "output": "", "error": "Timeout after 300s"
            }

    def compare_with_baseline(self, results: Dict,
                              baseline_file: str = "baseline.json"
                              ) -> Tuple[Dict, List[str]]:
        """比对结果与基线，检测回归缺陷"""
        baseline_path = self.results_dir / baseline_file
        regressions = []
        report = {
            "total": 0, "passed": 0, "failed": 0,
            "timeout": 0, "regressions": 0
        }

        if not baseline_path.exists():
            self.logger.warning("基线文件不存在，跳过比对")
            return report, regressions

        with open(baseline_path) as f:
            baseline = json.load(f)

        for level, tests in results.items():
            for t in tests:
                report["total"] += 1
                if t["status"] == "PASS":
                    report["passed"] += 1
                elif t["status"] == "FAIL":
                    report["failed"] += 1
                elif t["status"] == "TIMEOUT":
                    report["timeout"] += 1

                bl_key = f"{t['test']}@{self.gpu_model}"
                if bl_key in baseline:
                    bl_status = baseline[bl_key]["status"]
                    if bl_status == "PASS" and t["status"] == "FAIL":
                        regressions.append({
                            "test": t["test"],
                            "expected": bl_status,
                            "actual": t["status"],
                            "type": "PASS_TO_FAIL"
                        })
                        report["regressions"] += 1

        return report, regressions

    def generate_report(self, analysis: Dict, suite: Dict,
                        results: Dict, report: Dict,
                        regressions: List[str]) -> str:
        """生成Markdown格式的回归测试报告"""
        now = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
        lines = [
            f"# AMDGPU回归测试报告",
            f"",
            f"**会话ID**: {self.session_id}",
            f"**GPU型号**: {self.gpu_model}",
            f"**基线Commit**: {self.base_commit[:12]}",
            f"**测试Commit**: {analysis['commit'][:12]}",
            f"**报告生成时间**: {now}",
            f"",
            f"## 变更摘要",
            f"",
            f"- 变更文件数: {analysis['files_changed']}",
            f"- 影响模块: {', '.join(analysis['modules_affected'])}",
            f"",
            f"## 执行统计",
            f"",
            f"| 指标 | 数值 |",
            f"|------|------|",
            f"| 总测试数 | {report['total']} |",
            f"| 通过 | {report['passed']} |",
            f"| 失败 | {report['failed']} |",
            f"| 超时 | {report['timeout']} |",
            f"| 回归缺陷 | {report['regressions']} |",
        ]

        if regressions:
            lines.extend([
                f"",
                f"## 回归缺陷列表",
                f"",
                f"| 测试用例 | 期望结果 | 实际结果 |",
                f"|----------|----------|----------|",
            ])
            for r in regressions:
                lines.append(
                    f"| {r['test']} | {r['expected']} | "
                    f"{r['actual']} |"
                )

        lines.extend([f"", f"## 结论", f""])
        if report["regressions"] > 0:
            lines.append("**阻塞**: 存在回归缺陷，请修复后重新提交。")
        else:
            lines.append("**通过**: 未检测到回归缺陷，变更可合并。")

        report_content = "\n".join(lines)
        report_path = (self.results_dir /
                       f"report_{self.session_id}.md")
        with open(report_path, "w") as f:
            f.write(report_content)

        self.logger.info(f"报告已生成: {report_path}")
        return report_content


if __name__ == "__main__":
    parser = argparse.ArgumentParser(
        description="AMDGPU Regression Test Runner"
    )
    parser.add_argument("--gpu", required=True,
                        help="GPU model identifier")
    parser.add_argument("--base", required=True,
                        help="Baseline commit hash")
    parser.add_argument("--head", required=True,
                        help="Head commit hash to test")
    parser.add_argument("--repo", default=".",
                        help="Repository root path")

    args = parser.parse_args()

    runner = AMDGPURegressionRunner(args.repo, args.gpu, args.base)
    analysis = runner.analyze_changes(args.head)
    suite = runner.select_regression_tests(analysis)
    results = runner.run_test_suite(suite)

    report_count, regressions = runner.compare_with_baseline(results)
    report = runner.generate_report(
        analysis, suite, results, report_count, regressions
    )
    print(report)
```

```python
def analyze_regression_trend(history_file: str) -> None:
    """分析回归测试历史趋势"""
    import sqlite3
    from datetime import datetime, timedelta

    conn = sqlite3.connect(history_file)
    cur = conn.cursor()

    cur.execute("""
        SELECT
            strftime('%Y-%W', run_date) as week,
            COUNT(*) as total,
            SUM(CASE WHEN status = 'FAIL' THEN 1 ELSE 0 END) as failures,
            SUM(CASE WHEN status = 'FLAKY' THEN 1 ELSE 0 END) as flaky
        FROM test_runs
        WHERE run_date > date('now', '-90 days')
        GROUP BY week
        ORDER BY week
    """)

    print("回归测试90天趋势分析:")
    print(f"{'周':<12} {'总数':<8} {'失败':<8} {'Flaky':<8} "
          f"{'失败率':<10} {'Flaky率'}")
    print("-" * 55)
    for row in cur.fetchall():
        week, total, failures, flaky = row
        fail_rate = failures / total * 100 if total > 0 else 0
        flaky_rate = flaky / total * 100 if total > 0 else 0
        print(f"{week:<12} {total:<8} {failures:<8} {flaky:<8} "
              f"{fail_rate:.1f}%{'':<5} {flaky_rate:.1f}%")

    cur.execute("""
        SELECT test_name, COUNT(*) as fail_count
        FROM test_runs
        WHERE status = 'FAIL'
            AND run_date > date('now', '-30 days')
        GROUP BY test_name
        ORDER BY fail_count DESC
        LIMIT 10
    """)

    print("\n月度最不稳定测试 (Top 10):")
    print(f"{'测试名称':<30} {'失败次数'}")
    print("-" * 40)
    for row in cur.fetchall():
        print(f"{row[0]:<30} {row[1]}")

    conn.close()


if __name__ == "__main__":
    analyze_regression_trend("regression_history.db")

### 11. 扩展思考

1. **安全回归测试 (Safe RTS) 的局限性**: SafeRTS算法保证了不会漏报任何可能受影响的测试, 但代价是选择了过多的测试用例（可能达到全量的60-70%）。在实际AMDGPU驱动开发中, 如何在安全性和效率之间找到最佳平衡点? 是否可以引入增量式依赖分析来减少重复计算?

2. **跨架构回归测试策略**: AMDGPU驱动需要支持GCN、RDNA、CDNA三种完全不同的GPU架构。同一个代码变更可能在不同架构上产生完全不同的影响。如何设计一个跨架构的回归测试策略, 使得在RDNA上验证通过的变更不会在CDNA上引入回归?

3. **机器学习RTS的冷启动问题**: ML-based RTS需要大量的历史数据来训练模型。对于一个新启动的GPU驱动测试项目, 缺少历史数据时应该如何处理? 是否可以先用传统SafeRTS积累数据, 然后渐进式切换到ML增强的RTS?

4. **回归测试与混沌工程的结合**: 在GPU驱动的压力测试中, 能否引入混沌工程的思想, 在回归测试执行过程中随机注入GPU hang、内存ECC错误、热保护触发等故障, 验证驱动在异常条件下的恢复能力? 这将在多大程度上提升回归测试的可靠性?`
