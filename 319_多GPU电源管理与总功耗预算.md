# 多 GPU 电源管理与总功耗预算

## 1. 多 GPU 系统概述

### 1.1 多 GPU 应用场景

多 GPU 系统广泛应用于以下场景：

```text
场景                          GPU 数量   电源管理需求
──────────────────────────────────────────────────────────
深度学习训练                    2~8       各 GPU 独立功率控制
高性能计算 (HPC)              4~64       集群级功耗预算
科学研究 (分子模拟等)         2~16       精确功耗限制
加密货币挖矿                  6~12       最低功耗/最高能效
游戏多卡互联 (CF/SLI)         2~4        同步电压调节
虚拟化/云游戏                  1~8        动态功耗分配
嵌入式多显示                   2~4        严格 TDP 控制
```

### 1.3 关键功耗指标

```text
指标                  缩写    描述                        典型值范围
──────────────────────────────────────────────────────────────────────
单卡最大功耗          TBP     显卡总板功耗上限              75~600W
典型功耗预算          TGP     实际运行功耗目标              TBP 的 60~90%
功耗上限              PPT      PowerTune 功耗上限           TBP 的 50~150%
总系统功耗预算        SPP      系统总电源容量              1000~4000W
PCIe 插槽功耗         PCIeP    PCIe 供电能力              75W (x16)
辅助供电接头          AUXP    辅助电源功率                150~600W
数据中心最大功耗      DCP      机架级功耗上限              10~50kW/rack
单 GPU 能效比         EFF      性能/功耗 比率              FLOPS/W

多 GPU 系统功耗预算计算:
  总预算 = N × TBP × 利用系数
  利用系数 = 0.7~0.9 (取决于工作负载类型)
  安全余量 = 总预算 × 1.2 (考虑峰值和冗余)
```

### 2.3 ROCm MIOpen 多 GPU 电源

ROCm 环境下，MIOpen 库通过 rocm-smi 管理多 GPU 电源：

```python
#!/usr/bin/env python3
"""
ROCm 多 GPU 电源管理示例
"""

import pyrocm_smi as rsmi
import time
from typing import List, Dict


class MultiGPUPowerManager:
    """多 GPU 电源管理器"""

    def __init__(self):
        rsmi.rsmi_init(0)
        self.gpu_count = rsmi.rsmi_num_monitor_devices()

    def get_gpu_count(self) -> int:
        return self.gpu_count

    def get_power_usage(self, gpu_id: int) -> float:
        """获取单 GPU 当前功耗 (W)"""
        try:
            power_uw = rsmi.rsmi_dev_power_ave_get(gpu_id, 0)
            return power_uw / 1_000_000.0
        except Exception:
            return 0.0

    def get_power_cap(self, gpu_id: int) -> float:
        """获取单 GPU 功耗上限 (W)"""
        try:
            cap_uw = rsmi.rsmi_dev_power_cap_get(gpu_id, 0)
            return cap_uw / 1_000_000.0
        except Exception:
            return 0.0

    def set_power_cap(self, gpu_id: int, watts: float) -> bool:
        """设置单 GPU 功耗上限"""
        try:
            cap_uw = int(watts * 1_000_000)
            rsmi.rsmi_dev_power_cap_set(gpu_id, cap_uw)
            return True
        except Exception:
            return False

    def get_total_power(self) -> float:
        """获取所有 GPU 总功耗"""
        total = 0.0
        for i in range(self.gpu_count):
            total += self.get_power_usage(i)
        return total

    def set_uniform_power_cap(self, watts: float):
        """统一设置所有 GPU 功耗上限"""
        for i in range(self.gpu_count):
            self.set_power_cap(i, watts)
            print(f"  GPU {i}: 功耗上限 = {watts}W")

    def set_proportional_power_cap(self, total_budget: float):
        """按比例分配总功耗预算"""
        # 获取当前各 GPU 使用率
        usage_rates = []
        for i in range(self.gpu_count):
            try:
                busy = rsmi.rsmi_dev_busy_percent_get(i)
            except Exception:
                busy = 0
            usage_rates.append(busy)

        total_usage = sum(usage_rates)
        if total_usage == 0:
            # 平均分配
            per_gpu = total_budget / self.gpu_count
            for i in range(self.gpu_count):
                self.set_power_cap(i, per_gpu)
        else:
            # 按使用率比例分配
            for i in range(self.gpu_count):
                ratio = usage_rates[i] / total_usage
                gpu_budget = total_budget * ratio
                self.set_power_cap(i, gpu_budget)
                print(f"  GPU {i}: 使用率={usage_rates[i]}%, 分配={gpu_budget:.1f}W")

    def report(self) -> Dict:
        """生成多 GPU 电源报告"""
        report = {
            "gpu_count": self.gpu_count,
            "gpus": []
        }
        total_power = 0.0
        for i in range(self.gpu_count):
            power = self.get_power_usage(i)
            cap = self.get_power_cap(i)
            try:
                temp_uw = rsmi.rsmi_dev_temp_metric_get(i, 0, rsmi.RSMI_TEMP_CURRENT)
                temp = temp_uw / 1000.0
            except Exception:
                temp = 0.0
            try:
                busy = rsmi.rsmi_dev_busy_percent_get(i)
            except Exception:
                busy = 0.0

            gpu_info = {
                "gpu_id": i,
                "power_w": round(power, 1),
                "cap_w": round(cap, 1),
                "temp_c": round(temp, 1),
                "busy_pct": round(busy, 1),
            }
            report["gpus"].append(gpu_info)
            total_power += power

        report["total_power_w"] = round(total_power, 1)
        return report

    def cleanup(self):
        rsmi.rsmi_shut_down()


def print_report(report: Dict):
    """打印多 GPU 电源报告"""
    print(f"\n{'='*60}")
    print(f"  多 GPU 电源报告 (共 {report['gpu_count']} 个 GPU)")
    print(f"{'='*60}")
    print(f"  {'GPU':<6} {'功耗(W)':<10} {'上限(W)':<10} {'温度(°C)':<10} {'使用率(%)':<10}")
    print(f"  {'-'*46}")
    for gpu in report['gpus']:
        print(f"  {gpu['gpu_id']:<6} {gpu['power_w']:<10.1f} {gpu['cap_w']:<10.1f} "
              f"{gpu['temp_c']:<10.1f} {gpu['busy_pct']:<10.1f}")
    print(f"  {'-'*46}")
    print(f"  总功耗: {report['total_power_w']:.1f} W")
    print(f"{'='*60}")


def main():
    manager = MultiGPUPowerManager()
    gpu_count = manager.get_gpu_count()
    print(f"检测到 {gpu_count} 个 GPU")

    # 获取初始状态
    report = manager.report()
    print_report(report)

    # 演示: 功率预算分配
    if gpu_count > 1:
        print("\n演示功率预算分配...")
        total_budget = 500  # 总预算 500W
        print(f"总功耗预算: {total_budget}W")
        manager.set_proportional_power_cap(total_budget)

        time.sleep(2)
        report = manager.report()
        print_report(report)

    manager.cleanup()


if __name__ == "__main__":
    main()
```

## 3. Linux 多 GPU 电源管理接口

### 3.1 sysfs 多 GPU 遍历

```bash
#!/bin/bash
# multi_gpu_power_scan.sh - 多 GPU 电源状态扫描

echo "=========================================="
echo "  多 GPU 电源状态扫描"
echo "=========================================="

for drm_dir in /sys/class/drm/card*/device; do
    if [ ! -d "$drm_dir" ]; then
        continue
    fi

    card_name=$(basename $(dirname "$drm_dir"))
    echo ""
    echo "--- $card_name ---"

    # GPU 名称
    if [ -f "$drm_dir/product_name" ]; then
        echo "  GPU: $(cat $drm_dir/product_name)"
    fi

    # GPU 使用率
    if [ -f "$drm_dir/gpu_busy_percent" ]; then
        echo "  使用率: $(cat $drm_dir/gpu_busy_percent)%"
    fi

    # hwmon 信息
    for hwmon in $drm_dir/hwmon/hwmon*/; do
        if [ ! -d "$hwmon" ]; then
            continue
        fi

        if [ -f "${hwmon}name" ]; then
            echo "  hwmon: $(cat ${hwmon}name)"
        fi

        # 功耗
        if [ -f "${hwmon}power1_average" ]; then
            power=$(cat ${hwmon}power1_average)
            power_w=$(echo "scale=2; $power / 1000000" | bc)
            echo "  功耗: ${power_w}W"
        fi

        if [ -f "${hwmon}power1_cap" ]; then
            cap=$(cat ${hwmon}power1_cap)
            cap_w=$(echo "scale=2; $cap / 1000000" | bc)
            echo "  功耗上限: ${cap_w}W"
        fi

        # 温度
        for i in 1 2 3 4 5 6 7; do
            if [ -f "${hwmon}temp${i}_input" ]; then
                temp=$(cat ${hwmon}temp${i}_input)
                temp_c=$(echo "scale=1; $temp / 1000" | bc)
                label="temp${i}"
                if [ -f "${hwmon}temp${i}_label" ]; then
                    label=$(cat ${hwmon}temp${i}_label)
                fi
                echo "  温度(${label}): ${temp_c}°C"
            fi
        done

        # 风扇
        if [ -f "${hwmon}fan1_input" ]; then
            echo "  风扇: $(cat ${hwmon}fan1_input) RPM"
        fi
    done

    # DPM 状态
    if [ -f "$drm_dir/pp_dpm_sclk" ]; then
        echo "  SCLK:"
        cat "$drm_dir/pp_dpm_sclk" | sed 's/^/    /'
    fi

    if [ -f "$drm_dir/pp_dpm_mclk" ]; then
        echo "  MCLK:"
        cat "$drm_dir/pp_dpm_mclk" | sed 's/^/    /'
    fi

    echo ""
done

echo "=========================================="
echo "  扫描完成"
echo "=========================================="
```

### 3.3 PMBus 电源监控

在服务器环境中，使用 PMBus 监控 GPU 电源：

```bash
#!/bin/bash
# pmbus_gpu_monitor.sh - 通过 PMBus 监控显卡电源

# 依赖: sensors-detect, lm-sensors, i2c-tools

# 检测 PMBus 设备
echo "检测 PMBus 设备..."
for bus in /sys/bus/i2c/devices/i2c-*; do
    bus_num=$(basename $bus | cut -d- -f2)
    echo "I2C Bus $bus_num:"
    i2cdetect -y $bus_num 2>/dev/null | head -20
done

# 如果 GPU VRM 使用 PMBus (如 IR35215, ADM1278 等)
# 可以直接读取电压、电流、功率

echo ""
echo "GPU 电压调节器 PMBus 信息:"

# 常见 GPU PMBus 地址
PMBUS_ADDRS="0x40 0x41 0x42 0x43 0x44 0x45 0x46 0x47"

for bus in /sys/bus/i2c/devices/i2c-*/; do
    bus_num=$(basename $bus | cut -d- -f2)
    for addr in $PMBUS_ADDRS; do
        reg_path="/sys/bus/i2c/devices/i2c-${bus_num}/${bus_num}-${addr//0x/}"
        if [ -d "$reg_path" ]; then
            echo "  Bus $bus_num Addr $addr: $(basename $(readlink $reg_path))"
            # 读取 VOUT (输出电压)
            if i2cget -y $bus_num $addr 0x8b w 2>/dev/null; then
                echo "    VOUT: $(i2cget -y $bus_num $addr 0x8b w)mV"
            fi
            # 读取 IOUT (输出电流)
            if i2cget -y $bus_num $addr 0x8c w 2>/dev/null; then
                echo "    IOUT: $(i2cget -y $bus_num $addr 0x8c w)mA"
            fi
            # 读取 POWER (输出功率)
            if i2cget -y $bus_num $addr 0x97 w 2>/dev/null; then
                echo "    POWER: $(i2cget -y $bus_num $addr 0x97 w)mW"
            fi
        fi
    done
done
```

### 4.2 动态功耗预算分配策略

```python
#!/usr/bin/env python3
"""
动态功耗预算分配器
根据 GPU 负载动态调整各 GPU 功耗上限
"""

import time
import threading
from typing import List, Dict, Optional


class DynamicPowerBudget:
    """动态功耗预算分配器"""

    def __init__(self, gpu_count: int, total_budget: float):
        self.gpu_count = gpu_count
        self.total_budget = total_budget
        self.gpu_power: List[float] = [0.0] * gpu_count
        self.gpu_usage: List[float] = [0.0] * gpu_count
        self.gpu_caps: List[float] = [total_budget / gpu_count] * gpu_count
        self.running = False
        self._lock = threading.Lock()

    def update_metrics(self, gpu_id: int, power_w: float, usage_pct: float):
        """更新 GPU 度量"""
        with self._lock:
            self.gpu_power[gpu_id] = power_w
            self.gpu_usage[gpu_id] = usage_pct

    def strategy_uniform(self) -> List[float]:
        """策略 1: 平均分配"""
        return [self.total_budget / self.gpu_count] * self.gpu_count

    def strategy_proportional(self) -> List[float]:
        """策略 2: 按使用率比例分配"""
        with self._lock:
            usage = self.gpu_usage[:]

        total_usage = sum(usage)
        if total_usage == 0:
            return self.strategy_uniform()

        caps = []
        remaining = self.total_budget
        for i in range(self.gpu_count):
            if i < self.gpu_count - 1:
                cap = self.total_budget * (usage[i] / total_usage)
            else:
                cap = remaining
            cap = min(cap, self.total_budget * 0.5)  # 单 GPU 不超过 50%
            cap = max(cap, 50.0)  # 单 GPU 最低 50W
            caps.append(cap)
            remaining -= cap

        return caps

    def strategy_weighted(self, weights: List[float]) -> List[float]:
        """策略 3: 按权重分配"""
        total_weight = sum(weights)
        caps = []
        remaining = self.total_budget
        for i in range(self.gpu_count):
            if i < self.gpu_count - 1:
                cap = self.total_budget * (weights[i] / total_weight)
            else:
                cap = remaining
            caps.append(cap)
            remaining -= cap
        return caps

    def strategy_power_efficiency(self) -> List[float]:
        """策略 4: 按能效分配 (给能效高的 GPU 更多功率)"""
        with self._lock:
            power = self.gpu_power[:]
            usage = self.gpu_usage[:]

        efficiency = []
        for i in range(self.gpu_count):
            if power[i] > 0:
                eff = usage[i] / power[i]
            else:
                eff = 0
            efficiency.append(eff)

        total_eff = sum(efficiency)
        if total_eff == 0:
            return self.strategy_uniform()

        caps = []
        for i in range(self.gpu_count):
            cap = self.total_budget * (efficiency[i] / total_eff)
            cap = min(cap, self.total_budget * 0.5)
            cap = max(cap, 50.0)
            caps.append(cap)

        return caps

    def apply_caps(self, caps: List[float]):
        """应用功耗上限"""
        with self._lock:
            self.gpu_caps = caps

    def print_allocation(self, caps: List[float], strategy_name: str):
        """打印分配结果"""
        print(f"\n策略: {strategy_name}")
```

### 4.3 功耗预算约束条件

```text
功耗预算约束条件:

1. 硬件约束 (绝对上限)
   每个 GPU power_cap_max：由 VRM 设计决定
   PCIe 插槽供电极限：75W (x16)
   辅助供电接头极限：150W (8-pin) / 300W (12VHPWR)

2. 热约束 (Thermal Design Point)
   机箱散热能力：TDP = 散热器的热阻 × (T_junction_max - T_ambient)
   典型值：280W (单 GPU 风冷) / 400W (水冷)

3. 电源约束 (Power Supply Unit)
   PSU 额定功率 = Σ(GPU TDP) × 1.2 + CPU TDP + 其他外设
   多 GPU 系统建议 PSU 余量 ≥ 20%

4. 软件约束
   PowerTune 层级写入范围：power_cap_min ~ power_cap_max
   OD 范围：由 vbios 和 SMU 固件限制

5. 工作负载约束
   深度学习训练：利用系数 0.85~0.95，持续高负载
   科学计算：利用系数 0.8~0.95，长时间运行
   云游戏：利用系数 0.4~0.7，间歇性负载
   加密货币挖矿：利用系数 0.95~1.0，满载恒定

约束优先级:
  硬件约束 > 热约束 > 电源约束 > 软件约束 > 工作负载约束
```

```python
#!/usr/bin/env python3
"""
多 GPU 电源实时仪表盘
"""

import subprocess
import json
import time
import sys
import os
from datetime import datetime
from typing import Dict, List


class MultiGPU_Dashboard:
    """多 GPU 电源实时仪表盘"""

    def __init__(self, gpu_count: int = 0):
        self.gpu_count = gpu_count
        self.history: List[Dict] = []
        self.max_history = 3600  # 保存 1 小时数据 (10s 间隔)

    def scan_gpus(self) -> int:
        """扫描系统中的 GPU 数量"""
        try:
            result = subprocess.run(
                ["rocm-smi", "--json", "--showid"],
                capture_output=True, text=True, timeout=5
            )
            data = json.loads(result.stdout)
            self.gpu_count = len(data)
        except Exception:
            # 备选: 扫描 sysfs
            import glob
            drm_dirs = glob.glob("/sys/class/drm/card*/device/gpu_busy_percent")
            # 过滤 renderD
            drm_devices = [d for d in glob.glob("/sys/class/drm/card?") if not "render" in d]
            self.gpu_count = len(drm_devices)
        return self.gpu_count

    def read_all_gpus(self) -> Dict:
        """读取所有 GPU 数据"""
        data = {}
        try:
            result = subprocess.run(
                ["rocm-smi", "--json", "--showpower", "--showtemp",
                 "--showmeminfo", "vram", "--showuse"],
                capture_output=True, text=True, timeout=5
            )
            raw = json.loads(result.stdout)

            timestamp = time.time()
            total_power = 0.0

            for card_id, card_data in raw.items():
                gpu_id = card_id.replace("card", "")
                try:
                    power_str = card_data.get("Power", "0 W")
                    power = float(power_str.replace(" W", ""))
                except (ValueError, AttributeError):
                    power = 0.0

                try:
                    temp_str = card_data.get("Temperature", "0.0°C")
                    temp = float(temp_str.replace("°C", ""))
                except (ValueError, AttributeError):
                    temp = 0.0

                try:
                    use_str = card_data.get("GPU Use", "0%")
                    usage = float(use_str.replace("%", ""))
                except (ValueError, AttributeError):
                    usage = 0.0

                data[gpu_id] = {
                    "power_w": power,
                    "temp_c": temp,
                    "usage_pct": usage,
                    "timestamp": timestamp,
                }
                total_power += power

            data["total_power_w"] = total_power
            data["timestamp"] = timestamp

        except Exception as e:
            data["error"] = str(e)

        return data

    def update(self):
        """更新数据并保存历史"""
        snapshot = self.read_all_gpus()
        self.history.append(snapshot)
        if len(self.history) > self.max_history:
            self.history.pop(0)

    def get_summary(self, data: Dict) -> str:
        """生成文本摘要"""
        lines = []
        lines.append(f"{'='*60}")
        lines.append(f"  多 GPU 电源仪表盘 - {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}")
        lines.append(f"{'='*60}")
        lines.append(f"  {'GPU':<6} {'功耗(W)':<12} {'温度(°C)':<12} {'使用率(%)':<10}")
        lines.append(f"  {'-'*40}")

        for key in sorted(data.keys()):
            if key in ("total_power_w", "timestamp", "error"):
                continue
            gpu = data[key]
            lines.append(f"  {key:<6} {gpu['power_w']:<10.1f}  {gpu['temp_c']:<10.1f}  {gpu['usage_pct']:<8.1f}")

        lines.append(f"  {'-'*40}")
        lines.append(f"  总功耗: {data.get('total_power_w', 0):.1f} W")
        lines.append(f"{'='*60}")
        return "\n".join(lines)

    def print_summary(self):
        """打印摘要"""
        if self.history:
            data = self.history[-1]
            if "error" not in data:
                os.system("clear" if os.name == "posix" else "cls")
                print(self.get_summary(data))

    def export_csv(self, filepath: str):
        """导出历史数据为 CSV"""
        if not self.history:
            print("无数据可导出")
            return

        with open(filepath, "w") as f:
            # 写入表头
            gpu_ids = [k for k in self.history[0].keys()
                      if k not in ("total_power_w", "timestamp", "error")]
            headers = ["timestamp"] + [f"gpu_{gid}_power_w" for gid in gpu_ids] + \
                      [f"gpu_{gid}_temp_c" for gid in gpu_ids] + \
                      [f"gpu_{gid}_usage_pct" for gid in gpu_ids] + \
                      ["total_power_w"]
            f.write(",".join(headers) + "\n")

            # 写入数据
            for snapshot in self.history:
                if "error" in snapshot:
                    continue
                row = [str(snapshot.get("timestamp", ""))]
                gpu_ids = sorted([k for k in snapshot.keys()
                                 if k not in ("total_power_w", "timestamp", "error")])
                for gid in gpu_ids:
                    row.append(str(snapshot[gid]["power_w"]))
                for gid in gpu_ids:
                    row.append(str(snapshot[gid]["temp_c"]))
                for gid in gpu_ids:
                    row.append(str(snapshot[gid]["usage_pct"]))
                row.append(str(snapshot.get("total_power_w", "")))
                f.write(",".join(row) + "\n")

        print(f"数据已导出: {filepath}")

    def run(self, interval: float = 2.0):
        """连续监控运行"""
        try:
            while True:
                self.update()
                self.print_summary()
                time.sleep(interval)
        except KeyboardInterrupt:
            print("\n停止监控")
            csv_path = f"multi_gpu_power_{int(time.time())}.csv"
            self.export_csv(csv_path)


def main():
    dashboard = MultiGPU_Dashboard()
    gpu_count = dashboard.scan_gpus()
    print(f"检测到 {gpu_count} 个 GPU")
    if gpu_count == 0:
        print("未检测到 GPU")
        sys.exit(1)

    print("开始实时监控 (Ctrl+C 停止)...")
    time.sleep(1)
    dashboard.run(interval=2.0)


if __name__ == "__main__":
    main()
```

### 7.3 功耗趋势分析

```python
#!/usr/bin/env python3
"""
多 GPU 功耗趋势分析
"""

import json
from typing import List, Dict
from collections import deque


class PowerTrendAnalyzer:
    """功耗趋势分析器"""

    def __init__(self, window_size: int = 60):
        self.window_size = window_size
        self.data: Dict[str, deque] = {}

    def add_snapshot(self, snapshot: Dict):
        """添加一个数据快照"""
        timestamp = snapshot.get("timestamp", 0)
        for key, value in snapshot.items():
            if key in ("timestamp", "error"):
                continue
            if key == "total_power_w":
                if key not in self.data:
                    self.data[key] = deque(maxlen=self.window_size)
                self.data[key].append((timestamp, value))
            else:
                for metric in ("power_w", "temp_c", "usage_pct"):
                    metric_key = f"{key}_{metric}"
                    if metric_key not in self.data:
                        self.data[metric_key] = deque(maxlen=self.window_size)
                    self.data[metric_key].append((timestamp, value[metric]))

    def get_trend(self, metric_key: str) -> str:
        """获取趋势方向"""
        if metric_key not in self.data or len(self.data[metric_key]) < 2:
            return "unknown"

        values = [v for _, v in self.data[metric_key]]
        if len(values) < 5:
            return "unknown"

        # 计算线性回归斜率
        n = len(values)
        x = list(range(n))
        y = values

        sum_x = sum(x)
        sum_y = sum(y)
        sum_xy = sum(x[i] * y[i] for i in range(n))
        sum_x2 = sum(x[i] ** 2 for i in range(n))

        try:
            slope = (n * sum_xy - sum_x * sum_y) / (n * sum_x2 - sum_x ** 2)
        except ZeroDivisionError:
            return "unknown"

        if slope > 0.5:
            return "up"
        elif slope < -0.5:
            return "down"
        else:
            return "stable"

    def get_statistics(self, metric_key: str) -> Dict:
        """获取统计信息"""
        if metric_key not in self.data or not self.data[metric_key]:
            return {}

        values = [v for _, v in self.data[metric_key]]
        n = len(values)
        if n == 0:
            return {}

        avg = sum(values) / n
        variance = sum((v - avg) ** 2 for v in values) / n
        std_dev = variance ** 0.5
        sorted_vals = sorted(values)
        median = sorted_vals[n // 2] if n % 2 else (sorted_vals[n // 2 - 1] + sorted_vals[n // 2]) / 2

        return {
            "min": min(values),
            "max": max(values),
            "avg": round(avg, 2),
            "median": round(median, 2),
            "std_dev": round(std_dev, 2),
            "trend": self.get_trend(metric_key),
            "samples": n,
        }

    def report(self) -> str:
        """生成分析报告"""
        lines = []
        lines.append("=" * 60)
        lines.append("  多 GPU 功耗趋势分析报告")
        lines.append("=" * 60)

        # 总功耗趋势
        if "total_power_w" in self.data:
            stats = self.get_statistics("total_power_w")
            lines.append(f"\n总功耗:")
            lines.append(f"  均值: {stats.get('avg', 'N/A')} W")
            lines.append(f"  峰值: {stats.get('max', 'N/A')} W")
            lines.append(f"  谷值: {stats.get('min', 'N/A')} W")
            lines.append(f"  趋势: {stats.get('trend', 'N/A')}")

        # 各 GPU 功耗
        gpu_ids = set()
        for key in self.data.keys():
            if "_power_w" in key and key != "total_power_w":
                gpu_ids.add(key.replace("_power_w", ""))

        for gpu_id in sorted(gpu_ids):
            power_key = f"{gpu_id}_power_w"
            temp_key = f"{gpu_id}_temp_c"
            usage_key = f"{gpu_id}_usage_pct"

            power_stats = self.get_statistics(power_key)
            temp_stats = self.get_statistics(temp_key)
            usage_stats = self.get_statistics(usage_key)

            lines.append(f"\nGPU {gpu_id}:")
            lines.append(f"  功耗: {power_stats.get('avg', 'N/A')}W "
                        f"(峰值 {power_stats.get('max', 'N/A')}W)")
            lines.append(f"  温度: {temp_stats.get('avg', 'N/A')}°C "
                        f"(峰值 {temp_stats.get('max', 'N/A')}°C)")
            lines.append(f"  使用率: {usage_stats.get('avg', 'N/A')}% "
                        f"(趋势 {usage_stats.get('trend', 'N/A')})")

        lines.append(f"\n采样点数: {self.data.get('total_power_w', deque(maxlen=0)).__len__()}")
        lines.append("=" * 60)

        return "\n".join(lines)


def analyze_power_log(csv_path: str) -> str:
    """从 CSV 文件分析功耗趋势"""
    import csv
    analyzer = PowerTrendAnalyzer()

    with open(csv_path) as f:
        reader = csv.DictReader(f)
        for row in reader:
            snapshot = {"timestamp": float(row.get("timestamp", 0))}
            total = 0.0
            for col, val in row.items():
                if col == "timestamp":
                    continue
                if col == "total_power_w":
                    try:
                        snapshot["total_power_w"] = float(val)
                    except ValueError:
                        pass
                elif "_power_w" in col:
                    gpu_id = col.replace("_power_w", "")
                    if gpu_id not in snapshot:
                        snapshot[gpu_id] = {}
                    try:
                        snapshot[gpu_id]["power_w"] = float(val)
                        total += float(val)
                    except ValueError:
                        pass
                elif "_temp_c" in col:
                    gpu_id = col.replace("_temp_c", "")
                    if gpu_id not in snapshot:
                        snapshot[gpu_id] = {}
                    try:
                        snapshot[gpu_id]["temp_c"] = float(val)
                    except ValueError:
                        pass
                elif "_usage_pct" in col:
                    gpu_id = col.replace("_usage_pct", "")
                    if gpu_id not in snapshot:
                        snapshot[gpu_id] = {}
                    try:
                        snapshot[gpu_id]["usage_pct"] = float(val)
                    except ValueError:
                        pass
            if "total_power_w" not in snapshot:
                snapshot["total_power_w"] = total
            analyzer.add_snapshot(snapshot)

    return analyzer.report()
```

### 11.2 练习二: 动态功耗预算分配模拟

```python
#!/usr/bin/env python3
"""
dynamic_budget_sim.py - 动态功耗预算分配模拟器
模拟不同策略在动态负载下的表现
"""

import random
import math
from typing import List, Dict


class BudgetSimulator:
    def __init__(self, gpu_count: int = 4, total_budget: float = 1200.0):
        self.gpu_count = gpu_count
        self.total_budget = total_budget
        self.strategies = {
            "uniform": self.strategy_uniform,
            "proportional": self.strategy_proportional,
            "weighted": self.strategy_weighted,
            "efficiency": self.strategy_efficiency,
        }

    def simulate_load(self) -> List[float]:
        loads = []
        for _ in range(self.gpu_count):
            loads.append(random.uniform(10.0, 100.0))
        return loads

    def strategy_uniform(self, loads: List[float]) -> List[float]:
        return [self.total_budget / self.gpu_count] * self.gpu_count

    def strategy_proportional(self, loads: List[float]) -> List[float]:
        total_load = sum(loads)
        if total_load == 0:
            return self.strategy_uniform(loads)
        return [self.total_budget * (l / total_load) for l in loads]

    def strategy_weighted(self, loads: List[float]) -> List[float]:
        total_load = sum(loads)
        if total_load == 0:
            return self.strategy_uniform(loads)
        weights = [min(100.0, l * 1.2) for l in loads]
        total_weight = sum(weights)
        return [self.total_budget * (w / total_weight) for w in weights]

    def strategy_efficiency(self, loads: List[float]) -> List[float]:
        total_load = sum(loads)
        if total_load == 0:
            return self.strategy_uniform(loads)
        eff_scores = []
        for l in loads:
            perf_per_watt = random.uniform(0.8, 1.2)
            eff_scores.append(l * perf_per_watt)
        total_eff = sum(eff_scores)
        return [self.total_budget * (e / total_eff) for e in eff_scores]

    def evaluate_strategy(self, strategy_name: str, loads: List[float],
                          budgets: List[float]) -> float:
        utilization = 0
        for load, budget in zip(loads, budgets):
            watt_per_load = budget / max(load, 1.0)
            utilization += min(100.0, watt_per_load * 10)
        return utilization / self.gpu_count

    def run_simulation(self, steps: int = 100) -> Dict[str, List[float]]:
        results = {name: [] for name in self.strategies}
        for _ in range(steps):
            loads = self.simulate_load()
            for name, strategy_fn in self.strategies.items():
                budgets = strategy_fn(loads)
                score = self.evaluate_strategy(name, loads, budgets)
                results[name].append(score)
        return results

    def print_results(self, results: Dict[str, List[float]]):
        print(f"{'策略':<20} {'平均利用率':<15} {'最大利用率':<15} {'最小利用率':<15} {'标准差':<15}")
        print("-" * 80)
        for name, scores in results.items():
            avg = sum(scores) / len(scores)
            mx = max(scores)
            mn = min(scores)
            std = math.sqrt(sum((s - avg) ** 2 for s in scores) / len(scores))
            print(f"{name:<20} {avg:<15.2f} {mx:<15.2f} {mn:<15.2f} {std:<15.2f}")


if __name__ == "__main__":
    sim = BudgetSimulator(gpu_count=4, total_budget=1200.0)
    print("=" * 80)
    print("动态功耗预算分配模拟器")
    print(f"GPU数量: 4, 总预算: 1200W")
    print("=" * 80)
    results = sim.run_simulation(1000)
    sim.print_results(results)
```

### 11.3 练习三: PMBus VRM 功耗监控

```bash
#!/bin/bash
# vrm_monitor.sh - 通过 PMBus 读取 GPU VRM 功耗
# 依赖: i2c-tools, lm-sensors
# 使用前先加载内核模块: modprobe i2c-dev

I2C_BUS=${1:-2}
VRM_ADDR=${2:-0x40}

detect_vrm() {
    echo "扫描 PMBus VRM 控制器..."
    for bus in /dev/i2c-*; do
        BUS_NUM=$(echo "$bus" | grep -oP '\d+$')
        for addr in $(seq 0x20 0x4F); do
            result=$(i2cget -y "$BUS_NUM" "$addr" 0x00 2>/dev/null)
            if [ $? -eq 0 ]; then
                echo "  发现设备: bus=${BUS_NUM} addr=0x${addr} (${result})"
            fi
        done
    done
}

read_vrm_power() {
    local bus=$1 addr=$2
    # PMBus READ_VOUT (0x8B): 读取电压
    VOUT_RAW=$(i2cget -y "$bus" "$addr" 0x8B w 2>/dev/null)
    # PMBus READ_IOUT (0x8C): 读取电流
    IOUT_RAW=$(i2cget -y "$bus" "$addr" 0x8C w 2>/dev/null)
    # PMBus READ_POUT (0x96): 直接读取功率
    POUT_RAW=$(i2cget -y "$bus" "$addr" 0x96 w 2>/dev/null)

    if [ -n "$POUT_RAW" ]; then
        VOUT=$(printf "%d" 0x${VOUT_RAW:2} 2>/dev/null)
        IOUT=$(printf "%d" 0x${IOUT_RAW:2} 2>/dev/null)
        POUT=$((VOUT * IOUT / 1000))
        echo "VOUT=${VOUT}mV IOUT=${IOUT}mA POUT=${POUT}mW"
    fi
}

echo "PMBus VRM 功耗监控工具"
echo "======================"

if [ "$1" = "scan" ]; then
    detect_vrm
    exit 0
fi

echo "监控 VRM (bus=${I2C_BUS}, addr=${VRM_ADDR})"
echo "时间戳, 电压(mV), 电流(mA), 功率(mW)"
echo "----------------------------------------"

while true; do
    read_vrm_power "$I2C_BUS" "$VRM_ADDR"
    sleep 2
done
```

## 12. 常见问题与排查

### 12.1 GPU 功耗读数不一致

```text
现象:
  rocm-smi 显示的功耗与 hwmon sysfs 读取的值不一致

原因分析:
  ┌─────────────────────────────────────────────────────────────┐
  │  读数来源         │ 采样位置          │ 精度    │ 更新频率 │
  ├────────────────────┼──────────────────┼─────────┼─────────┤
  │ rocm-smi          │ SMU 固件接口      │ ±1W     │ 100ms   │
  │ hwmon power1_input│ GPU VDDCR 电压域  │ ±3W     │ 500ms   │
  │ hwmon power2_input│ VDDCI/MEM 电压域  │ ±2W     │ 500ms   │
  │ PMBus VRM         │ VRM 输出侧        │ ±0.5W   │ 10ms    │
  │ PSU 读数          │ 交流输入端        │ ±5W     │ 1s      │
  └─────────────────────────────────────────────────────────────┘

排查步骤:
  1. 确认读取的是同一时间点的数据
  2. 检查是否使用了正确的 hwmon 索引
  3. 确认 GPU 处于稳定负载状态 (非瞬态)
  4. 校准: 使用精密万用表测量 12V 输入电流作为基准
```

### 12.2 多GPU功耗分配不均

```text
现象:
  同型号 GPU 在相同负载下功耗差异 > 10%

可能原因:
  - ASIC 质量差异 (binning): 不同 GPU 的漏电流不同
  - 散热条件差异: 风道设计导致温度不同, 影响漏电流
  - PCIe 插槽供电差异: 靠近 PSU 的插槽压降较小
  - 固件版本不一致: 各 GPU 的 VBIOS 版本可能不同

解决方案:
  1. 核对所有 GPU 的 VBIOS 版本:
     rocm-smi --showvbios
  2. 统一固件版本:
     amdvbflash -p -f unified_bios.rom <gpu_id>
  3. 调整散热风道, 保证各 GPU 进风温度差 < 3°C
  4. 使用 Weighted 策略补偿 ASIC 差异:
     - 漏电流大的 GPU 分配较低功耗预算
     - 能效高的 GPU 分配较高功耗预算
```

### 12.3 SR-IOV VF 功耗限制失效

```text
现象:
  设置 VF 功耗上限后, VF 实际功耗超过限制值

排查步骤:
  1. 确认 PF 已启用 SR-IOV:
     cat /sys/class/drm/card0/device/sriov_numvfs
  2. 检查 VF 的电源策略:
     cat /sys/class/drm/card0/device/virtfn0/power_policy
  3. 确认固件支持 VF 电源限制:
     cat /sys/class/drm/card0/device/vbios_version
     (需 VBIOS >= 0x2010 才支持 VF 电源限制)
  4. 检查 SMU 固件日志:
     cat /sys/kernel/debug/dri/0/amdgpu_smu_log

  注意: SR-IOV VF 电源限制需要硬件和固件同时支持
  部分旧型号 GPU (GCN 架构) 不支持 VF 级别电源管理
```

### 12.4 集群电源管理中的常见问题

```text
问题 1: SLURM 功耗上限未实际生效
  检查:
    - slurmd 是否以 root 权限运行
    - cgroup 是否启用 powercap 控制器
    - GPU 电源上限是否在作业结束后重置

问题 2: Kubernetes GPU 功耗 Pod 启动失败
  检查:
    - ConfigMap 中的 GPU ID 是否正确
    - DaemonSet 是否具有特权模式
    - 主机上 rocm-smi 是否可正常执行

问题 3: 多 GPU 节点过载保护误触发
  检查:
    - PSU 额定功率 vs 实际峰值功率
    - 功耗采样窗口大小 (默认 100ms, 建议改为 1s)
    - 温度传感器是否故障导致误报
```

## 13. 总结与思考

### 13.1 核心要点回顾

```text
多GPU电源管理的关键层次:

  ┌─────────────────────────────────────────────────────────────┐
  │                    数据中心级电源管理                       │
  │  (SLURM/K8s, 机架级预算, 碳排放优化)                       │
  ├─────────────────────────────────────────────────────────────┤
  │                    节点级电源管理                           │
  │  (PSU容量规划, CPU/GPU协同, 散热设计)                      │
  ├─────────────────────────────────────────────────────────────┤
  │                    多GPU协同级                              │
  │  (Crossfire/ROCm, 功耗预算分配策略, 负载均衡)               │
  ├─────────────────────────────────────────────────────────────┤
  │                    单GPU电源域级                            │
  │  (VDDCR/VDDCI/MEM, DVFS, Power Gating)                     │
  ├─────────────────────────────────────────────────────────────┤
  │                    硬件电路级                              │
  │  (VRM PMBus, 相数控制, 效率曲线)                            │
  └─────────────────────────────────────────────────────────────┘

设计原则:
  1. 全局最优 > 局部最优: 牺牲单 GPU 性能换取整体效率
  2. 预测 > 响应: 基于工作负载预测的电源管理优于被动响应
  3. 分层管理: 各层独立决策, 上层指导下层
  4. 安全优先: 任何情况下不超过硬件和 PSU 的限制
```
